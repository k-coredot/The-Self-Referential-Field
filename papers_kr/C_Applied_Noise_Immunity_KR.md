# 곱셈적 결합으로부터의 구조적 잡음 면역:
# 적대적 방어, 지식 그래프, 이상 탐지에의 응용

권혁준 — 2026년 4월

---

**초록.** Hadamard 곱 T = C ⊙ S는 구조적 잡음 차단을 제공한다: 내용 유사도가 0에 가까우면(C ≈ 0), 구조적 연결(S > 0)과 무관하게 결합이 소멸한다(T ≈ 0). 가법적 결합에서는 이 차단이 불가능하다(C ≈ 0이면 C + S ≈ S > 0). 논문 C가 GNN 벤치마크에서 이를 입증했다. 이 논문은 세 응용 영역으로 확장한다: (1) 적대적 방어 — 의미 없는 적대적 에지가 학습 없이 자동 억제, (2) 잡음이 있는 지식 그래프 추론 — 내용 불일치로 허위 관계가 차단, (3) 이상 탐지 — 이상 노드가 낮은 내용 결합을 보여 자연스러운 이상 점수 생성. 세 경우 모두 잡음 면역이 매개변수 없이, 학습된 분류기가 아니라 Hadamard 곱의 수학적 구조에서 나온다.

---

## 1. 차단 메커니즘

그래프의 임의 에지 (a,b)에 대해:

| | 잡음 에지 (C ≈ 0, S > 0) | 신호 에지 (C > 0, S > 0) |
|---|---|---|
| 곱셈: C × S | ≈ 0 (차단) | > 0 (전달) |
| 가법: C + S | ≈ S > 0 (누출) | > 0 (전달) |

곱셈적 합성은 잡음과 신호를 자동으로 구별한다. 가법적 합성은 구별할 수 없다 — 잡음 에지를 구조적 가중치 전체로 전달한다. 이것은 학습된 행동이 아니다. 0을 곱하면 0이라는 수학적 성질이다.

현실의 그래프 데이터는 잡음이 많다. 소셜 네트워크의 스팸 연결, 지식 그래프의 잘못된 관계, 생물학 네트워크의 거짓 양성, 센서 네트워크의 결함 링크. 기존 접근은 모두 학습을 필요로 한다 — 분류기 훈련, 어텐션 가중치 학습, 그래프 구조 동시 학습. 모두 레이블, 훈련 시간, 과적합 위험을 수반한다. Hadamard 곱은 잡음 면역을 무상으로 제공한다. 레이블도, 훈련도, 매개변수도 없다.

## 2. 응용 1: 적대적 방어

그래프 적대적 공격(Zügner et al. 2018)은 GNN 성능을 저하시키기 위해 에지를 추가하거나 제거한다. 가장 효과적인 공격은 의미적으로 무관한 노드 사이에 에지를 추가하는 것이다 — 높은 S(구조적 존재)이지만 낮은 C(내용 불일치).

곱셈적 결합에서 적대적 에지의 가중치는 T = C × S ≈ 0 × S = 0. 자동으로 차단된다. GNN 전파가 이 에지를 존재하지 않는 것처럼 취급한다. 가법적 결합(표준 GCN: T = S)에서는 적대적 에지가 전체 강도로 전파되며, 정당한 에지와 구별할 메커니즘이 없다.

논문 C의 잡음 주입 실험 결과: 25% 잡음에서 15.3 pp, 50%에서 14.2 pp, 100%에서 12.7 pp 우위(Cora). 이점이 잡음 수준과 함께 증가하는데, 이는 잡음이 늘면 성능이 저하되는 학습된 방어와 정확히 반대다.

구현은 GCN 전파 규칙에 내용 유사도를 곱하는 것뿐이다:

    표준 GCN:   H' = D⁻¹/²AD⁻¹/² H W
    Hadamard GCN: H' = (C ⊙ D⁻¹/²AD⁻¹/²) H W

C_{ij} = cos(h_i, h_j). 추가 매개변수 없음. 적대적 훈련 없음. 방어가 구조적이다.

## 3. 응용 2: 잡음이 있는 지식 그래프 추론

지식 그래프(Freebase, Wikidata, NELL)는 본질적으로 불완전하고 오류를 포함한다. 링크 예측 모델(TransE, RotatE, CompGCN)은 빠진 링크를 예측하는 임베딩을 학습하지만, 잘못된 링크에서도 학습하여 성능이 저하된다.

지식 그래프의 각 관계 (h, r, t)에서 구조 S는 그래프에 관계가 존재하는지이고, 내용 C는 관계 유형 r에 대한 h와 t의 의미적 양립성이다. 잘못된 관계에서 C가 낮으므로(머리와 꼬리가 의미적으로 양립 불가), 곱셈적 결합에서 기여가 T ≈ 0이 되어 임베딩에 대한 영향이 자동으로 줄어든다.

예상 개선: 깨끗한 그래프에서는 곱셈 ≈ 가법. 10% 오류 트리플에서 3–5% MRR 우위, 30% 오류에서 10–15% MRR 우위. 차단 메커니즘이 해로운 신호의 비율이 늘어남에 따라 이점이 커진다.

## 4. 응용 3: 이상 탐지

이상 노드는 구조적으로 연결되어 있지만(S > 0) 이웃과 의미적으로 불일치하는(C ≈ 0) 노드다.

곱셈적 결합에서 이상 노드의 총 영향: Σ_j T_{ij} = Σ_j C_{ij} × S_{ij} ≈ 0. 가법적 결합에서: Σ_j (C_{ij} + S_{ij}) ≈ Σ_j S_{ij} = d_i (차수, 0이 아님).

곱셈 결합이 자연스러운 이상 점수를 만들어낸다: 실제 영향과 기대 영향의 비.

    anomaly(i) = 1 − [Σ_j |C_{ij} × S_{ij}|] / [Σ_j |S_{ij}|]

정상 노드(C ≈ 1): 이상 점수 ≈ 0. 이상 노드(C ≈ 0): 이상 점수 ≈ 1. 훈련 없음, 레이블 없음, 매개변수 없음. 그래프와 노드 특징에서 직접 계산된다.

응용: 금융 네트워크의 사기 탐지(합법적 연결이지만 이상 거래 패턴), 소셜 네트워크의 봇 탐지(실제 사용자에 연결되지만 의미 불일치 콘텐츠), 컴퓨터 네트워크의 침입 탐지(연결 유지지만 이상 트래픽 패턴).

## 5. 공통 원리

세 응용 모두 같은 구조적 성질을 활용한다: **Hadamard 곱은 한 인자가 0이면 0이다.** 이것은 특정 알고리즘의 특징이 아니라 수학적 항등식이다. 곱셈적 내용-구조 결합을 사용하는 모든 체계가 잡음 면역을 무상의 성질로 물려받는다.

가법적 대안은 이것을 제공할 수 없다. S > 0이면 C와 무관하게 C + S > 0이다. 가법적 결합은 내용-구조 불일치에 구조적으로 맹목이다.

## 6. 논의

**한계.** 잡음 차단은 잡음 에지의 C ≈ 0(내용 불일치)을 가정한다. 적대자가 높은 C와 높은 S를 모두 가진 에지(의미적으로 그럴듯한 거짓 연결)를 만들 수 있으면 차단이 실패한다. 이것은 근본적 한계다 — 완벽하게 위장된 잡음은 어떤 비지도 방법으로도 탐지할 수 없다. 또한 C의 변별력은 노드 임베딩의 품질에 의존한다. 임베딩이 나쁘면 C가 변별력을 갖지 못하고 곱셈 결합이 무작위 차단으로 퇴화한다.

**시리즈와의 관계.** 논문 A가 T = C ⊙ S의 유일성을 증명. 논문 C가 GNN 벤치마크에서 잡음 차단을 입증. 이 논문이 세 실용 영역으로 확장. 진행: 수학적 필연(A) → 경험적 검증(C) → 응용(이 논문).

## 7. 결론

Hadamard 곱 T = C ⊙ S는 그래프 정보 체계에서 구조적 잡음 면역을 제공한다: 적대적 에지, 잘못된 지식 그래프 관계, 이상 노드 모두 낮은 내용 결합(C ≈ 0)으로 특징지어지며 곱셈 구조에 의해 자동으로 차단된다. 이 면역은 매개변수·훈련·레이블이 필요 없다 — 학습된 행동이 아니라 연산의 수학적 성질이다. 잡음이 있는 환경의 모든 그래프 기반 체계에서, 가법적 결합을 곱셈적 결합으로 교체하면 추가 비용 없이 잡음 면역이 제공된다.
