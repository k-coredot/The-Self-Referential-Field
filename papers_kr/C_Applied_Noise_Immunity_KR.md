# 곱셈적 결합에 의한 구조적 잡음 면역:
# 적대적 방어, 지식 그래프, 이상 탐지에의 응용

권혁준 — 2026년 4월

---

**초록.** Hadamard 곱 T = C⊙S는 구조적 잡음 차단을 제공한다: 내용 유사도가 영 근방(C ≈ 0)이면 구조적 연결성(S > 0)과 무관하게 결합이 소멸한다(T ≈ 0). 이 차단은 가법적 결합(C ≈ 0일 때 C + S ≈ S)으로는 불가능하다. 논문 C가 GNN 벤치마크에서 이를 시연했으며, 잡음 주입 아래 곱셈적 결합이 가법적 결합을 최대 15.3 백분율 포인트 능가했다. 이 결과를 세 응용 영역으로 확장한다: (1) 적대적 방어 — 적대적 에지(구조적으로 존재, 의미적으로 무의미)가 학습된 탐지 없이 자동 억제된다; (2) 잡음 있는 지식 그래프 추론 — 허위 관계가 내용 불일치에 의해 차단되어 불완전 그래프에서 링크 예측이 개선된다; (3) 이상 탐지 — 이상 노드가 구조적 이웃과 낮은 내용 결합을 보여 지도 레이블 없이 자연스러운 이상 점수를 생성한다. 세 경우 모두 잡음 면역은 매개변수-자유이며, 훈련된 분류기가 아닌 Hadamard 곱의 수학적 구조에서 출현한다.

---

## 1. 차단 메커니즘

### 1.1 곱셈적 대 가법적

그래프의 임의의 에지 (a,b)에 대해:

| | 잡음 에지 (C ≈ 0, S > 0) | 신호 에지 (C > 0, S > 0) |
|---|---|---|
| 곱셈: C × S | ≈ 0 (차단) | > 0 (전송) |
| 가법: C + S | ≈ S > 0 (누설) | > 0 (전송) |

곱셈적 합성은 잡음과 신호를 자동 구별한다. 가법적 합성은 불가능 — 잡음 에지를 완전한 구조 강도로 전송한다.

이것은 학습된 행동이 아니다. 곱셈의 수학적 성질이다: 영 곱하기 무엇이든 영이다.

### 1.2 왜 중요한가

현실의 그래프 구조 데이터는 잡음이 있다. 소셜 네트워크는 스팸 연결을 포함한다. 지식 그래프는 부정확한 관계를 포함한다. 생물학 네트워크는 실험적 위양성을 포함한다. 센서 네트워크는 결함 링크를 포함한다.

이 잡음을 처리하는 모든 기존 접근은 학습을 수반한다. 잡음 에지를 탐지하는 분류기 훈련, 억제할 주의 가중치 학습, 과제와 함께 그래프 구조 학습. 모두 레이블 데이터, 훈련 시간을 필요로 하며 과적합 위험이 있다.

Hadamard 곱은 무상으로 잡음 면역을 제공한다. 레이블 없음. 훈련 없음. 매개변수 없음. 연산의 구조 자체가 잡음을 차단한다.

## 2. 응용 1: 적대적 방어

### 2.1 위협

그래프 적대적 공격(Zügner et al. 2018, Xu et al. 2019)은 GNN 성능을 저하시키기 위해 에지를 추가하거나 제거한다. 가장 효과적인 공격은 의미적으로 무관한 노드 사이에 에지를 추가한다 — 높은 S(구조적 존재)와 낮은 C(내용 불일치)를 가진 에지.

### 2.2 방어

곱셈적 결합 아래, 적대적 에지는 T = C × S ≈ 0 × S = 0. 자동 차단된다. GNN 전파가 이들을 존재하지 않는 것처럼 취급한다.

가법적 결합(표준 GCN: T = S) 아래, 적대적 에지가 완전 강도로 전파한다. 모델은 합법적 에지와 구별할 메커니즘이 없다.

### 2.3 예상 성능

논문 C의 잡음 주입 실험으로부터:
- 25% 잡음 에지: 곱셈적이 가법적을 15.3 pp 능가 (Cora)
- 50% 잡음 에지: 곱셈적이 가법적을 14.2 pp 능가 (Cora)
- 100% 잡음 에지: 곱셈적이 가법적을 12.7 pp 능가 (Cora)

이점이 잡음 수준과 함께 **증가** — 잡음 증가 시 저하되는 학습된 방어와 정확히 반대.

### 2.4 구현

GCN 전파 규칙을 교체:

```
표준 GCN:    H' = D⁻¹/²AD⁻¹/² H W
Hadamard GCN: H' = (C ⊙ D⁻¹/²AD⁻¹/²) H W
```

여기서 C_{ij} = cos(h_i, h_j)는 노드 특징 벡터 사이의 내용 유사도. 추가 매개변수 없음. 적대적 훈련 없음. 방어는 구조적이다.

## 3. 응용 2: 잡음 있는 지식 그래프 추론

### 3.1 문제

지식 그래프(Freebase, Wikidata, NELL)는 본질적으로 불완전하고 오류를 포함한다. 링크 예측 모델(TransE, RotatE, CompGCN)은 누락된 링크를 예측하는 임베딩을 학습한다. 그러나 부정확한 링크로부터도 학습하여 성능이 저하된다.

### 3.2 해결

지식 그래프에서 각 관계 (h, r, t)는:
- 구조적 성분 S: 관계가 그래프에 존재
- 내용적 성분 C: 관계 유형 r이 주어졌을 때 h와 t의 의미적 호환성

부정확한 관계에서 C는 전형적으로 낮다(머리와 꼬리가 관계 유형에 대해 의미적으로 비호환). 곱셈적 결합 아래 이들은 메시지 전달에 T ≈ 0을 기여하여 임베딩에 대한 영향을 자동 감소시킨다.

### 3.3 예상 개선

통제된 잡음 주입이 있는 지식 그래프 완성 벤치마크:
- 깨끗한 그래프: 곱셈 ≈ 가법 (깨끗한 데이터에서 둘 다 잘 작동)
- 10% 부정확 삼중: 곱셈이 3–5% MRR로 능가 예상
- 30% 부정확 삼중: 곱셈이 10–15% MRR로 능가 예상

이점이 잡음 비율에 따라 스케일하며, 차단 메커니즘이 유해 신호의 증가하는 비율을 억제한다.

## 4. 응용 3: 이상 탐지

### 4.1 통찰

그래프의 이상 노드는 구조적으로 연결되어 있으나(에지에 대해 S > 0) 이웃과 의미적으로 불일치하는(C ≈ 0) 노드다.

곱셈적 결합 아래, 이상 노드의 총 영향:

Σ_j T_{ij} = Σ_j C_{ij} × S_{ij} ≈ 0   (모든 이웃에 대해 C_{ij} ≈ 0이므로)

가법적 결합 아래:

Σ_j (C_{ij} + S_{ij}) ≈ Σ_j S_{ij} = d_i   (차수, 영이 아님)

곱셈적 결합은 자연스러운 이상 점수를 생성한다: **실제 영향 대 기대 영향의 비율.** 이상 노드는 높은 구조적 연결에도 영 근방 실제 영향을 갖는다.

### 4.2 이상 점수

```
anomaly(i) = 1 − [Σ_j |C_{ij} × S_{ij}|] / [Σ_j |S_{ij}|]
```

모든 이웃에 대해 C_{ij} ≈ 1(정상 노드): 이상 ≈ 0.
모든 이웃에 대해 C_{ij} ≈ 0(이상 노드): 이상 ≈ 1.

이 점수는 훈련 영, 레이블 영, 매개변수 영을 필요로 한다. 그래프와 노드 특징으로부터 직접 계산된다.

### 4.3 응용

- 금융 네트워크의 사기 탐지: 사기 계좌가 합법적 연결(S > 0)을 갖지만 이상 거래 패턴(C ≈ 0)
- 소셜 네트워크의 봇 탐지: 봇이 실제 사용자에 연결(S > 0)되지만 의미 불일치 콘텐츠 생성(C ≈ 0)
- 컴퓨터 네트워크의 침입 탐지: 침해 노드가 연결 유지(S > 0)하지만 이상 트래픽 패턴(C ≈ 0)

## 5. 공통 원리

세 응용 모두 같은 구조적 성질을 활용한다: **Hadamard 곱은 어느 한 인수가 영이면 영이다.** 이것은 특정 알고리즘의 특성이 아니다 — 수학적 항등식이다. 곱셈적 내용-구조 결합을 사용하는 임의의 체계가 잡음 면역을 무상 성질로 상속한다.

가법적 대안은 이를 제공할 수 없다. S > 0이면 C와 무관하게 C + S > 0이다. 가법적 결합은 내용-구조 불일치에 구조적으로 맹목이다.

## 6. 논의

### 6.1 한계

잡음 차단은 잡음 에지가 C ≈ 0(내용 불일치)을 가짐을 가정한다. 적대자가 높은 C와 높은 S 모두를 가진 에지(의미적으로 그럴듯한 거짓 연결)를 제작할 수 있다면, 차단이 실패한다. 이것은 근본적 한계다: 완벽히 위장된 잡음은 비지도 방법으로 탐지할 수 없다.

내용 유사도 C는 노드 임베딩의 품질에 의존한다. 임베딩이 빈약하면(무작위, 무정보) C가 판별을 제공하지 않으며 곱셈적 결합이 무작위 차단으로 퇴화한다. 좋은 임베딩이 전제조건이다.

### 6.2 시리즈와의 관계

논문 A가 T = C⊙S의 유일성을 증명. 논문 C가 GNN 벤치마크에서 잡음 차단을 시연. 이 논문이 잡음 차단을 세 실용 영역으로 확장. 진행: 수학적 필연성(A) → 경험적 검증(C) → 응용(이 논문).

## 7. 결론

Hadamard 곱 T = C⊙S는 그래프 정보 시스템에서 구조적 잡음 면역을 제공한다: 적대적 에지, 부정확한 지식 그래프 관계, 이상 노드 모두 낮은 내용 결합(C ≈ 0)에 의해 특성화되며 곱셈 구조에 의해 자동 차단된다. 이 면역은 매개변수-자유, 훈련-자유, 레이블-자유 — 학습된 행동이 아닌 연산의 수학적 성질이다. 잡음 환경에서 작동하는 그래프 기반 체계에서 가법적 결합을 곱셈적 결합으로 교체하면 영 비용으로 잡음 면역을 제공한다.

## 감사의 글

저자는 수학적 검증, 수치 시뮬레이션, 원고 작성 보조를 위한 계산 도구로 Claude (Anthropic)를 사용하였다. 공리, 정리, 증명, 물리적 해석, 전략적 판단을 포함한 모든 이론적 내용은 저자의 독창적 작업이다. 저자가 모든 내용에 대해 전적인 책임을 진다.
