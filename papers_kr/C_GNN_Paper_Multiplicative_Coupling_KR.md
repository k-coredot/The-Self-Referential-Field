# 그래프 신경망에서 곱셈적 내용-구조 결합의 필연성

권혁준 — 2026년 4월

---

**초록.** 그래프 신경망은 노드 특징(내용)과 그래프 위상(구조)을 결합하여 정보를 전파하지만, 내용과 구조를 *어떻게* 결합해야 하는지에 대한 원리적 방법론은 존재하지 않는다. Hadamard(원소별) 곱 T = C ⊙ S — 여기서 C는 쌍별 상태 내적이고 S는 자기수반 정규화 인접행렬 D⁻¹/²AD⁻¹/² — 가 국소성, 회전 불변성, 자기수반 스펙트럼 정합성, 변분 안정성을 만족하는 유일한 에지 연산자임을 증명한다. 가법적 결합(C + S), 행렬 곱, 텐서 곱, (α,β) ≠ (1,1)인 모든 비선형 대안 C^α S^β가 이 공리에 의해 배제된다. 매개변수-자유 장 역학을 사용하여 Cora, CiteSeer, PubMed에서 이론을 검증한다: 잡음 주입(0–100% 허위 에지) 아래, 곱셈적 결합이 가법적 결합을 최대 15.3 백분율 포인트(Cora, 25% 잡음) 능가하며, 이점이 잡음 수준과 함께 *증가*한다. 에지별 결합 가중치 분석이 메커니즘을 밝힌다: 곱셈적 결합은 잡음 에지가 영 근방 가중치를 받는(C ≈ 0 ⟹ C⊙S ≈ 0) 자연스러운 차단 효과를 생성하는 반면, 가법적 결합은 잡음을 증폭한다(C ≈ 0이지만 C+S ≈ S > 0). 이 결과들은 GNN 전파 연산자가 설계 선택이 아닌 수학적 필연성이며, 그 곱셈 구조가 학습된 매개변수 없이 자동 잡음 억제를 제공함을 확립한다.

---

## 1. 서론

이 논문의 이론적 기여는 한 줄이다: C = 0이면 C ⊙ S = 0. 이후의 모든 것 — 잡음 면역, 적대적 방어, 이상 탐지 — 은 영에 의한 곱셈의 귀결이다. 이 논문의 나머지는 이 산술을 표준 벤치마크에서 검증한다.

임의의 그래프 신경망의 핵심 연산은 메시지 전달이다: 그래프 구조를 통해 이웃 노드로부터의 정보를 집계한다. 모든 GNN 아키텍처는 — 명시적이든 암묵적이든 — 두 근본적으로 다른 정보 원천을 어떻게 결합할지 선택한다:

- **내용(C)**: 노드 상태 사이의 유사도 또는 관계
- **구조(S)**: 그래프의 위상과 차수 분포

GCN (Kipf & Welling, 2017)은 S만(정규화 인접행렬)으로 메시지에 가중치를 두며, 내용은 학습된 변환을 통해서만 진입한다. GAT (Veličković et al., 2018)는 내용 의존 주의 가중치를 학습하지만 구조적 정규화와 독립적으로 취급한다. 이 접근들 중 어느 것도 *왜* 특정 내용-구조 결합이 선호되어야 하는지에 대한 이론적 정당화를 제공하지 않는다. 선택은 초매개변수로 취급되며, 벤치마크에서 경험적으로 평가된다.

이 선택이 자유롭지 않음을 증명한다.

## 2. 이론적 프레임워크

### 2.1 공리

**공리 1 (국소성).** 에지 가중치 T_{ab}는 국소 데이터에만 의존한다: 상태 φ(a), φ(b)와 (a,b) 이웃의 그래프 구조.

**공리 2 (회전 불변성).** 모든 직교 변환 R ∈ SO(d)에 대해 T_{ab}는 불변.

**공리 3 (자기수반 스펙트럼 정합성).** 전파 연산자의 구조적 성분은 자기수반이고 차수 이질성 아래 그래프 라플라시안의 스펙트럼 성질을 보존한다.

### 2.2 성분의 유일성

**명제 1** (내용 유일성). 공리 2 아래, C_{ab} = ⟨φ_a, φ_b⟩가 단위 벡터 위의 유일한 회전 불변 쌍선형 형식(스칼라 배까지).

*증명.* f: ℝ^d × ℝ^d → ℝ를 쌍선형이고 SO(d)-불변으로 두자. 임의의 쌍선형 형식은 f(x,y) = x^T M y로 쓸 수 있다. 불변성은 모든 R ∈ SO(d)에 대해 R^T M R = M을 요구한다. Schur 보조정리에 의해 M = λI. 따라서 f(x,y) = λ⟨x,y⟩. □

*GNN 해석.* 이것은 노드 임베딩 사이의 코사인 유사도다. −1(반대)에서 +1(동일)까지 범위. 결정적으로 *음수*가 될 수 있다 — 과평활화 방지에 필수적인 성질.

**명제 2** (구조 유일성). 공리 3 아래, S = D⁻¹/²AD⁻¹/²가 유일한 자기수반 차수 정규화.

*증명.* D^α A D^γ 형태의 차수 정규화 행렬을 고려. 자기수반성은 α = γ를 요구. D^α A D^α가 그래프 라플라시안과 정합적 스펙트럼 성질([−1,1] 고유값)을 갖으려면 α = −1/2 (Chung, 1997). □

*GNN 해석.* GraphSAGE 등 많은 아키텍처에서 사용하는 확률보행 정규화 D⁻¹A는 자기수반이 *아니다*. 비방향 그래프에서 D⁻¹A는 비대칭 가중치를 부여한다: d_u ≫ d_v이면 허브에서 잎으로의 정보가 우선 흐른다. 비방향 그래프에서 이 방향 편향은 물리적 정당화가 없으며, 평균 집계를 사용하는 심층 GNN에서 허브 지배의 구조적 원인이다.

### 2.3 합성 유일성

**보조정리 1** (Hadamard 필연성). 임의의 쌍선형, 점별 의존 연산 •: M_E × M_E → M_E는 • = λ(⊙)를 만족한다.

*증명.* 점별 의존은 (A • B)_{ab} = f(A_{ab}, B_{ab})를 의미. 두 스칼라 인수에 대한 f의 쌍선형성: f(x,y) = λxy. 따라서 (A • B)_{ab} = λ · A_{ab} · B_{ab}, 즉 • = λ(⊙). □

### 2.4 유일한 연산자

**정리 1** (변분 필연성). 공리 1–3과 변분 안정성 아래, 에지 연산자는 유일하게:

    T_{ab} = C_{ab} · S_{ab} = ⟨φ_a, φ_b⟩ / √(d_a · d_b)

*증명.* 명제 1–2가 C와 S를 유일하게 결정. 보조정리 1이 합성을 Hadamard 곱으로 결정. 비선형 대안 T = C^α · S^β를 배제하는 것이 남는다. ∂²T/∂C² = α(α−1) C^{α−2} S^β = 0이 요구되며, α = 0 또는 α = 1. α = 0은 T를 노드 상태와 무관하게 만들어 공리 1에 모순. 따라서 α = 1. 같은 논증으로 β = 1. □


## 3. GNN 설계에 대한 함의

### 3.1 잡음 차단 성질

곱셈적 결합의 가장 중대한 함의는 자동 잡음 억제다.

무관한 노드를 연결하는 허위 에지 (a,b)를 고려. 상태 φ_a와 φ_b는 근사적으로 직교하므로 C_{ab} = ⟨φ_a, φ_b⟩ ≈ 0. 곱셈적 결합 아래:

    T_{ab} = C_{ab} · S_{ab} ≈ 0 · S_{ab} = 0

구조 가중치 S_{ab}와 무관하게 잡음 에지가 자동 침묵한다.

가법적 결합 아래:

    T_{ab} = C_{ab} + S_{ab} ≈ 0 + S_{ab} = S_{ab} > 0

잡음 에지가 완전한 구조 강도로 전송한다. 내용이 보호를 제공하지 않는다.

이것은 정량적 이점이 아니라 *정성적* 차이다. 곱셈적 결합은 잡음을 차단할 수 있고; 가법적 결합은 구조적으로 불가능하다.

### 3.2 과평활화 설명

**(i) 반발의 부재.** 표준 GNN은 양의 집계 가중치만 사용(ReLU 활성화, 양의 주의). T = C ⊙ S에서 내용 항 C_{ab} = cos(φ_a, φ_b)는 *음수*가 될 수 있다. 음의 C는 비유사 노드가 전파 중 서로를 밀어냄을 의미한다. 이 반발력이 표현 붕괴를 방지한다.

**(ii) 비자기수반 정규화.** 확률보행 연산자 D⁻¹A는 정상 분포 π_v ∝ d_v로 수렴하여 모든 노드 정보를 소거한다. 자기수반 연산자 D⁻¹/²AD⁻¹/²는 스펙트럼 성분을 보존하며, 단일 점으로의 수렴이 아닌 고유값 감쇠에 의해 지배되는 자연스러운 감쇠를 갖는다.

### 3.3 초매개변수 제거

연산자 T = C ⊙ S (S = D⁻¹/²AD⁻¹/²)는 제거한다:
- **정규화 유형**: 유일하게 결정(명제 2)
- **집계 유형**: Hadamard 곱으로 유일하게 결정(보조정리 1)
- **감쇠/제동**: cos 누적이 자연 감쇠 제공(r 홉당 cos^r)
- **학습된 주의 가중치**: C_{ab} = ⟨φ_a, φ_b⟩는 매개변수 영개 필요

### 3.4 계산 이점

표준 메시지 전달은 AX를 O(|E|·d) 비용으로 계산. 연산자 T = C ⊙ S는 에지별 스칼라를 O(|E|) 비용으로 계산하며, 중간 행렬 저장이 없다.


## 4. 실험 검증: 장 역학 벤치마크

### 4.1 동기

표준 GNN 벤치마크는 학습된 매개변수로 분류 정확도를 평가하여, 전파 연산자와 훈련 역학, 최적화 선택, 정규화를 혼동한다. 결합 연산자 단독의 효과를 격리하기 위해 *장 역학*을 사용한다: 결정론적 국소 힘 아래 노드 표현이 진화하는 비지도적, 매개변수-자유 과정.

### 4.2 설정

**데이터셋.** Cora (2,708 노드, 10,556 에지, 7 클래스), CiteSeer (3,327 노드, 9,104 에지, 6 클래스), PubMed (19,717 노드, 88,648 에지, 3 클래스).

**장 역학.** 각 노드 i는 노드 특징 행렬을 64차원으로 무작위 사영한 후 L2 정규화하여 단위 구에 초기화된 표현 벡터 h_i를 갖는다. 각 사이클에서 노드 i에 대한 국소 힘:

    force_i = Σ_{j∈N(i)} T_{ij} · dir(h_j → h_i) · η

여기서 dir(h_j → h_i) = (h_j − h_i) / ‖h_j − h_i‖는 단위 방향 벡터. 레이블, 손실 함수, 학습 매개변수 없음.

**비교 결합 연산자:** 곱셈(C⊙S), 가법(C+S), 구조만(S), 내용만(C).

**잡음 주입.** 원래 에지 수의 0%, 25%, 50%, 100%에 해당하는 허위 에지를 무작위 노드 쌍 사이에 추가.

**평가.** 피크 자기조직 시간(사이클 100–400)에서의 꼭짓점 k-최근접 이웃(kNN) 분류 정확도.

네 가지 일관된 패턴이 모든 데이터셋에 걸쳐 나타난다:

**(i) 잡음이 곱셈적 이점을 증폭한다.** 0% 잡음에서 방법들이 비슷하게 수행. 잡음이 증가하면 곱셈적 결합은 정확도를 유지하는 반면 가법적 결합은 급격히 저하. 이것은 §3.1의 잡음 차단 예측을 직접 확인한다.

**(ii) 가법적 결합이 잡음 아래 일관되게 최악.** C+S는 C_{ab}와 무관하게 S_{ab} > 0이므로 잡음 에지를 차단할 수 없다.

**(iii) CiteSeer의 깨끗한 그래프 결과가 이론을 확인한다.** CiteSeer는 낮은 동질성(이웃이 자주 다른 클래스)을 가져 실제 에지에서도 C_{ab}가 작은 경향. 깨끗한 그래프에서 곱셈적 결합은 신호와 잡음을 동등하게 감쇠시켜 S만보다 낮은 정확도를 산출(C⊙S = 51.3% 대 S = 60.4%). 이것은 이론의 실패가 아닌 직접 예측이다. 결정적으로, 잡음 주입 아래 이 같은 감쇠가 선택적으로 유익해져 — 25% 잡음 이상에서 C⊙S가 S만을 추월한다.

**(iv) 사중 소거: 두 성분 모두 필요.** C만 소거가 특히 시사적이다. 잡음 아래 C만은 기회 수준 근처로 붕괴한다 — 차수 정규화 없이 내용 유사도만으로는 일관된 자기조직을 유지할 수 없다. S만은 차수 보정 전파를 제공하지만 실제와 잡음 에지를 구별할 수 없다. 곱셈적 결합 C⊙S만이 둘 다를 달성한다: S로부터의 차수 정규화 전파와 C로부터의 내용 기반 차단.

**이것이 중심 실험적 발견이다: C⊙S가 내용이 판별적인 데이터셋(Cora, CiteSeer)에서 학습 매개변수 없이 선택적 에지 차단(비율이 실질적으로 1 초과)을 달성하는 유일한 연산자다.**


## 5. 격자 게이지 이론과의 연결

여기서 GNN을 위해 도출된 연산자 T = C ⊙ S는 격자 그래프 위의 게이지-기하 결합에 필연적으로 증명된 에지 연산자 [K, 2026a; K, 2026b]와 동일하다. 그 맥락에서 C는 게이지 링크 변수이고 S는 Regge 기하다. 7 격자 배위(2D–4D, U(1)과 SU(2), 42 측정, 전부 13σ 이상)에 걸친 T = C ⊙ S의 보편성 [Kwon, 2026b]은 여기서 GNN에 적용된 수학적 구조의 독립적 물리적 검증을 제공한다. 이 교차 영역 보편성은 우연이 아니다 — 특정 응용 영역을 참조하지 않는 공리로부터 따른다.


## 6. 관련 연구

**그래프 정규화.** Kipf & Welling (2017)이 D⁻¹/²AD⁻¹/²를 스펙트럼 근사로 도입. 우리의 명제 2는 이것이 유일한 자기수반 정규화임을 증명하며, 편리한 선택이 아니다.

**GNN의 내용-구조 결합.** GAT (Veličković et al., 2018)는 내용 의존 주의를 학습하지만 매개변수와 레이블을 필요로 한다. 우리의 T = C ⊙ S는 영 매개변수로 내용 의존 가중치를 달성한다.

**잡음 견고성.** GNNGuard (Zhang & Zitnik, 2020)와 ProGNN (Jin et al., 2020)은 적대적 에지를 학습하여 탐지하고 제거한다. 우리의 곱셈적 결합은 연산자의 구조적 성질로서 학습 *없이* 잡음 차단을 제공한다.


## 7. 논의

**한계.** 유일성 결과는 비방향 그래프를 가정; 방향 그래프로의 확장은 자기수반성(공리 3)의 완화를 요구하며, 결과 연산자 체계는 열린 문제다. 장 역학 벤치마크는 과제 특정 정확도가 아닌 자기조직 품질을 평가한다; 잡음 차단 이점이 지도 설정으로 전이될 것을 예상하지만 이를 시연하지 않았다. 세 표준 인용 네트워크에서 검증; 대규모 벤치마크(예: OGB) 평가는 향후 과제다.


## 8. 결론

GNN 전파 연산자 T = C ⊙ S가 설계 선택이 아닌 수학적 필연성임을 증명했다. 국소성, 회전 불변성, 자기수반 스펙트럼 정합성, 변분 안정성에 의해 유일하게 결정된다. 곱셈적(Hadamard) 구조는 가법적 결합으로는 구조적으로 불가능한 자동 잡음 차단을 제공하며, 통제된 잡음 주입 아래 세 표준 벤치마크에서 검증되었다. 이 결과들은 GNN 설계를 위한 원리적 기초를 확립한다: 전파 연산자, 정규화, 내용-구조 합성이 모두 공리에 의해 유일하게 결정되어, 이 선택에서 초매개변수 조율의 여지가 없다.
