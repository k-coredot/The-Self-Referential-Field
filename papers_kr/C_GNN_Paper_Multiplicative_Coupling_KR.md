# 그래프 신경망에서 곱셈적 내용-구조 결합의 필연성

권혁준 — 2026년 4월

---

**초록.** 그래프 신경망은 노드 특징(내용)과 그래프 위상(구조)을 결합하여 정보를 전파하지만, 내용과 구조를 어떻게 결합해야 하는지에 대한 원리적 방법은 존재하지 않는다. Hadamard 곱 T = C ⊙ S — C는 쌍별 상태 내적, S는 자기수반 정규화 인접 D⁻¹/²AD⁻¹/² — 가 국소성, 회전 불변성, 자기수반 스펙트럼 정합성, 변분 안정성을 만족하는 유일한 에지 연산자임을 증명한다. 가법적 결합(C + S), 행렬 곱, 텐서 곱, (α,β) ≠ (1,1)인 비선형 대안 C^α S^β 모두 배제된다. Cora, CiteSeer, PubMed에서 매개변수 없는 장 역학으로 검증한다: 잡음 주입(0–100% 허위 에지) 아래, 곱셈적 결합이 가법적 결합을 최대 15.3 퍼센트포인트(Cora, 25% 잡음) 앞서며, 이점은 잡음 수준이 높아질수록 증가한다. 메커니즘은 잡음 에지의 내용 유사도가 0에 가까울 때 곱셈에 의해 가중치가 자동으로 0이 되는 차단 효과다.

---

## 1. 서론

이 논문의 이론적 기여는 한 줄이다: C = 0이면 C · S = 0. 잡음 면역, 적대적 방어, 이상 탐지 — 이 모든 것이 0을 곱하면 0이 된다는 산술의 귀결이다. 이 논문의 나머지는 이 산술을 표준 벤치마크에서 검증한다.

모든 GNN 아키텍처의 핵심 연산은 메시지 전달이며, 이 과정에서 내용(C)과 구조(S)를 결합하는 방법을 선택해야 한다. GCN(Kipf & Welling 2017)은 S만 사용하고, GAT(Veličković et al. 2018)는 학습된 어텐션을 사용하며, GraphSAGE(Hamilton et al. 2017)는 이웃 샘플링과 풀링을 쓴다. 어떤 접근도 왜 특정 결합 방식이 선호되어야 하는지에 대한 이론적 정당화를 제공하지 않는다.

우리는 이 선택이 자유롭지 않음을 증명한다. 세 공리와 변분 안정성으로부터, 내용은 내적 C_{ab} = ⟨φ_a, φ_b⟩, 구조는 D⁻¹/²AD⁻¹/², 합성은 Hadamard 곱, 지수는 (α,β) = (1,1)로 유일하게 결정된다. 결과 연산자 T_{ab} = ⟨φ_a, φ_b⟩ / √(d_a · d_b)는 학습 매개변수와 초매개변수가 전혀 필요 없다.

## 2. 이론적 틀

### 2.1 공리

**공리 1 (국소성).** T_{ab}는 국소 데이터 — φ(a), φ(b), (a,b) 주변의 그래프 구조 — 에만 의존한다.

**공리 2 (회전 불변성).** 모든 노드 상태에 직교 변환 R ∈ SO(d)를 적용해도 T_{ab}가 변하지 않는다.

**공리 3 (자기수반 스펙트럼 정합성).** 구조 전파 연산자가 대칭이고 그래프 라플라시안의 스펙트럼 성질을 보존한다.

### 2.2 성분의 유일성

**내용(명제 1).** Schur 보조정리에 의해, C_{ab} = ⟨φ_a, φ_b⟩가 유일한 회전 불변 쌍선형 형식이다. RBF 커널 같은 비쌍선형 불변량은 변분 안정성에 의해 배제되며, 또한 스케일 매개변수 σ를 도입하여 매개변수 없는 요건에 위배된다. 쌍선형성은 독립 가정이 아니라 공리의 귀결이다.

GNN 관점에서: 코사인 유사도다. −1(정반대)에서 +1(동일)까지. 음수가 될 수 있다는 점이 과평활화 방지에 필수적이다.

**구조(명제 2).** 자기수반성으로부터 S = D⁻¹/²AD⁻¹/²가 유일하다. 확률보행 정규화 D⁻¹A는 자기수반이 아니다 — 무방향 그래프에서 u→v와 v→u의 가중치가 다르며, d_u ≫ d_v이면 허브에서 말단으로 정보가 편향된다. 깊은 GNN에서 허브 지배의 구조적 원인이다.

### 2.3 합성의 유일성

**보조정리 1 (Hadamard 필연성).** 쌍선형이면서 점별 의존적인 연산은 Hadamard 곱의 스칼라 배뿐이다. 행렬 곱은 점별 의존이 아니고(각 출력이 한 행과 한 열 전체에 의존), 텐서 곱은 에지당 스칼라가 아니며, 직접 합은 쌍이지 스칼라가 아니다.

### 2.4 유일한 연산자

**정리 1 (변분 필연성).** T = C^α S^β에서 변분 안정성은 ∂²T/∂C² = α(α−1)C^{α−2}S^β = 0을 요구한다. α(α−1) = 0이므로 α = 0 또는 α = 1. α = 0이면 노드 상태에 무관하므로 국소성 공리에 위배. 따라서 α = 1. 같은 논증으로 β = 1. 결과:

    T_{ab} = C_{ab} · S_{ab} = ⟨φ_a, φ_b⟩ / √(d_a · d_b)

## 3. GNN 설계에 대한 함의

### 3.1 잡음 차단

곱셈적 결합의 가장 중요한 함의는 자동 잡음 억제다. 무관한 노드를 잇는 허위 에지에서 상태가 거의 직교하므로 C_{ab} ≈ 0이다.

    곱셈: T_{ab} = C_{ab} · S_{ab} ≈ 0 · S_{ab} = 0      (잡음 침묵)
    가법: T_{ab} = C_{ab} + S_{ab} ≈ 0 + S_{ab} = S_{ab}   (잡음 통과)

정량적 차이가 아니라 정성적 차이다. 곱셈적 결합은 잡음을 차단할 수 있고, 가법적 결합은 구조적으로 차단할 수 없다.

### 3.2 과평활화

과평활화는 반복적 메시지 전달로 모든 노드 표현이 하나로 수렴하는 현상이다. 두 원인이 있다. (i) 표준 GNN이 양의 가중치만 사용하여 반발력이 없다. T = C ⊙ S에서 C는 음수가 될 수 있으므로, 비유사 노드 사이에 반발이 생겨 표현 붕괴를 방지한다. (ii) 확률보행 연산자 D⁻¹A는 정상 분포 π_v ∝ d_v로 수렴하여 노드 정보를 소거한다. 자기수반 연산자 D⁻¹/²AD⁻¹/²는 스펙트럼 성분을 보존하며, 고유값 감쇠에 의한 자연적 감쇠만 있을 뿐 단일점 수렴이 아니다.

### 3.3 초매개변수 제거

T = C ⊙ S는 정규화 유형(명제 2), 집계 유형(보조정리 1), 감쇠(코사인 누적이 자연 감쇠), 학습 어텐션 가중치(C가 매개변수 없이 계산) 모두를 제거한다.

### 3.4 계산 이점

표준 메시지 전달은 AX를 O(|E|·d) 비용으로 계산한다. T = C ⊙ S는 에지별 스칼라를 O(|E|)로 계산하며, 중간 행렬 저장이 필요 없다.

## 4. 실험 검증: 장 역학 벤치마크

### 4.1 동기

표준 GNN 벤치마크는 학습 매개변수와 전파 연산자의 효과를 혼동한다. 전파 연산자만의 효과를 분리하기 위해, 비지도·매개변수 없는 장 역학을 사용한다: 노드 표현이 결합 연산자에 의한 결정론적 국소 힘 아래 진화한다.

### 4.2 설정

**데이터.** Cora(2,708 노드, 10,556 에지, 7 클래스), CiteSeer(3,327 노드, 9,104 에지, 6 클래스), PubMed(19,717 노드, 88,648 에지, 3 클래스).

**장 역학.** 노드 표현 h_i를 64차원으로 무작위 투영 후 L2 정규화. 매 순환마다 국소 힘 force_i = Σ_{j∈N(i)} T_{ij} · dir(h_j → h_i) · η를 적용하고 정규화. 레이블, 손실 함수, 학습 매개변수 없음.

**비교 연산자.** 곱셈(C ⊙ S), 가법(C + S), 구조만(S), 내용만(C) — 완전 절제 설계.

**잡음 주입.** 원래 에지 수의 0%, 25%, 50%, 100%에 해당하는 허위 에지를 무작위 노드 쌍 사이에 추가.

### 4.3 결과

Cora에서: 깨끗한 그래프에서 C ⊙ S(70.2%)와 S(70.3%)가 유사하지만, 25% 잡음에서 C ⊙ S(48.2%) 대 C+S(32.9%)로 **15.3 퍼센트포인트** 차이. 100% 잡음에서도 C ⊙ S(39.1%) 대 C+S(32.6%)로 6.5 퍼센트포인트 우위.

CiteSeer에서: 깨끗한 그래프에서 C ⊙ S(51.3%)가 S(60.4%)보다 낮다 — 동질성이 낮아 실제 에지에서도 C가 작기 때문이다. 이것은 이론의 실패가 아니라 직접적 예측이다: 내용이 정보가 없으면 곱셈 결합이 그것을 정직하게 보고한다. 그러나 25% 잡음부터 C ⊙ S가 S를 역전하며, 50% 잡음에서 C ⊙ S(33.8%) 대 C+S(20.2%) — 12.8 퍼센트포인트 차이.

PubMed에서: 잡음 모든 수준에서 C ⊙ S가 우세. 50% 잡음에서 C ⊙ S(63.8%) 대 S(57.2%).

네 가지 일관된 패턴: (i) 잡음이 곱셈 이점을 증폭한다. (ii) 가법적 결합이 잡음 아래 일관되게 최악이다. (iii) CiteSeer의 깨끗한 결과가 이론을 확인한다. (iv) 네 방향 절제에서 두 성분 모두 필요하다 — C만으로는 차수 보정이 없고, S만으로는 잡음 구별이 없다.

### 4.4 메커니즘: 에지별 가중치 분석

PubMed 100% 잡음에서, 에지 유형별 평균 결합 가중치의 실제/잡음 비:

    C ⊙ S: 1.19   (실제 에지에 더 높은 가중치 — 선택적 차단)
    C만:   0.80   (차수 편향이 잡음을 증폭)
    S만:   0.65   (구분 불가)
    C+S:   0.92   (잡음이 더 큰 가중치)

C ⊙ S만이 비 > 1을 달성한다. Cora에서 1.52–1.59, CiteSeer에서 1.93–2.07로, 내용이 변별력 있는 데이터에서 강한 선택적 차단을 보인다. 내용만(C)은 모든 조건에서 1.0 근방 또는 이하이며, 가법(C+S)도 거의 항상 1 미만이다.

**이것이 핵심 실험 결과다: C ⊙ S가 내용에 변별력이 있는 데이터에서 학습 매개변수 없이 선택적 에지 차단(비 > 1)을 달성하는 유일한 연산자다.**

### 4.5 과평활화 역학

세 방법 모두 결국 과평활화에 도달하지만, 곱셈적 결합의 피크 조직화는 순환 150–400에서, 구조만은 50–100에서 일어난다. |C| ≤ 1이므로 |C ⊙ S| ≤ |S|, 역학이 느려져 자기조직화에 더 많은 시간이 주어진다. 변분 안정성(α = β = 1)이 초선형 증폭을 방지하여 자명 고정점으로의 수렴을 늦추는 것이다.

## 5. 격자 게이지 이론과의 연결

여기서 GNN에 대해 도출한 T = C ⊙ S는 격자 그래프 위의 게이지-기하 결합에 대해 증명된 에지 연산자와 동일하다. 7가지 격자 배치(2D–4D, U(1)/SU(2), 42개 측정, 전부 13σ 이상)에서의 보편성이 여기서 GNN에 적용된 수학적 구조의 독립적 물리적 검증을 제공한다. 이 교차 영역 보편성은 우연이 아니다 — 공리가 특정 응용 영역을 참조하지 않기 때문이다.

## 6. 관련 연구

**그래프 정규화.** Kipf & Welling(2017)이 D⁻¹/²AD⁻¹/²를 스펙트럼 근사로 도입했다. 명제 2는 이것이 유일한 자기수반 정규화임을 증명한다. PairNorm, NodeNorm 등은 과평활화의 증상을 다루지만 근본 원인(비자기수반성과 반발적 내용 결합의 부재)을 식별하지 않는다.

**GNN의 내용-구조 결합.** GAT는 내용 의존 어텐션을 학습하지만 매개변수와 레이블이 필요하다. APPNP는 개인화 PageRank(확률보행 변형, 비자기수반)를 사용한다. T = C ⊙ S는 매개변수 없이 내용 의존 가중치를 달성한다.

**잡음 견고성.** GNNGuard, ProGNN 등은 적대적 에지를 학습으로 탐지·제거한다. 곱셈적 결합은 학습 없이 연산자의 구조적 성질로 잡음을 차단한다.

**신경망의 Hadamard 곱.** LSTM과 GRU의 게이팅에 점별 곱이 나타난다. 보조정리 1이 Hadamard 곱이 유일한 쌍선형 국소 합성임을 증명하여, 이 경험적 선택에 이론적 정당화를 부여한다.

## 7. 논의

**한계.** 유일성 결과는 무방향 그래프를 가정한다. 방향 그래프로의 확장은 공리 3의 완화를 요구하며, 열린 문제다. 장 역학 벤치마크는 자기조직화 품질을 평가하며 과업 특화 정확도는 아니다. 과평활화 지연은 과평활화를 제거하지 않고 연기한다. 세 인용 네트워크에서만 검증했다.

**쌍선형성 가정의 범위.** 보조정리 1의 쌍선형 요건이 RBF 커널 같은 비선형 유사도를 배제한다. 비선형 유사도는 조율이 필요한 스케일 매개변수를 도입하여 매개변수 없는 요건에 위배되며, 변분 안정성이 독립적으로 내용 함수의 선형 진입을 강제한다.

## 8. 결론

GNN 전파 연산자 T = C ⊙ S는 설계 선택이 아니라 수학적 필연이며, 국소성, 회전 불변성, 자기수반 스펙트럼 정합성, 변분 안정성에 의해 유일하게 결정된다. 곱셈(Hadamard) 구조가 가법적 결합에서는 구조적으로 불가능한 자동 잡음 차단을 제공하며, 세 표준 벤치마크에서 잡음 주입으로 검증되었다. 전파 연산자, 정규화, 내용-구조 합성 모두 공리에 의해 유일하게 결정되어, 이 선택들에 대한 초매개변수 조율의 여지가 없다.
