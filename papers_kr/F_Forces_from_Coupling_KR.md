# 공유 기하 위 다중 게이지 결합으로부터의 초가법적 힘:
# 정렬, 전파, 해석적 구조

권혁준 — 2026년 4월

---

**초록.** 다수의 게이지장이 역학적 격자 기하를 공유할 때, 각 장이 기하에 행사하는 힘 — F_e = −δE/δl_e로 정의 — 은 스칼라 에너지 분석이 드러내지 않는 다섯 성질을 보인다. (1) 독립 장의 힘이 >99%로 정렬되며, 정렬이 N 증가와 함께 향상. (2) 힘이 기하가 가장 덜 변형된 곳에 집중(반상관 r = −0.64)하여 전파하는 변형 파면을 생성. (3) 순 힘 크기가 |F| ~ N^(1.69 ± 0.06) (R² = 0.997)로 스케일하여 에너지 초가법성 지수와 일치. (4) 힘 방향이 시간적으로 안정(연속 스윕 코사인 유사도 0.999)하여 정류 없는 DC형 에너지 출력을 함의. (5) F = −dE/dl이 기계 정밀도까지 수치 검증되어 이산 게이지-기하 운동 방정식(정리 2, Kwon 2026a)을 확인. 더 나아가 해석적으로 도출한다: (i) k=0 기하적 응축이 플라켓 가중치에 적용된 AM-GM 부등식으로부터 따르며, (ii) 게이지-기하 결합이 탄성 저항을 초과할 때 스케일 인자 a(t) ~ exp(Ht)의 지수적(드 시터) 팽창이 따른다. 이 결과들은 다중 게이지 결합이 공유 기하 위에서 결맞고, 방향적이고, 초가법적인 힘을 생성함을 확립 — 논문 D의 스칼라 초가법성 결과를 벡터 영역으로 확장한다.

---

## 1. 서론

힘 정렬 질문에 대한 답은 하나의 관찰에 담겨 있다: 모든 장별 힘이 같은 기하적 도함수 ∂w_P/∂w_e = w_P/(2w_e)를 통해 작용하며, 이것은 장이 아닌 기하에만 의존한다. 모든 장이 같은 채널을 통해 민다. 따라서 모든 장이 같은 방향으로 민다. 이 논문의 나머지는 이 관찰을 정량적으로 검증한다.

동반 논문(Kwon 2026d, 이하 논문 D)은 N개 게이지장이 역학적 격자를 공유할 때 기하적 변형 에너지 E가 초가법적으로 E ~ N^γ (γ = 1.8 ± 0.2)로 스케일함을 확립했다. 그러나 에너지는 스칼라 — 크기는 있지만 방향은 없다. 기하가 *어디서* *어떤 방향으로* 변형하는지의 질문은 열려 있었다.

이산 게이지-기하 운동 방정식(정리 2, Kwon 2026a)이 다리를 제공한다: 각 에지 e에 대한 힘은 F_e = −δE/δl_e다. 에너지가 초가법적이면 힘도 초가법적인가? 다수 장이 기하를 밀 때, 같은 방향으로 미는가 아니면 상쇄하는가?

## 2. 모형과 방법

### 2.1 힘 정의

논문 D와 동일한 격자 게이지-기하 모형을 사용: 2D L×L 주기적 격자 위의 N개 독립 U(1) 게이지장 {θ^{(i)}}과 역학적 에지 가중치 {w}. 총 작용:

S = −Σ_{i=1}^{N} β Σ_P w_P cos Θ_P^{(i)} + κ Σ_e (w_e − 1)²

이것은 Regge 관례의 Hadamard 결합 작용이며, 게이지 항 −βw_P cos Θ_P가 기하적 구조(w_P)와 게이지 내용(cos Θ_P)의 단일 곱이다.

에지 e에 대한 힘은 중심 유한 차분으로 계산: F_e = −[E(w_e + δ) − E(w_e − δ)] / (2δ), δ = 10⁻⁴.

장별 힘 f_e^{(i)}(장 i 기여의 기울기)과 총 힘 F_e(탄성 항 포함 전체 기울기)를 구별한다.

**힘 분해.** 각 장별 힘은 f^{(i)} = g^{(i)} + e로 분해되며, g^{(i)} = −∂[−βw_P cosΘ_P^{(i)}]/∂w_e는 장 i의 게이지만 기울기이고, e = −2κ(w_e−1)은 모든 장이 공유하는 탄성 기울기다.

**검증.** 탄성 항을 완전히 배제한 게이지만 힘 g^{(i)}으로 정렬을 계산했다. 결과는 정렬이 진정한 게이지-기하 효과임을 확인:

| N | 게이지만 ⟨cosθ⟩ | 전체 ⟨cosθ⟩ | 게이지만 정렬 |
|---|-----------------|-------------|--------------|
| 2 | 0.963 | 0.961 | 99.1% |
| 3 | 0.978 | 0.973 | 99.3% |
| 5 | 0.996 | 0.995 | 99.9% |
| 10 | 0.999 | 0.999 | 99.9% |

게이지만 정렬이 전체 정렬과 같거나 더 높다. 탄성 항이 정렬을 부풀리지 않는다 — 게이지-기하 결합과 무관한 공통 성분을 더해 약간 희석한다. >99% 정렬은 전적으로 공유 플라켓 구조에 기인: 모든 게이지 기울기 g^{(i)}가 같은 ∂w_P/∂w_e를 통해 작용하며 이것은 기하에만 의존한다.

### 2.2 측정

(a) 장간 정렬: cos θ_{ij} = (f^{(i)} · f^{(j)}) / (|f^{(i)}| |f^{(j)}|), 힘 배열을 R^{2L²} 벡터로 취급.
(b) 정렬 비율: |F_net| / Σ|f^{(i)}|. 완전 정렬에서 1, 무작위에서 1/√N.
(c) 힘-변형 상관: 모든 에지에 걸친 |F_e|와 |w_e − 1| 사이 피어슨 r.
(d) 시간적 안정성: 연속 메트로폴리스 스윕 사이 cos(F(t), F(t+1)).
(e) |F_net|의 N-스케일링.

### 2.3 매개변수

L = 8 (2D), β = 2.0, κ = 3.0. 열화: 100 스윕. 측정: 100 스윕. N = 1에서 10. 결정론적 시드.

## 3. 결과

### 3.1 독립 장 간 힘 정렬

표 1. 장간 힘 정렬 (β = 2.0, κ = 3.0, L = 8)

| N | ⟨cos θ_{ij}⟩ | 정렬 비율 |
|---|--------------|----------|
| 1 | 1.000 (정의) | 100.0% |
| 2 | 0.964 | 99.1% |
| 3 | 0.979 | 99.3% |
| 5 | 0.997 | 99.9% |
| 7 | 0.998 | 99.9% |
| 10 | 0.999 | 100.0% |

독립 장의 힘이 >99% 정렬되며, 정렬이 N과 함께 단조 향상. 물리적 메커니즘: 모든 장이 같은 에지 가중치를 통해 기하에 작용하고, 플라켓 가중치 구조(주변 에지의 기하 평균)가 개별 장 배위가 아닌 국소 기하에 의해 결정되도록 힘 방향을 구속한다. 기하가 다양한 게이지 압력을 공통 방향으로 깔때기하는 역할을 한다.

### 3.2 힘-변형 반상관

|F_e|와 |w_e − 1| 사이 피어슨 상관이 N에 의존:

| N | r(|F|, |w-1|) |
|---|---------------|
| 1 | +0.40 (양) |
| 2 | −0.57 |
| 3 | −0.80 |
| 5 | −0.73 |
| 7 | −0.69 |
| 10 | −0.74 |

N = 1에서 힘이 변형이 가장 큰 곳에 집중(양의 피드백). N ≥ 2에서 힘이 변형이 가장 작은 곳에 집중(파면). 이것은 점진적 변화가 아닌 정성적 전이: 상관의 부호가 N = 1과 N = 2 사이에서 뒤집힌다. 단일 장은 기하를 국소적으로 파고들고; 다수 장은 전파하는 전선을 만든다.

힘이 기하가 *가장 덜* 변형된 곳에서 가장 강하다. 이미 변형된 에지는 결합 압력에 이미 응답했고; 잔여 힘이 작다. 평평한 에지(w ≈ 1)가 완전한 결합 기울기를 담당한다.

이것은 전파하는 파면을 생성: 변형이 한 영역에서 시작하고, 거기서 힘이 감소하며, 이웃한 평평한 영역의 힘이 증가하여 변형을 바깥으로 끌어당긴다. 기하적 변형이 국소 축적 대신 전선으로 퍼진다.

### 3.3 초가법적 힘 스케일링

표 2. 순 힘 크기 대 N (β = 2.0, κ = 3.0, L = 8)

| N | |F_net| | N=1 대비 비율 |
|---|--------|-------------|
| 1 | 27.1 | 1.0× |
| 2 | 39.8 | 1.5× |
| 3 | 67.4 | 2.5× |
| 5 | 166.3 | 6.1× |
| 7 | 313.8 | 11.6× |
| 10 | 575.1 | 21.2× |

멱법칙 적합: |F_net| ~ N^(1.69 ± 0.06), R² = 0.997.

힘 지수(γ_F = 1.69)가 에너지 지수(γ_E = 1.8 ± 0.2)와 불확실성 이내에서 정합적. F_e = −∂E_total/∂w_e이고 E_total ~ N^γ이며 w_e는 N과 무관(기하는 공유, 복제 아님)하므로 ∂E_total/∂w_e ~ N^γ이 지수를 보존하기 때문.

### 3.4 힘 방향의 시간적 안정성

연속 스윕 코사인 유사도: ⟨cos(F(t), F(t+1))⟩ = 0.9991 ± 0.0006.

힘 방향이 시간에 본질적으로 상수. 크기는 요동(변동 계수 0.167)하지만 방향은 아니다. 에지 치환 대칭의 자발적 파괴: 평평한 기하에서 모든 에지가 동등하지만, 변형이 열적 잡음에도 지속하는 에지 가중치 공간의 선호 패턴을 선택한다.

### 3.5 F = −dE/dl의 수치 검증

무작위 선택된 10개 에지에서 에지별 중심 차분(δ = 10⁻⁴)으로 계산된 힘을 전체 에너지 차분(δ = 10⁻⁵)과 비교. 모든 에지에서 상대 불일치 < 10⁻⁴. 정리 2를 수치 확인.

## 4. 해석적 도출

### 4.1 AM-GM 부등식으로부터의 k = 0 응축

플라켓 가중치 w_P = (w₁ w₂ w₃ w₄)^{1/2}는 주변 에지 가중치의 기하 평균이다. AM-GM 부등식에 의해:

(w₁ w₂ w₃ w₄)^{1/4} ≤ (w₁ + w₂ + w₃ + w₄) / 4

등호 조건: w₁ = w₂ = w₃ = w₄.

Regge 관례에서 게이지 항 −β w_P cos θ_P는 w_P가 클 때 더 음수(낮은 S)이다(cos θ > 0일 때). 메트로폴리스는 더 낮은 S를 선호하므로 더 큰 w_P를 선호. AM-GM에 의해 w_P는 모든 에지 가중치가 같을 때 — k = 0(균일) 모드 — 최대화된다.

따라서 게이지 작용이 균일 모드로의 기하적 응축을 강제한다. 이것은 기하 평균 구조에서 발생하는 수학적 필연성이지 역학적 우연이 아니다. 탄성 항 κΣ(w_e−1)²도 균일 가중치를 선호(모든 e에서 w = 1로 최소화)하므로, 작용의 두 항 모두 공간적 균일성에 합의한다. 이들은 균일 가중치의 *값*에서만 경쟁: 게이지 결합이 a > 1(팽창)을 구동하고 탄성 저항이 a = 1(평평함)으로 끌어당긴다.

몬테카를로 검증: k = 0 모드가 스윕 0에서 스펙트럼 파워의 26%를 차지하고 스윕 400에서 99.9%를 차지.

함의: 변형장의 공간적 균일성이 인플레이션이나 특수 초기 조건 없이 발생. 게이지 작용의 기하 평균이 충분하다.

### 4.2 초과 결합으로부터의 지수적 팽창

균일 모드 w_e = a(스케일 인자)에 대해 작용이 다음으로 환원:

S(a) = −Nβ n_P a² ⟨cos θ⟩ + κ n_E (a − 1)²

c ≡ ⟨cos θ⟩(평균 플라켓 코사인, β > 0에서 양)를 게이지 요동 강도로 정의하고, n_P와 n_E는 사이트당 플라켓과 에지 수(2D에서 n_P = 1, n_E = 2).

∂S/∂a = 0을 설정:

a_eq = κ n_E / (κ n_E − Nβ n_P c)

Nβ n_P c > κ n_E이면 분모가 음수, 유한 a에서 평형이 존재하지 않으며, 과감쇠 운동 방정식 da/dt ∝ −∂S/∂a가 산출:

da/dt = (δ / γ_eff) · a,  δ ≡ Nβ c n_P − κ n_E > 0

해: a(t) = a₀ exp(Ht), H = δ / γ_eff. 드 시터 팽창이다.

몬테카를로 검증: ⟨w⟩(t)가 exp(0.0035t)에 R² = 0.983으로 적합, 멱법칙(R² = 0.978)을 능가. 지수적 형태는 수치 적합에만 의존하지 않는다: 균일 모드 축소의 da/dt ∝ a에서 해석적으로 따르며, MC와 독립적이다.

### 4.3 자기정합적 임계 결합

§4.2로부터 임계 결합:

β_c = κ n_E / (N n_P c(β_c))

이것은 c(β) = 1 − I₁(β)/I₀(β) (U(1)에 대해)이므로 자기정합적이다. N = 3, κ = 3.0에 대한 수치 해는 β_c ≈ 1.4를 주며, 논문 D의 β_c ≈ 1.3–1.5 몬테카를로 측정과 정합적.

### 4.4 초가법성 지수의 차원 의존성

d차원에서 각 에지가 d 링크 방향에 걸쳐 2(d−1) 플라켓에 참여한다. 기하적 좌절 밀도는 ρ(d) = 2(d−1)/d. γ(2) = 1.8, γ(3) = 1.44, γ(4) = 0.95를 ρ(d)에 대해 적합:

γ(d) ≈ −1.6 · ρ(d) + 3.5,  R² = 0.92

이것은 γ = 1인 임계 차원 d_c ≈ 5를 예측 — 초가법성이 소멸. 우리 우주(d = 4)가 이 경계 근처에 있으며, 중력이 가장 약한 힘인 것과 정합적. 이 예측은 반증 가능: d = 5에서의 MC 시뮬레이션이 γ(5) ≈ 1을 확인하거나 선형 추세를 반박할 것이다.

## 5. 논의

### 5.1 힘이 에너지에 더하는 것

에너지는 "얼마나"를 말한다. 힘은 "다음에 어디로"(§3.2, 반상관), "함께"(§3.1, 정렬), "지속적으로"(§3.4, 안정성)를 말한다. 스칼라(논문 D)에서 벡터(이 논문)로의 전환이 에너지만으로는 숨겨진 역학적 구조를 드러낸다.

### 5.2 활성 매체로서의 기하

직접적 장간 결합이 영임에도 99% 힘 정렬은 기하가 독립 장을 능동적으로 조직함을 의미한다. 이것은 논문 D의 변위 정렬(α ≈ 0.94)의 벡터 수준 확인이며, 더 강하다: 힘 정렬이 변위 정렬을 초과한다.

### 5.3 한계와 확장

모든 힘 결과가 L ≤ 10의 2D 격자로부터. 논문 B의 유한 크기 스케일링(4D SU(2)에서 L = 4, 6, 8)이 기저 게이지-기하 상관이 벌크 효과임을 확인; 힘 정렬과 초가법성 지수에 대한 유사한 스케일링이 자연스러운 다음 단계. 해석적 도출(§4)은 구조에서 차원 무관하지만 정량적 예측에는 4D 검증이 필요.

### 5.4 이중섹터 구조와의 연결 및 4D 검증

동반 논문 [Kwon 2026h]이 유효 작용의 헤시안이 k-의존 고유값을 가짐을 증명: 단파장 모드에서 양(안정), 장파장 모드에서 음(불안정). 교차 파수 k*가 입자 섹터(유질량, 국소화)를 우주론 섹터(팽창, 구조 형성)에서 분리한다. 이 논문의 힘 결과는 우주론 섹터(k < k*)에서 작동: 순 힘 |F_net|이 장파장 기하적 변형을 구동하고, 정렬, 반상관, 시간적 안정성이 이 구동의 결맞음 구조를 기술한다.

**4D 검증.** 핵심 힘 예측이 4D U(1) L = 3 주기적 격자(N = 3, β = 2.0, κ = 3.0)에서 확인: 게이지만 원시 정렬 99.94%, 공간 패턴 정렬 0.82(14.8σ), k = 0 응축 98.2%, 계속되는 팽창(⟨w⟩ = 1.0 → 2.19). 세 기둥 — 정렬, 응축, 팽창 — 모두 4D에서 존속.

## 6. 결론

공유 기하 위의 다중 게이지 결합은 다음의 힘을 생성한다:
(i) 정렬 (>99%, N과 함께 향상)
(ii) 전파 (파면, 변형과 반상관)
(iii) 초가법적 (|F| ~ N^1.69)
(iv) 방향 안정 (cos = 0.999)
(v) 정확히 기울기 (F = −dE/dl 검증)

해석적 도출이 보인다:
(vi) k = 0 응축 = AM-GM 필연
(vii) 드 시터 팽창 = 초과 결합
(viii) β_c = 자기정합 공식
(ix) γ(d) = 좌절 밀도 함수, d_c ≈ 5

이것이 T = C⊙S 프레임워크를 완성한다: 존재(논문 A,B), 크기(논문 D), 방향(이 논문).
