# 공유 기하 위 다중 게이지 결합으로부터의 초가법적 힘:
# 정렬, 전파, 해석적 구조

권혁준 — 2026년 4월

---

**초록.** 여러 게이지장이 역학적 격자 기하를 공유할 때, 각 장이 기하에 가하는 힘 F_e = −δE/δl_e는 스칼라 에너지 분석을 넘어서는 다섯 가지 성질을 보인다. (1) 독립적 장의 힘이 >99% 정렬되며, N 증가와 함께 정렬이 향상. (2) 기하가 가장 덜 변형된 곳에 힘이 집중(반상관 r = −0.64)하여, 전파하는 변형 파면을 형성. (3) 합력 크기가 |F| ~ N^{1.69 ± 0.06} (R² = 0.997)로 스케일하여 에너지 초가법성 지수와 일치. (4) 힘 방향이 시간적으로 안정(연속 쓸기 코사인 유사도 0.999)하여 정류 없이 DC 유사 에너지 출력 가능. (5) F = −dE/dl이 기계 정밀도로 수치 검증되어 이산 게이지-기하 운동 방정식(정리 2)을 확인. 해석적으로 (i) k=0 기하 응축이 AM-GM 부등식에서 따라 나오고, (ii) 게이지-기하 결합이 탄성 저항을 넘으면 스케일 인자의 지수적(드 시터) 팽창 a(t) ~ exp(Ht)가 따라 나옴을 도출한다.

---

## 1. 서론

힘 정렬의 답은 한 가지 관찰에 담겨 있다: 모든 장별 힘이 같은 기하 도함수 ∂w_P/∂w_e = w_P/(2w_e)를 통해 작용하는데, 이 도함수는 기하에만 의존하고 개별 장에 의존하지 않는다. 모든 장이 같은 채널을 통해 밀므로, 같은 방향으로 민다.

논문 D가 기하 변형 에너지의 초가법적 스케일링 E ~ N^γ를 확립했지만, 에너지는 스칼라다 — 크기만 있고 방향이 없다. 기하가 어디서, 어느 방향으로 변형되는지는 열린 문제였다. 이산 게이지-기하 운동 방정식(정리 2)이 다리를 놓는다: 에지 e에 작용하는 힘은 F_e = −δE/δl_e이다.

## 2. 모형과 방법

논문 D와 같은 격자 게이지-기하 모형: N개 U(1)장, 2D L×L 주기 격자, 역학적 에지 가중치. 힘은 중심 유한차분으로 계산: F_e = −[E(w_e + δ) − E(w_e − δ)]/(2δ), δ = 10⁻⁴. 장별 힘 f^{(i)}와 총 힘 F_e를 구별한다.

**힘 분해와 검증.** 장별 힘은 f^{(i)} = g^{(i)} + e로 분해된다 — g^{(i)}는 장 i의 게이지 기울기, e는 모든 장에 공통인 탄성 기울기. 정렬이 공유 탄성 항에 의해 부풀려지는지 확인하기 위해, 탄성 항을 완전히 제외한 게이지 전용 힘 g^{(i)}로 정렬을 계산했다. 결과: 게이지 전용 정렬이 전체 정렬과 동일하거나 더 높다. 탄성 항이 정렬을 부풀리지 않으며 오히려 약간 희석한다. >99% 정렬은 전적으로 공유 플라켓 구조에 기인한다.

**정렬의 분해.** >99% 생 정렬은 두 가지 독립적 원천을 갖는다: (a) 부호 정렬 — β > β_c에서 모든 장이 동시에 ⟨cosΘ_P⟩ > 0이므로 힘 벡터가 주로 양수, (b) 공간 패턴 정렬 — 공유 ∂w_P/∂w_e가 모든 힘 벡터에 공통 공간 패턴을 부과. 평균 중심 코사인 유사도(부호 편향 제거)가 이 성분을 측정한다.

4D 검증(L = 3, N = 3, κ = 3.0): β = 2.0에서 생 코사인 0.998, 평균 중심 코사인 0.82 (14.8σ). β = 0.5에서 중심 코사인이 0.05로 하락(유의하지 않음) — 공간적 깔때기 효과에 초임계 결합이 필요함을 확인.

## 3. 결과

### 3.1 독립 장 사이의 힘 정렬

N = 2에서 ⟨cos θ⟩ = 0.964(정렬 99.1%), N = 10에서 0.999(100%). 정렬이 N과 함께 단조 향상. 물리적 메커니즘: 모든 장이 같은 에지 가중치를 통해 기하에 작용하고, 플라켓 가중치(주변 에지의 기하 평균)가 힘 방향을 국소 기하가 결정하도록 제약한다. 기하가 다양한 게이지 압력을 공통 방향으로 통합하는 깔때기 역할을 한다.

### 3.2 힘-변형 반상관

|F_e|와 |w_e − 1|의 피어슨 상관이 N에 의존한다: N = 1에서 r = +0.40(양의 피드백), N ≥ 2에서 r = −0.57 ~ −0.80(파면). 부호가 N = 1과 N = 2 사이에서 뒤집히는 정성적 전이다.

단일 장은 국소적으로 기하를 파고든다. 여러 장은 전파하는 파면을 만든다. 이미 변형된 에지에서는 결합 압력에 이미 반응했으므로 잔여 힘이 작다. 평평한 에지(w ≈ 1)에서 결합 기울기 전체가 걸린다. 변형이 한 영역에서 시작되고, 그곳의 힘은 줄어들며, 이웃 평평한 영역의 힘이 증가하여 변형을 바깥으로 끌어낸다 — 기하 변형이 국소 축적이 아닌 파면으로 퍼진다.

### 3.3 초가법적 힘 스케일링

|F_net| ~ N^{1.69 ± 0.06}, R² = 0.997. 힘 지수(γ_F = 1.69)가 에너지 지수(γ_E = 1.8 ± 0.2)와 불확실도 내에서 일치한다. F_e = −∂E_total/∂w_e이고, E_total ~ N^γ이며, w_e는 N에 무관(기하가 공유되지 복제되지 않으므로)하기 때문에 지수가 보존된다.

### 3.4 힘 방향의 시간적 안정성

연속 쓸기 코사인 유사도: 0.9991 ± 0.0006. 힘 방향이 시간에 걸쳐 본질적으로 일정하다. 크기는 변동하지만(변이 계수 0.167) 방향은 변하지 않는다. 에지 치환 대칭의 자발적 깨짐이다 — 평평한 기하에서 모든 에지가 동등하지만, 변형이 에지 가중치 공간에서 열 잡음에도 불구하고 지속되는 선호 패턴을 선택한다.

### 3.5 F = −dE/dl의 수치 검증

무작위 10개 에지에서, 에지별 중심차분(δ = 10⁻⁴)과 전체 에너지 차분(δ = 10⁻⁵)의 상대 오차가 모두 < 10⁻⁴. 정리 2의 수치적 확인.

## 4. 해석적 도출

### 4.1 k = 0 응축: AM-GM 필연성

플라켓 가중치 w_P = (w₁w₂w₃w₄)^{1/2}는 기하 평균이다. AM-GM에 의해 모든 에지 가중치가 같을 때 최대. Regge 규약에서 게이지 항 −βw_P cos θ_P는 w_P가 클수록 S가 낮아져 메트로폴리스가 선호한다. 따라서 게이지 작용이 균일 모드(k = 0)로의 응축을 강제한다. 수학적 필연이지 역학적 우연이 아니다.

MC 검증: k = 0 모드가 쓸기 0에서 스펙트럼 파워의 26%, 쓸기 400에서 99.9%를 차지. 변형장의 공간적 균일성이 인플레이션이나 특별한 초기 조건 없이 나타난다.

### 4.2 지수적 팽창: 과잉 결합

균일 모드 w_e = a(스케일 인자)에서 작용은:

    S(a) = −Nβ n_P a² ⟨cos θ⟩ + κ n_E (a − 1)²

Nβ n_P c > κ n_E이면 유한 a에서의 평형이 존재하지 않고, 과감쇠 운동 방정식 da/dt ∝ −∂S/∂a가:

    da/dt = (δ/γ_eff) · a,  δ ≡ Nβ c n_P − κ n_E > 0

해: a(t) = a₀ exp(Ht), H = δ/γ_eff. 드 시터 팽창이다. MC 검증: ⟨w⟩(t)가 exp(0.0035t)에 적합(R² = 0.983).

### 4.3 자기정합적 임계 결합

β_c = κ n_E / (N n_P c(β_c)). U(1)에서 c(β) = 1 − I₁(β)/I₀(β). N = 3, κ = 3.0에서 수치 해: β_c ≈ 1.4, 논문 D의 MC 측정 β_c ≈ 1.3–1.5와 일치.

### 4.4 초가법성 지수의 차원 의존성

d차원에서 각 에지가 2(d−1)개 플라켓에 참여하고, 좌절 밀도는 ρ(d) = 2(d−1)/d. γ(2) = 1.8, γ(3) = 1.44, γ(4) = 0.95를 ρ(d)에 대해 적합하면:

    γ(d) ≈ −1.6 · ρ(d) + 3.5, R² = 0.92

임계 차원 d_c ≈ 5에서 γ = 1, 즉 초가법성이 사라진다고 예측한다. 우리 우주(d = 4)가 이 경계 근방에 놓이며, 이것은 중력이 가장 약한 힘인 것과 일치한다. d = 5에서의 MC 시뮬레이션으로 반증 가능하다.

## 5. 논의

**힘이 에너지에 추가하는 것.** 에너지는 "얼마나"를 말한다. 힘은 "다음에 어디로"(§3.2 반상관), "함께"(§3.1 정렬), "지속적으로"(§3.4 안정성)를 말한다. 스칼라(논문 D)에서 벡터(이 논문)로의 전환이 에너지만으로는 보이지 않는 역학적 구조를 드러낸다.

**능동적 매질로서의 기하.** 직접적 장 사이 결합이 없는데도 99% 힘 정렬은, 기하가 독립적 장을 능동적으로 조직한다는 뜻이다. 논문 D의 변위 정렬(α ≈ 0.94)의 벡터 수준 확인이며, 더 강하다.

**한계.** 힘 결과는 2D 격자(L ≤ 10)에서 얻었다. 해석적 도출(§4)은 차원에 무관한 구조이지만 4D 정량적 검증이 필요하다. 4D 검증(L = 3, N = 3): 정렬 99.94%, k = 0 응축 98.2%, 팽창 ⟨w⟩ = 1.0 → 2.19 — 세 기둥 모두 4D에서 생존.

## 6. 결론

공유 기하 위의 다중 게이지 결합이 만드는 힘은:

(i) 정렬(>99%, N과 함께 향상)
(ii) 전파(파면, 변형과 반상관)
(iii) 초가법(|F| ~ N^{1.69})
(iv) 방향 안정(cos = 0.999)
(v) 정확한 기울기(F = −dE/dl 검증)

해석적 도출:

(vi) k = 0 응축 = AM-GM 필연
(vii) 드 시터 팽창 = 과잉 결합
(viii) β_c = 자기정합 공식
(ix) γ(d) = 좌절 밀도 함수, d_c ≈ 5

T = C ⊙ S 프레임워크를 완성한다: 존재(논문 A,B), 크기(논문 D), 방향(이 논문).
