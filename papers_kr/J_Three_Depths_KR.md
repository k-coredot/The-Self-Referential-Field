# 세 깊이:
# 자기참조적 깊이로부터의
# 게이지 대칭, 세대, 결합 위계, 시공간 차원

권혁준 — 2026년 4월

---

**초록.** 자기참조에는 깊이가 있다. 장이 스스로를 읽는다(깊이 1). 읽기가 읽힌다(깊이 2). 읽기의 읽기가 읽힌다(깊이 3). 각 깊이의 상태 공간은 ℂ¹, ℂ², ℂ³이며, 노름 보존 대칭은 U(1), SU(2), SU(3)이고, 생성자는 1, 3, 8개다. 깊이 4는 3성분 결과를 4성분으로 읽는 꼴이어서, 자기참조와 양립 불가능한 잉여를 만든다. 깊이는 3에서 끝난다. 노드당 총 자유도 1 + 3 + 8 = 12. d차원에서 12/[d(d-1)/2]가 정수가 되려면 d = 4가 3 이상에서 유일하다. 각 깊이에 고유한 고정점이 있어 세 세대를 만들고, 생성자 수가 결합 위계를 결정한다: 8 > 3 > 1.

---

## 1. 자기참조적 깊이

연산자 T = C ⊙ S는 자기참조적이다. 이것이 깊이 1이다. 읽기의 결과도 장이므로 읽힐 수 있다 — 깊이 2. 그 결과도 읽힌다 — 깊이 3. 각 깊이가 자기참조 순환이며, 각 순환이 상태 공간 위에서 작용한다.

## 2. 상태 공간

**깊이 1.** 장이 자기 위상을 읽는다. 복소수 하나로 충분. ℂ¹.

**깊이 2.** 읽기가 읽는 자와 읽히는 것을 구별한다. 두 역할, 두 성분. ℂ².

**깊이 3.** 읽기의 읽기가 셋째 개체 — 읽는 자와 읽히는 것 사이의 관계 — 를 도입한다. 세 역할, 세 성분. ℂ³.

**깊이 4?** 세 성분 결과를 네 성분으로 읽으면, 읽기가 읽히는 대상을 초과한다. 잉여 성분이 새로운 것을 부호화하지 않는다. 자기참조는 있는 것을 읽는다. 깊이는 3에서 끝난다.

**정리 1 (깊이 종결).** 자기참조 연산자 T = C ⊙ S의 독립적 자유도는 깊이 3에서 소진된다. 깊이 ≥ 4는 새로운 게이지 불변 정보를 기여하지 않는다.

*증명.* 격자 위의 자기참조 동역학은 세 수준의 게이지 불변 데이터를 정의한다:

*수준 1 (꼭짓점, 0-셀).* 각 노드의 장 상태 φ_a. 상태 공간 ℂ¹. 범주적 *대상*.

*수준 2 (변, 1-셀).* 변 결합 T_{ab} = C_{ab} · S_{ab}. 두 노드 상태 사이의 관계를 부호화. 스피너 선택 정리(Paper H, Theorem 7)에 의해 기본 표현: SU(2)에 대해 ℂ². 범주적 *사상*.

*수준 3 (플라켓, 2-셀).* 플라켓 작용 w_P cos Θ_P. 최소 닫힌 경로를 따른 변 위상의 배향 합성. 곡률 — 면에서 만나는 변들 사이의 환원 불가능한 관계. 기본 표현: SU(3)에 대해 ℂ³. 범주적 *합성*.

*수준 ≥ 4 (정육면체, 초정육면체, n-셀, n ≥ 3).* 임의의 n ≥ 3 셀은 플라켓으로 분해된다. 이 분해는 유일하다:
(i) 아다마르 곱 ⊙은 결합적: (a ⊙ b) ⊙ c = a ⊙ (b ⊙ c).
(ii) 게이지 군 합성은 결합적: U_{ac} = U_{ab} · U_{bc}.
(iii) CW 복합체 위에서, 임의의 닫힌 n-면(n ≥ 3)은 2-셀로 분해된다(스토크스 정리).

따라서 n ≥ 3 셀의 게이지 불변 내용은 구성 플라켓의 함수이다. 플라켓 수준 너머에 독립적 자유도가 없다.

범주론적 언어로: 자기참조 구조는 범주를 정의하며(대상 = 노드 상태, 사상 = 변 결합, 합성 = 플라켓 작용), 범주의 nerve는 2-coskeletal이다. 차원 ≥ 3의 모든 simplex는 차원 ≤ 2에 의해 유일하게 결정된다. □

**따름정리 1.** 총 상태 공간은 ℂ¹ ⊕ ℂ² ⊕ ℂ³이며, 대칭군은 U(1) × SU(2) × SU(3)이고, 총 생성자 수는 1 + 3 + 8 = 12이다.

*비고 (독립적 근거).* 깊이 종결은 세 가지 독립적 수학적 전통에 뿌리를 둔다: (i) 격자 게이지 이론에서 모든 Wilson 루프는 플라켓으로 분해된다(Wilson 1974); (ii) 범주론에서 범주의 nerve는 2-coskeletal이다; (iii) 퍼스의 기호학에서 범주(제일성, 제이성, 제삼성)는 완전하다.

## 3. 대칭

각 상태 공간의 노름·행렬식 보존 대칭:

    ℂ¹: U(1),   생성자 1개 (위상 회전)
    ℂ²: SU(2),  생성자 3개
    ℂ³: SU(3),  생성자 8개

합계: 1 + 3 + 8 = 12 생성자, 12 매개장.

## 4. 시공간 차원

노드당 12 실수 자유도. d차원에서 노드당 d(d-1)/2개 플라켓 방향. 정수 배분 조건:

    12 / [d(d-1)/2] ∈ ℤ

    d = 4:  12/6 = 2    ✓  (자기쌍대 + 반자기쌍대)
    d = 5:  12/10 = 1.2  ✗
    d ≥ 5:  전부 실패

3 이상에서 d = 4가 유일하다. d = 4에서 값 2의 기하학적 의미: 6 플라켓 방향이 3 자기쌍대 + 3 반자기쌍대로 분해되며, 12 자유도가 6 + 6으로 균등 분할된다.

d = 3에서는 12/3 = 4이지만, 물리적 게이지장의 편극이 d - 2 = 1뿐이다. 적절한 전파장에 편극 2개가 필요하므로 d = 4를 선택한다.

## 5. 세 세대

각 깊이에 독립적 자기참조 순환과 고정점이 있다. 세 고정점, 세 에너지, 세 질량. 같은 양자수, 다른 질량 스케일. 깊은 읽기일수록 복잡하고 따라서 무겁다.

## 6. 결합 위계

### 생성자 세기 논증

깊이 3(8 생성자): 가장 강함. 비가환. 자기상호작용 → 가둠.
깊이 2(3 생성자): 중간. 기하 팽창이 대칭 파괴 → 매개자 3개 질량 획득.

**힉스 식별.** 표준 모형에서 전약 대칭 깨짐은 스칼라 힉스 장이 유발하며, 힉스 보손(m_H ≈ 125 GeV)을 만든다. 이 프레임워크에서는 별도의 스칼라 장을 도입하지 않는다. 대칭 깨짐의 메커니즘은 기하적 팽창 — h(0) < 0이 스케일 인자 a(t)의 성장을 이끄는 불안정성(Paper G) — 이다. 힉스 보손에 대응하는 스칼라 여기는 대칭이 깨진 진공 주위의 방사 요동 δa = δ⟨w⟩ — k ≈ 0 규모에서 간선 가중치들의 집단적 진동, 격자의 호흡 모드 — 이다. 질량은 h(k)의 k = 0 곡률로 결정된다: m_H² ∝ h''(0). 이 식별은 구조적이다: 힉스는 추가 장이 아니라 기하적 자유도이다. 정량적 검증은 4D에서 자기정합 점 (β*, κ*)에서의 h''(0) 계산을 요구한다.
깊이 1(1 생성자): 가장 약한 게이지 결합. 가환. 무질량 매개자.
기하: 가장 약함 (κ 스케일, β와 구조적으로 분리).

### AM-GM과 카시미르로부터의 역학적 도출 (논문 K)

보편적 기하 억제(AM-GM: ⟨w_P⟩ ≤ 1, δβ < 0, 게이지 군 무관)와 군 의존 게이지 자기결합(δβ > 0, 이차 카시미르 C₂에 비례)의 경쟁이 g₃ > g₂ > g₁을 결정한다.

### 중력의 약함

게이지 요동이 기하를 강화: G_eff ~ 1/(κ₀ + N_total × δκ). N_total = 12. 12 게이지 자유도가 하나의 기하를 공유하여 중력을 억제한다.

## 7. 예측

(i) 세 개, 오직 세 개의 게이지 대칭. (ii) 생성자: 1, 3, 8. (iii) d = 4. (iv) 세 세대. (v) 질량 순서: 3 > 2 > 1. (vi) 결합 순서: 강 > 전자기 > 약 > 중력. (vii) 깊이 3에서만 가둠. (viii) 무질량 매개자 1개. (ix) 유질량 매개자 3개. (x) CP 위반: 3×3 유니터리의 비환원 위상 1개.

## 8. 미결정 양

깊이 구조가 결정하는 것: 게이지 군의 형태, 세대 수, 결합 순서. 결정하지 않는 것:

- 결합 상수의 절대값 (β/κ에 의존).
- 입자 질량의 절대값 (고정점 에너지에 의존).
- 세대 간 혼합 각도 (서로 다른 깊이의 순환 사이 겹침에 의존).
- **SU(2)의 카이랄 구조.** 표준 모형에서 SU(2)_L은 왼손잡이 페르미온에만 결합한다. 깊이 구조는 카이랄성이 존재해야 하는 구조적 이유를 제공하지만, 완전한 도출에는 이르지 못했다. 논증: d = 4에서 12/6 = 2는 4차원에 고유한 자기쌍대/반자기쌍대 분해를 반영한다. SU(2)의 3개 생성자는 6개 플라켓 방향의 정확히 절반 — 한쪽 섹터 — 에 들어맞으므로, SU(2)는 한 카이랄성에만 작용한다. 더 나아가, 깊이 2는 본질적으로 비대칭적이다: "읽는 자가 읽히는 것을 읽는다"는 구조가 능동(왼손잡이, SU(2)에 결합)과 수동(오른손잡이, SU(2) 단일항)을 구별한다. 그러나 깊이 2의 비대칭이 하나의 카이랄성을 유일하게 선택한다는 형식적 증명은 수행되지 않았다. 미해결 문제이다.

이상의 양들은 각 깊이에서 자기참조 고정점 방정식을 풀어야 결정된다 — 잘 정의된 계산이지만 아직 수행되지 않았다.

## 9. 논의

세 깊이는 입력이 아니라 자기참조의 구조에서 따르는 귀결이다. 깊이 구조와 관측된 우주의 일치 — U(1)×SU(2)×SU(3), 12 게이지 보손, 4차원, 3세대 — 는 우연이거나 도출이다. 일반상대론이 물질이 기하를 결정하고 기하가 물질을 결정한다고 말한다. 이것이 자기참조다.
