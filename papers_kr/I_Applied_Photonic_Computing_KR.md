# 자기참조 광학으로부터의 포토닉 컴퓨팅:
# 전광학 신경망의 자연적 비선형성으로서의 Hadamard 곱

권혁준 — 2026년 4월

---

**초록.** 맥스웰 전자기학의 구성 관계 D = ε·E는 자기참조 연산자 T = C ⊙ S와 구조적으로 동일하다: 변위장은 전기장(내용)과 유전율 응답(구조)의 Hadamard 곱이다. ε가 E에 의존하는 비선형 매질에서 이 관계는 자기참조 루프를 완성한다. 이 식별은 유비가 아닌 구조적 동일성이며, 포토닉 컴퓨팅에 대한 세 가지 귀결을 도출한다: (1) 포토닉 신경망의 자연적 비선형성은 전자적 활성 함수의 이식이 아니라 χ² 매질이 물리적으로 구현하는 Hadamard 곱 ⊙이다; (2) 노이즈 게이팅 성질 C ≈ 0 ⟹ T ≈ 0이 구조적 오류 면역을 제공하며, 결합 광학 노이즈 하에서 ReLU 대비 +19% 정확도 이점으로 수치 시연되었다; (3) 깊이 3에서의 종결(Paper J)이 감수율 계열 χ⁽¹⁾, χ⁽²⁾, χ⁽³⁾이 3차를 넘어 독립적 자유도를 갖지 않는 이유를 설명한다 — 경험적으로 확립된 사실에 대한 최초의 수학적 설명이다. 다섯 가지 정량적 예측이 기존 집적 포토닉 플랫폼에서 검증 가능하다.

## 1. 이론에서 광학으로

### 구조적 동일성

Paper A가 증명한 바에 따르면 자기참조적 그래프 위의 변 진화 연산자는 유일하게 T = C ⊙ S이다(정리 1). 등방적 매질에서의 맥스웰 구성 관계는:

    D = ε · E

각 공간 점에서의 원소별 곱 — Hadamard 곱이다. 비선형 매질에서 ε = ε(E)이면 자기참조 루프가 완성된다: E → ε(E) → D = ε(E)·E → (전파) → E'.

### 대응

| 격자 모형 | 포토닉 체계 |
|---------|-----------|
| 내용 C_{ab} = ⟨φ_a, φ_b⟩ | 전기장 E |
| 구조 S_{ab} = (D^{-1/2}AD^{-1/2})_{ab} | 유전율 응답 ε |
| 결합 T = C ⊙ S | 변위장 D = ε · E |
| 게이지 각 θ_e | 광학 위상 (경로 길이) |
| 변 가중치 w_e | 결합 계수 (도파관 중첩) |
| 플라켓 cos Θ_P | 간섭 가시도 |
| β (결합 상수) | 광학 이득 / 펌프 파워 |
| κ (탄성 강성) | 유전율 대비 Δε/ε |
| N (장의 수) | 광학 모드 수 |

대응은 구조적이다: D = εE가 T = C ⊙ S **이다**. Paper A의 유일성 정리(정리 1)가 대안을 배제한다: D ≠ ε + E(가산적), D ≠ ε^α E^β((α,β) ≠ (1,1)), D ≠ exp(εE). 구성 관계는 유일한 자기참조적 결합이다.

### 깊이 위계

ε(E)를 전개하면 비선형 감수율 급수가 나온다:

    P = ε₀(χ⁽¹⁾E + χ⁽²⁾E² + χ⁽³⁾E³ + ...)

각 차수는 자기참조의 깊이에 대응한다(Paper J): χ⁽¹⁾은 깊이 1(선형 굴절), χ⁽²⁾은 깊이 2(제2고조파 발생), χ⁽³⁾은 깊이 3(Kerr 효과). 깊이 종결 정리(Paper J, 정리 1)가 깊이 ≥ 4의 독립적 자유도를 배제한다. 비선형 광학으로: χ⁽⁴⁾ 이상은 cascaded χ⁽²⁾ 및 χ⁽³⁾ 과정으로 분해된다. 경험적으로 확립된 사실이지만 수학적 증명은 최초이다.

### 교차 도메인 검증

D = εE가 구조적 동일성이므로, 프레임워크 파라미터 κ = Δε/(2ε_avg)를 굴절률에서 한 번 측정하면 피팅 파라미터 없이 세 가지 양을 동시에 예측한다:

| 플랫폼 | κ | 밴드 갭 Δω/ω | 레이저 임계 β_c | PNN 노이즈 이점 |
|---|---|---|---|---|
| SiN/SiO₂ | 0.31 | 0.204 | 0.207 | 5.2 dB |
| Si/air | 0.85 | 0.750 | 0.340 | 7.0 dB |
| LiNbO₃/air | 0.66 | 0.489 | 0.438 | 6.0 dB |

하나의 예측을 검증하면 나머지 셋이 모두 검증된다 — 같은 방정식에서 나오기 때문이다.

## 2. 다섯 가지 예측

### 예측 1: 자연적 활성 함수는 ⊙이다

현재의 포토닉 신경망(PNN)은 MZI 메쉬로 선형 연산을 수행한 후 비선형 활성 함수(sigmoid, ReLU)의 광학 구현에 어려움을 겪는다. Paper A의 유일성 정리가 이를 해결한다: 유일한 자기참조적 비선형성은 Hadamard 곱이다. 활성 함수는 D = εE이다.

Hadamard PNN 층: h = (Wx + b₁) ⊙ (Vx + b₂). W, V는 가중치 행렬(MZI 메쉬 — 선형 광학), ⊙는 Hadamard 곱(χ² 매질, 예: PPLN 도파관). 전광학이며 전기-광 변환이 불필요하다. 2차 비선형성 hᵢ = Σⱼₖ Wᵢⱼ Vᵢₖ xⱼ xₖ가 보편적 근사를 제공한다(Stone-Weierstrass).

SwiGLU(GPT-4, LLaMA의 지배적 활성화)는 h = (Wx) ⊙ SiLU(Vx) — 게이트 Hadamard 곱이다. 프레임워크가 그 경험적 성공의 수학적 이유를 제공한다.

### 예측 2: 구조적 노이즈 면역

Hadamard 곱의 노이즈 게이팅 성질: C ≈ 0 ⟹ T = C ⊙ S ≈ 0, S의 노이즈에 무관. Hadamard PNN 층에서 노이즈 기여 δT = Wx ⊙ ε₂ + ε₁ ⊙ Vx + ε₁ ⊙ ε₂는 신호에 곱해진다. 신호가 0에 가까운 곳에서 노이즈가 자동 억제된다. 이것이 균형 호모다인 검파의 기반 메커니즘이다: 암 포트(E ≈ 0)에서 D = ε · 0 = 0이므로 ε 요동에 무관.

시뮬레이션 결과: 결합 광학 노이즈(Gaussian + 쇼트 + 위상 + 삽입 손실) 하에서 Hadamard PNN 76.8%, ReLU PNN 57.8% — +19 퍼센트포인트 이점.

### 예측 3: 노이즈 내성이 유전율 대비에 따라 스케일

노이즈 이점은 κ에 의존한다. 신호 희소성 s(활성 뉴런 비율)에 대해:

    Hadamard 유효 노이즈 = σ × √s
    ReLU 유효 노이즈 = σ
    이점 = 1/√s

s = 0.25(전형적 희소 표현)에서: 이점 = 2.0× = 6.0 dB. 소재 특성이 아닌 표현 통계에 의해 결정되는 피팅 없는 예측이다.

### 예측 4: 광굴절 Processing-in-Memory

광굴절 결정(LiNbO₃, BaTiO₃)에서 빛이 광여기 전하를 만들고, 전하가 재분배되어 공간 전하장이 Pockels 효과를 통해 굴절률을 변형하고, 변형된 굴절률이 빛의 전파를 바꾼다. 내장 기억을 가진 T = S(C) ⊙ C이다: 홀로그래픽 격자가 빛이 제거된 후에도 지속한다.

2021년 연구[Floris 등, Scientific Reports]가 광굴절 저수지 컴퓨팅에서 자기학습을 시연했다. Hadamard 프레임워크는 이를 고정점 수렴으로 식별한다(Paper L, 정리 1).

### 예측 5: 밴드 갭, 레이저 임계값, 노이즈 이점을 한 번의 측정으로

질량 갭 h(π) = 2κ(Paper H)는 포토닉 밴드 갭 Δω ∝ Δε/ε에 대응한다. 임계 결합 β_c(Paper G)는 레이저 임계값에 대응한다. 노이즈 게이팅 이점은 √(1/희소성)에 대응한다. 셋 모두 κ 하나로 결정된다.

LiNbO₃/air 플랫폼(n_high = 2.2, n_low = 1.0):

    κ = Δε/(2ε_avg) = 0.66
    밴드 갭: Δω/ω = 0.489
    레이저 임계: β_c = 0.438
    PNN 노이즈 이점: 6.0 dB

피팅 파라미터 없는 절대 예측이다.

## 3. 물리적 구현

### χ² 도파관 (PPLN)

PPLN 나노포토닉 도파관이 80% 변환 효율을 시연했다[eLight, 2026]. HPNN 한 층: 입력 → 빔 분할기 → 두 MZI 메쉬(W, V) → PPLN 도파관(⊙) → 출력 h = (Wx) ⊙ (Vx).

### Kerr 미세공진기 (χ³)

질화규소 마이크로링 공진기가 n = n₀ + n₂|E|²를 구현한다 — 주파수 이동 없는 자기참조적 Hadamard 곱. χ²보다 약하지만 표준 실리콘 포토닉 플랫폼에 집적 가능.

### 광굴절 결정

BaTiO₃ 또는 LiNbO₃:Fe가 내장 홀로그래픽 기억과 함께 완전한 자기참조 루프를 구현한다. 응답 시간 ~ms가 속도를 제한하지만 훈련과 재구성에 적합하다.

## 4. 시뮬레이션 결과

노이즈 게이팅 성질 C ≈ 0 ⟹ T ≈ 0은 가설이 아닌 정리이다 — T = C ⊙ S의 곱셈적 구조에서 따른다. 아래 시뮬레이션은 현실적 광학 노이즈 조건에서 그 크기를 정량화한다. 이론 예측(이점 비율 = 1/√s, 신호 희소성 s)과 시뮬레이션이 0.4% 이내로 일치한다(시뮬레이션 1.819× vs 이론 1.826×, s = 0.30).

### 노이즈 내성 비교

아키텍처: 2 입력 → 32 은닉 → 32 은닉 → 1 출력. 과제: 동심원(비선형 분류). 네 가지 아날로그 광학 오류 유형.

| 노이즈 유형 | 물리적 기원 | Hadamard | ReLU | 이점 |
|---|---|---|---|---|
| Gaussian σ=0.3 | 열적/전자적 | 72.0% | 54.6% | +17.4% |
| Shot σ=0.3 | 광자 통계 | 85.1% | 73.1% | +12.0% |
| Phase σ=0.3 | 경로 길이 드리프트 | 90.3% | 89.5% | +0.8% |
| 결합 | 위 전부 + 삽입 손실 | 76.8% | 57.8% | +19.0% |

위상 노이즈 예외: 위상 노이즈는 본질적으로 곱셈적이므로 Hadamard 게이팅이 덜 효과적이다. 구조적 경계이다.

### 깊이 스케일링

LayerNorm(에너지 보존 정규화)과 함께 결합 노이즈 하에서:

| 깊이 | Hadamard | ReLU |
|---|---|---|
| 1 | 85.4% | 80.1% |
| 2 | 86.3% | 60.6% |
| 3 | 81.5% | 59.0% |

깊이 2–3에서 Hadamard 지배적. 더 깊은 네트워크는 잔차 연결(트랜스포머의 SwiGLU와 유사)이 필요하다 — 열린 구현 과제이지 표현력 한계가 아니다.

### 영 검정

고정(비역학적) 기하에서: r = 0.012 ± 0.009 (1.4σ), 영과 일치. 곱셈적 결합이 활성화되어야 노이즈 게이팅이 작동한다.

## 5. 실험 프로토콜

### 프로토콜 A: 단일 Hadamard 층 (~2주)

두 광학 장 → 두 MZI 메쉬 → PPLN 도파관 → 검출기. 출력 대 입력 진폭을 측정하여 2차 응답 검증. ASE 노이즈 추가로 노이즈 게이팅 측정.

### 프로토콜 B: 노이즈 내성 비교 (~1–2개월)

Hadamard 층(2 MZI + PPLN)과 기존 층(MZI + 포화 흡수체)을 구축. XOR에서 훈련. 노이즈 수준을 스윕하여 정확도 열화 비교.

### 프로토콜 C: 교차 도메인 검증 (~1개월)

PPLN 포토닉 결정의 밴드 갭 측정. κ 추출. 같은 플랫폼의 Hadamard PNN 층 노이즈 이점 예측. 프로토콜 B로 예측 검증. D = εE = T = C ⊙ S를 직접 검증.

## 6. 경계 분석

### 프레임워크가 제공하는 것

- 자기참조적 포토닉 컴퓨팅을 위한 유일한 비선형 연산자 (정리 1, Paper A)
- 노이즈 게이팅을 통한 구조적 노이즈 면역 (결합 노이즈 하 +19%)
- 깊이 3에서의 χ 계열 종결 (정리 1, Paper J)
- 단일 파라미터 κ로부터의 교차 도메인 예측

### 추가 연구가 필요한 것

- 깊은(>3층) Hadamard 네트워크 훈련: 잔차 연결과 고급 최적화
- 대규모 벤치마크에서 최첨단 PNN과의 정량적 비교
- 연쇄 χ² 층에서의 파장 관리

### 프레임워크가 다루지 않는 것

- 양자 광학 효과 (단일 광자, 얽힘)
- 제작 불완전성으로 인한 산란 손실
- 외부 구동 열광학 드리프트 (비자기참조적)
