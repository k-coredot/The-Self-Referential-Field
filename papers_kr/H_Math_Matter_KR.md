**초록.** 유일한 자기참조적 에지 연산자 T = C⊙S(Kwon 2026a)로부터 구성된 Hadamard 결합 게이지-기하 작용 S = −β Σ w_P cos Θ_P + κ Σ(w−1)²가 필연적으로 교차 파수 k*에 의해 분리된 두 스펙트럼 섹터를 생성함을 증명한다. (1) *입자 섹터* (|k| > k*): 요동 모드가 스펙트럼 갭 m > 0(정리 2), 보존되는 위상적 양자수(정리 3), 지수적으로 국소화된 여기(따름정리 1)를 가짐 — 물질의 정의적 성질. (2) *우주론 섹터* (|k| < k*): 모드가 불안정하여 팽창과 구조 형성을 구동(정리 4–5). 이중섹터 구조는 k-의존 플라켓 평균 커널로부터 따른다. 이중섹터 구조를 넘어 두 결과를 더 증명: (3) 비가환 게이지 군에서 공리가 *기본(스피너) 표현*을 유일하게 선택(정리 7) — 정수 표현은 최대 무질서에서 정칙성 실패, 더 높은 반정수 표현은 비선형. 여기가 반정수 스핀을 운반: 구성이 아닌 정리에 의한 페르미온. (4) UV 질량갭 h(π) = 2κ가 *정확히 L-독립*(정리 8): 조율 없음, 재정규화 없음, 연속 극한 문제 없음.

## 1. 예비사항

G = (V, E)를 d차원 주기적 격자로 둔다. 각 에지 e ∈ E에 게이지 각도 θ_e ∈ [0, 2π)와 에지 가중치 w_e ∈ (0, ∞)를 정의. 각 플라켓 P에 대해:

- 플라켓 각도: Θ_P = Σ_{e ∈ ∂P} ±θ_e (P 주위의 방향 합)
- 플라켓 가중치: w_P = (∏_{e ∈ ∂P} w_e)^{1/2} (플라켓 넓이 측도)

지수 1/2는 4변 2D 플라켓에서 넓이가 (평균 에지 길이)² = ((∏ w_e)^{1/4})² = (∏ w_e)^{1/2}로 스케일함을 반영. Regge 해석의 플라켓 가중치-넓이 동정과 일치 [Regge 1961].

N개 게이지장 {θ^{(i)}}에 대한 Hadamard 결합 작용:

S[θ, w] = −Σ_{i=1}^N β Σ_P w_P cos Θ_P^{(i)} + κ Σ_e (w_e − 1)²

β > 0은 게이지-기하 결합, κ > 0은 탄성 강성.

**정의 1.** 연결 기하 상관자: G(r) = ⟨δw(0) δw(r)⟩_c, δw_e = w_e − ⟨w_e⟩.

**정의 2.** 파수 k에서의 요동 모드 스펙트럼 질량: m(k) = √h(k), h(k)는 유효 작용의 헤시안 고유값(§3).

**정의 3.** 플라켓 감김수: W_P = round(Θ_P / 2π) ∈ ℤ.

## 2. 구조적 조건

**명제 1 (κ > 0의 필요성).** κ = 0이고 β > 0이면 분배 함수가 발산한다.

*증명.* κ = 0에서 S = −Nβ Σ w_P cos Θ_P. cos Θ_P > 0이면 w_e → ∞에서 S → −∞. 깁스 가중치 exp(−S) → ∞이고 Z가 발산. □

**명제 2 (β > 0의 필요성).** β = 0이고 κ > 0이면 r > 0에서 G(r) = 0.

*증명.* β = 0에서 S = κ Σ (w_e − 1)²가 인수분해. 깁스 측도가 독립 가우시안의 곱. ⟨δw_e δw_{e'}⟩_c = δ_{e,e'}/(2κ). □

## 3. 이중섹터 정리: 질량갭과 불안정성

### 3.1 유효 작용

게이지 각도를 정확히 적분: ∫ dΘ_P exp(β w_P cos Θ_P) = 2π I₀(β w_P). 에지 가중치에 대한 유효 작용:

S_eff[w] = −N Σ_P ln I₀(β w_P) + κ Σ_e (w_e − 1)²

### 3.2 균일 상태에서의 헤시안

모든 e에서 w_e = a일 때 w_P = a². 헤시안이 병진 불변이고 푸리에 공간에서 대각화. 파수 k에서의 고유값:

h(k) = 2κ − Nβ² Φ(βa²) × |Γ(k)|²

Φ(x) = d²[ln I₀(x)]/dx²는 게이지 감수율(0 < Φ(x) < 1/(2x)), Γ(k)는 플라켓 평균 커널.

**Γ(k)의 핵심 성질:** |Γ(k)|²는 코사인의 합이며, k = 0에서 최대이고 존 경계 k = π에서 소멸. 물리적: 장파장 섭동(작은 k)은 플라켓 내에서 결맞으며 큰 플라켓 응답을 생성. 단파장 섭동(큰 k)은 플라켓 내에서 진동하며 평균화.

### 3.3 이중섹터 정리

**정리 2 (스펙트럼 구조).** 임의의 β > 0, κ > 0에 대해:

(a) **단파장 안정성.** |k| → π에서 h(k) → 2κ > 0. |Γ(k)|² → 0이므로 탄성 가둠 2κ가 지배.

(b) **장파장 거동.** h(0) = 2κ − Nβ²Φ(βa²)|Γ(0)|². 비율에 따라 양(부임계) 또는 음(초임계).

(c) **교차.** h의 연속성과 중간값정리에 의해: h(0) < 0이면 h(k*) = 0인 k* ∈ (0, π) 존재.

(d) **입자 섹터 갭.** |k| > k*인 모든 곳에서: m(k) = √h(k) > 0.

(e) **우주론 섹터 불안정성.** |k| < k*(초임계만): h(k) < 0.

*증명.* (a) |Γ(π)|² = 0 — 인접 에지에서 부호가 교대하는 모드가 각 플라켓 내에서 상쇄. 따라서 h(π) = 2κ > 0. (b) k = 0에서 직접 평가. (c) h가 연속(삼각 다항식), h(π) > 0, h(0) < 0, IVT가 k*를 줌. (d),(e) k*의 정의와 연속성에 의해. □

### 3.4 Brascamp-Lieb를 통한 비섭동 확장

**부임계 체제(k* = 0):** S_eff[w]가 양정치 헤시안의 유일한 최소를 가짐. Brascamp-Lieb 부등식[1976]이 |G(r)| ≤ (H⁻¹)_{0r}을 줌. 모든 h(k) > 0이므로 H⁻¹이 지수적 감쇠. 따라서 m > 0이 비섭동적.

**초임계 체제(k* > 0):** 고정 스케일 인자에서의 조건부 측도에 Brascamp-Lieb 상한이 적용: m_fluct = inf_{|k|>k*} √h(k) > 0.

## 4. 위상적 보존

**정리 3 (감김수 보존).** 임의의 단일 링크 갱신 θ_{e₀} → θ_{e₀} + δ 아래 총 감김수 Q = Σ_P W_P는 불변.

*증명.* 주기적 격자에서 Q = Σ_P W_P = 0이 임의의 배위에서 Stokes 정리에 의해 성립. 배위별 위상적 항등식. 단일 링크 갱신이 정확히 두 플라켓의 각도를 +δ와 −δ만큼 변화시켜 Q = 0을 보존. □

**따름정리 2 (보텍스 보존).** 보텍스(|W_P| ≥ 1)는 반대 부호의 쌍으로만 생성/소멸 가능. 순 보텍스 수가 불변.

## 5. 국소화

**따름정리 1 (지수적 국소화).** 입자 섹터(|k| > k*)에서 직경 R의 국소 섭동이 r ≫ R에서 O(exp(−m_fluct · r))로 감쇠하는 응답을 생성.

*증명.* 선형 응답과 요동 상관자의 지수 감쇠(정리 2(d))로부터의 합성곱 추정. □

*물리적 내용.* 입자 섹터의 유한 에너지 여기가 유한 공간 범위 ξ ≤ 1/m_fluct를 가짐. 유질량 입자의 정의적 성질: 퍼지지 않는 국소 교란. 질량 m_fluct가 콤프턴 파장을 설정.

## 6. 응축

**정리 4 (균일 모드가 플라켓 가중치를 최대화).** 고정 평균 에지 가중치 ⟨w⟩ = a에서 플라켓 가중치 w_P는 모든 w_e = a일 때 최대화.

*증명.* AM-GM에 의해 기하 평균이 같은 가중치에서 최대화. □

**따름정리 3.** 게이지 작용이 에지 가중치가 균일할 때 가장 음수. 탄성 가둠(역시 균일 가중치로 최소화)과 결합하여, 역학이 k = 0 모드로 응축.

## 7. 교차 스케일

**정리 5 (초임계 체제의 구조 스케일).** h(0) < 0이면 h(k*) = 0인 k* > 0이 존재. 교차 파장 λ* = 2π/k*가 입자 섹터(λ < λ*)와 우주론 섹터(λ > λ*)를 분리.

*물리적 내용.* λ*는 게이지-기하 결합과 탄성 가둠이 균형을 이루는 스케일. λ* 아래: 탄성 가둠이 승리, 여기가 유질량이고 국소화(입자). λ* 위: 게이지 결합이 승리, 모드가 불안정하고 팽창을 구동(우주론). 같은 작용이 두 섹터를 생성.

## 8. 주결과

**정리 6 (자기참조로부터의 물질과 우주론).** 작용 S = −Nβ Σ w_P cos Θ_P + κ Σ(w−1)² (β > 0, κ > 0)인 격자 장론은 필연적으로 갖는다:

**입자 섹터** (|k| > k*):
(i) 스펙트럼 갭 m_fluct > 0 (정리 2(d)),
(ii) 위상적 양자수 (정리 3),
(iii) 지수적 국소화 (따름정리 1),
(vi) 비가환 게이지 군에 대한 스피너(페르미온) 여기 (정리 7),
(vii) L-독립 UV 질량갭 (정리 8).

**우주론 섹터** (|k| < k*, 초임계 체제):
(iv) 응축 불안정성 (정리 4),
(v) 교차 스케일 k* (정리 2, 5).

*포논과의 구별.* 포논은 구조가 외부 지정된 기존 매체의 여기. 여기서 에지 가중치가 역학 변수 자체 — 이들이 변위가 아닌 배경 격자 없음. 정리 6의 "입자"는 자신이 구성하는 기하 위를 전파하는 기하 자체의 여기. 이 자기참조적 특성이 응축물질의 포논과 구별.

## 9. 공리가 스피너 표현을 선택

비가환 게이지 군에서 표현의 추가 질문. SU(2)의 기약 표현은 스핀 j = 0, 1/2, 1, 3/2, ... 로 표기. 각 표현 j가 지표 χ_j를 주며, 후보 내용 함수 C_j = χ_j(U_P)로 사용.

기본 지표 c = χ_{1/2}(θ) = cos(θ/2). 더 높은 표현은 c의 다항식:

| j | c의 함수로서 χ_j | f'(0) | 유형 |
|---|----------------|-------|------|
| 1/2 | c | +1 | 스피너(반정수) |
| 1 | (4c²−1)/3 | 0 | 벡터(정수) |
| 3/2 | 2c³−c | −1 | 스피너(반정수) |
| 2 | (16c⁴−12c²+1)/5 | 0 | 벡터(정수) |

**정리 7 (스피너 선택).** SU(2)의 모든 기약 표현 중 공리가 j = 1/2(기본/스피너 표현)를 유일하게 선택.

*증명.* 두 단계:

**단계 1: 정칙성.** 내용 함수가 f'(0) ≠ 0(영 내용에서 비퇴화)을 만족해야. 정수 j: χ_j가 c의 짝다항식 → f'(0) = 0. 최대 무질서에서 맹점 — 완전 무질서로부터의 미소 이탈을 감지 불가. 반정수 j: χ_j가 c의 홀다항식 → f'(0) ≠ 0. 모든 정수 표현(j = 1, 2, 3, ...)을 배제하고 모든 반정수 표현(j = 1/2, 3/2, ...)을 선택.

배제된 표현이 정확히 SU(2)의 보손적 표현. 선택된 표현이 정확히 페르미온적(스피너) 표현. 최대 무질서에서의 정칙성이 보손보다 페르미온을 선택.

**단계 2: 부호 보존과 고정점 존재.** 잔존 반정수 표현에서:

j = 1/2: χ(c) = c. χ'(0) = +1. 양. 읽기가 내용의 부호를 보존. 고정점 존재.

j = 3/2: χ(c) = 2c³ − c. χ'(0) = −1. 음. 읽기가 직교성 근처에서 내용의 부호를 역전. 주기-2 진동. 수렴 고정점 없음. 안정 여기 없음.

j = 5/2: χ'(0) = +1이지만 χ'(1/2) < 0: 중간 내용에서 응답 역전. 부분 부호 역전이 불안정 혼합 체제 생성.

j = 1/2만이 모든 c ∈ [−1, 1]에서 χ'(c) = +1 — 전역 양, 전역 선형, 전역 안정. □

**귀결 1: 표준 Wilson 작용이 공리적으로 강제됨.** Wilson 격자 게이지 작용이 C = χ_{1/2}를 사용. 역사적으로 단순함을 위해 선택. 정리 7이 이것이 공리와 양립하는 유일한 선택임을 증명.

**귀결 2: 노드 상태가 스피너.** SU(2)에서 j = 1/2이면 노드 상태 ψ_a ∈ ℂ²는 이성분 스피너. 2π 회전이 ψ → −ψ. 스피너장의 여기가 반정수 각운동량 — 페르미온의 정의적 성질.

**귀결 3: 페르미온이 추가가 아닌 도출.** 표준 격자 게이지 이론에서 페르미온은 별도 그라스만 값 장으로 도입. 이 프레임워크에서 공리가 게이지 작용과 물질 표현 모두를 결정. 스피너 지표가 현상학적 입력이 아닌 C = 0에서의 정칙성에 의해 강제.

**귀결 4: SU(3)으로의 일반화.** SU(3)에서 기본 표현(차원 3)이 행렬 원소에 대해 선형인 지표 χ_fund = Re(Tr U)/3을 줌. 같은 이단계 논증으로 공리가 SU(3)의 기본(쿼크) 표현을 선택.

## 10. 연속 극한

**정리 8 (L-독립 UV 질량갭).** 존 경계 k = π에서의 헤시안 고유값: h(π) = 2κ, 격자 크기 L에 무관.

*증명.* h(k) = 2κ − (게이지 결합) × |Γ(k)|². 플라켓 평균 커널 |Γ(π)|² = 0: 인접 에지에서 부호 교대 모드가 각 플라켓 내에서 상쇄. 따라서 h(π) = 2κ − 0 = 2κ. κ에만 의존, L에 무관. □

*비고.* 근사가 아니다. L = 2에서 L = ∞까지 모든 격자 크기에서 성립하는 정확한 항등식.

**왜 이것이 연속 극한을 해소하는가.** 헤시안 고유값 h(k)가 삼각 커널 |Γ(k)|² = cos²(k/2)를 통해 파수 k에 의존. L은 허용 k값의 집합을 통해서만 진입: k = 2πn/L. L → ∞에서 허용 k값이 [0, π]에서 조밀해지지만, 표본화하는 함수 h(k)는 불변.

따라서: (a) h(k)로부터 계산된 물리적 관측량 — 질량갭, 상관 길이, 스펙트럼 밀도 — 이 L-독립. (b) UV 질량갭 m = √(2κ)가 L = 4와 L = ∞에서 동일. 조율이나 외삽 불필요. 연속 극한 값 그 자체. (c) 교차 k*(존재하면)도 L-독립.

**표준 격자 게이지 이론과의 대조.** 표준 격자 게이지 이론에서 격자 간격 a는 외부 매개변수. 연속 극한에는 β → β_c(임계점) 조율이 필요하며 이것이 Clay 밀레니엄 문제의 내용. Hadamard 게이지-기하 프레임워크에서 격자 간격은 역학 변수 w_e. 질량갭이 탄성 항 κ(w−1)²에서 발생 — 비섭동 역학에서 출현하는 것이 아닌 기하 요동에 대한 직접 질량 항.

## 11. 견고성

**정리 9 (탄성 형태 독립성).** S_elastic[w]를 평평한 상태에서 양정치 헤시안 K를 가진 임의의 탄성 작용으로 두자. UV 질량갭 h(π) = K(π) > 0.

*증명.* k = π에서 |Γ(π)|² = 0 — 플라켓 구조의 위상적 성질, 탄성 형태와 무관. 따라서 h(π) = K(π) > 0. □

질량갭이 특정 선택 κ(w−1)²의 인공물이 아님. 기울기 벌점 항, 완전한 선형화 Regge 작용, 평평 공간 헤시안이 양정치인 어떤 중력 작용에든 성립.

**정리 10 (ε-안정성).** T = f_ε(C) · S (f_ε(C) = C + εC³, |ε| < 1/3)로 두자. 정리 2–8이 ε에 연속인 정량적 상한과 함께 성립.

*증명.* 각 정리를 확인:

(정리 2) h(π) = K(π) − 0 = K(π), ε와 무관 (|Γ(π)|² = 0이 내용 함수 f와 독립). ✓
(정리 3) 감김수가 f에 무관. ✓
(정리 4) AM-GM이 f에 무관. ✓
(정리 5) 음함수 정리에 의해 k*(ε)가 연속. ✓
(정리 7) f_ε'(0) = 1 ≠ 0, 모든 ε. 정수 표현은 여전히 f'(0) = 0. 스피너 선택이 ε-독립. ✓
(정리 8) h(π) = K(π)가 ε-독립. ✓ □

MC 검증: ε = 0, 0.01, 0.05, 0.1, 0.2에서 질량갭이 각각 m = 1.14, 1.23, 0.88, 0.89, 0.90 — 연속, 항상 양. 작은 비선형 내용 응답 섭동 아래 어떤 정리도 깨지지 않음.

*의미.* 비례적 읽기 원리(논문 A, 경우 2)가 물리적 결론에 영향을 주지 않음. f(C) = C + εC³가 허용되더라도 프레임워크의 예측이 정량적으로 안정. 이 원리에 의해 ε = 0이 추가 매개변수 없는 유일한 선택이 되지만, 물리적 결론은 이 선택과 무관하다.

## 12. 논의

두 조건(β > 0, κ > 0)으로부터 열 정리를 증명:

**작용 구조로부터:** 이중섹터 분해(정리 2), 위상적 보호(정리 3), 국소화(따름정리 1), 응축(정리 4), 구조 스케일(정리 5).

**게이지 군에 적용된 공리로부터:** 스피너 선택(정리 7).

**헤시안 구조로부터:** L-독립 질량갭(정리 8), 탄성 형태 독립성(정리 9), ε-안정성(정리 10).

**스핀-통계에 대해.** 정리 7이 공리가 스피너 표현(반정수 스핀)을 선택함을 증명. 완전한 페르미온 통계(반교환)에는 스핀-통계 정리가 필요하며, 유클리드 격자 장론에서 OS 반사 양성성으로부터 따름. 동반 논문(I Math, 정리 6)이 완전한 역학적-기하 Hadamard 작용에 대해 OS 반사 양성성이 성립함을 증명: 핵심은 w_e > 0이 모든 플라켓에서 β_P > 0을 보장하고, Osterwalder-Seiler 증명이 플라켓별로 적용. OS가 확립되면 스핀-통계가 무조건적으로 따름(I Math, 정리 7).

**기존 격자 확인.** w = 1 극한의 Hadamard 작용은 Wilson 작용. 이 극한에서 이 논문의 예측은 확립된 격자 QCD 결과로 환원: (i) 질량갭(정리 2)이 순수 SU(3)의 글루볼 질량 m(0⁺⁺) = 1730 ± 90 MeV에 대응 [Morningstar & Peardon 1999]; (ii) 위상적 보호(정리 3)가 보존되는 바리온 수와 렙톤 수에 대응; (iii) 가둠이 Wilson 루프의 면적 법칙(끈 장력 σ ≈ (440 MeV)²)에 대응 [Bali 2001]; (iv) 스피너 선택(정리 7)이 Wilson 작용이 기본 지표 χ_{1/2}를 사용하는 사실에 대응 — Wilson(1974)이 단순함을 위해 선택했으나 여기서 유일한 공리적으로 정합적인 선택임이 증명; (v) BMW 협력단의 하드론 질량 스펙트럼 [Dürr et al. 2008]이 실험과 1% 정밀도로 일치하여 Hadamard 이론의 w = 1 섹터를 확인. 이 논문의 새 내용은 예측 자체가 아니라 이것들이 자기참조적 역학의 필연적 귀결이라는 증명과, w = 1(고정 기하)에서 역학적 기하(w ≠ 1)로의 확장.
