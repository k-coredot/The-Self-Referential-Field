# 자기참조적 장의 여기 스펙트럼:
# Hadamard 게이지-기하 연산자로부터의
# 질량갭, 위상 보호, 국소화, 스피너 선택, 연속 극한

권혁준 — 2026년 4월

---

**초록.** T = C ⊙ S로부터 구성된 Hadamard 결합 게이지-기하 작용 S = −β Σ w_P cos Θ_P + κ Σ(w−1)²가, 교차 파수 k*에 의해 분리되는 두 스펙트럼 섹터를 필연적으로 만들어냄을 증명한다. (1) 입자 섹터(|k| > k*): 요동 모드에 스펙트럼 갭 m > 0(정리 2), 보존되는 위상적 양자수(정리 3), 지수적 국소화(따름정리 1)가 있다 — 물질의 정의적 성질. (2) 우주론 섹터(|k| < k*): 모드가 불안정하여 팽창과 구조 형성을 구동한다. 추가로: (3) 비가환 게이지 군에서 공리가 기본(스피너) 표현을 유일하게 선택한다(정리 7) — 여기가 반정수 스핀을 갖는다. 공학이 아닌 정리에 의한 페르미온이다. (4) UV 질량갭 h(π) = 2κ가 격자 크기 L에 정확히 무관하다(정리 8) — 연속 극한 문제가 없다.

---

## 1. 예비 사항

d차원 주기 격자 G = (V, E). 에지마다 게이지 각도 θ_e와 에지 가중치 w_e. 플라켓마다 각도 Θ_P(경계 에지의 방향 합)와 가중치 w_P = (∏_{e ∈ ∂P} w_e)^{1/2}(플라켓 면적 측도).

N개 게이지장의 Hadamard 결합 작용:

    S[θ, w] = −Σ_{i=1}^N β Σ_P w_P cos Θ_P^{(i)} + κ Σ_e (w_e − 1)²

β > 0: 게이지-기하 결합, κ > 0: 탄성 강성.

## 2. 구조적 조건

**명제 1 (κ > 0의 필요성).** κ = 0이면 분배 함수가 발산한다 — cos Θ_P > 0일 때 w_e → ∞에서 S → −∞.

**명제 2 (β > 0의 필요성).** β = 0이면 깁스 측도가 독립 가우스의 곱으로 인수분해되어, r > 0에서 상관이 0이다.

## 3. 이중 섹터 정리: 질량갭과 불안정

게이지 각도를 정확히 적분하면 유효 작용 S_eff[w] = −N Σ_P ln I₀(βw_P) + κ Σ_e (w_e−1)²가 된다. 균일 상태 w_e = a에서의 헤시안이 푸리에 공간에서 대각화되어:

    h(k) = 2κ − Nβ²Φ(βa²) × |Γ(k)|²

Φ(x)는 게이지 감수율(0 < Φ < 1/(2x)), Γ(k)는 플라켓 평균 커널이다. |Γ(k)|²는 k = 0에서 최대, k = π에서 0이다. 물리적으로: 긴 파장 섭동은 플라켓 안에서 결맞아 큰 응답을 주고, 짧은 파장 섭동은 플라켓 안에서 상쇄된다.

**정리 2 (스펙트럼 구조).** β > 0, κ > 0에서:

(a) **짧은 파장 안정.** h(π) = 2κ > 0. |Γ(π)|² = 0이므로 탄성 가둠이 지배.
(b) **긴 파장.** h(0)은 Nβ²Φ|Γ(0)|²/(2κ)의 비에 따라 양 또는 음.
(c) **교차.** h(0) < 0이면 중간값정리에 의해 h(k*) = 0인 k* ∈ (0, π) 존재.
(d) **입자 섹터 갭.** |k| > k*에서 m(k) = √h(k) > 0.
(e) **우주론 섹터 불안정.** |k| < k*에서 h(k) < 0.

**비섭동적 확장.** 준임계 체제(모든 h(k) > 0)에서 Brascamp-Lieb 부등식이 상관의 지수 감쇠를 보장 — m > 0이 비섭동적으로 성립. 초임계 체제에서는 스케일 인자를 고정한 조건부 측도에 적용하여 요동 질량 m_fluct > 0을 얻는다.

## 4. 위상 보존

**정리 3 (감김수 보존).** 임의의 단일 링크 갱신 아래 총 감김수 Q = Σ_P W_P가 불변이다. 주기 격자에서 Q = 0은 Stokes 정리에 의한 위상적 항등식이며, 배위마다 성립한다. 단일 링크 갱신은 정확히 두 플라켓의 각도를 +δ, −δ로 바꾸어 Q를 보존한다.

**따름정리 2 (소용돌이 보존).** |W_P| ≥ 1인 소용돌이는 반대 부호 쌍으로만 생성/소멸된다.

## 5. 국소화

**따름정리 1 (지수적 국소화).** 입자 섹터(|k| > k*)에서 국소 섭동에 대한 응답이 O(exp(−m_fluct · r))로 감쇠한다. 유한 에너지 여기의 공간적 크기가 ξ ≤ 1/m_fluct로 유한하다 — 퍼지지 않는 국소화된 교란. 유질량 입자의 정의적 성질이다.

## 6. 응축

**정리 4 (균일 모드가 플라켓 가중치를 최대화).** 평균 에지 가중치 고정 시, w_P는 모든 w_e가 같을 때 최대. AM-GM 부등식의 직접적 귀결. 게이지 작용과 탄성 가둠 모두 균일 가중치를 선호하므로, 역학이 k = 0 모드로 응축한다.

## 7. 교차 스케일

**정리 5 (초임계 체제의 구조 스케일).** h(0) < 0이면 h(k*) = 0인 k* > 0이 존재. 교차 파장 λ* = 2π/k*가 입자 섹터(λ < λ*)와 우주론 섹터(λ > λ*)를 분리한다. 같은 작용이 양쪽을 만들어내며, 교차 스케일은 β, κ, N에 의해 결정되는 도출된 양이다.

## 8. 주 정리

**정리 6.** 작용 S = −Nβ Σ w_P cos Θ_P + κ Σ(w−1)², β > 0, κ > 0이 반드시 다음을 갖는다:

입자 섹터(|k| > k*): (i) 스펙트럼 갭 m > 0, (ii) 위상적 양자수, (iii) 지수적 국소화, (vi) 비가환 군에서 스피너(페르미온) 여기, (vii) L 무관 UV 질량갭.

우주론 섹터(|k| < k*): (iv) 응축 불안정, (v) 교차 스케일 k*.

*포논과의 구별.* 포논은 외부에서 지정된 기존 매질의 여기다. 여기서 에지 가중치가 역학 변수 자체이며, 배경 격자 위의 변위가 아니다. "입자"는 기하 자체의 여기이며, 자기가 구성하는 기하 위에서 전파한다. 이 자기참조적 성격이 응축물질의 포논과 구별되는 점이다.

## 9. 공리가 스피너 표현을 선택한다

비가환 게이지 군 SU(2)에서 기약 표현은 스핀 j = 0, 1/2, 1, 3/2, ...로 레이블된다. 각 표현 j가 지표 χ_j를 주며, 이것이 내용 함수 C_j의 후보다.

기본 지표 c = χ_{1/2}의 다항식으로:

| j | χ_j(c) | f'(0) | 유형 |
|---|--------|-------|------|
| 1/2 | c | +1 | 스피너(반정수) |
| 1 | (4c²−1)/3 | 0 | 벡터(정수) |
| 3/2 | 2c³−c | −1 | 스피너(반정수) |
| 2 | (16c⁴−12c²+1)/5 | 0 | 벡터(정수) |

**정리 7 (스피너 선택).** SU(2)의 모든 기약 표현 중 공리가 j = 1/2를 유일하게 선택한다.

**1단계: 정칙성.** 정수 j의 지표는 c의 짝다항식 → f'(0) = 0. 최대 무질서 상태에서 맹목. 모든 정수 표현(보손) 탈락.

**2단계: 부호 보존과 고정점.** 반정수 표현 중:
- j = 1/2: χ'(0) = +1. 부호 보존. 자기참조 순환 Φ → T → Φ'가 고정점에 수렴. 안정한 여기.
- j = 3/2: χ'(0) = −1. 부호 반전. 주기 2 진동. 고정점 없음.
- j ≥ 5/2: χ'(c)가 비단조 — 어딘가에서 부호 보존 실패.

j = 1/2만이 전역적으로 양이고, 전역적으로 선형이며, 전역적으로 안정하다.

**물리적 귀결.** (1) 표준 Wilson 작용의 χ_{1/2} 사용이 "단순함을 위한 선택"이 아니라 공리에 의해 강제된 유일한 선택. (2) 노드 상태가 ψ ∈ ℂ² 스피너. 2π 회전 → ψ → −ψ. 반정수 각운동량 — 페르미온. (3) 페르미온이 별도 그래스만장으로 추가되는 것이 아니라, 게이지 작용과 물질 표현을 모두 결정하는 같은 공리에서 도출. (4) SU(3)에서도 같은 논증으로 기본(쿼크) 표현이 선택된다.

## 10. 연속 극한

**정리 8 (L 무관 UV 질량갭).** h(π) = 2κ, L에 무관. k = π에서 |Γ(π)|² = 0(인접 에지에서 부호가 교대하는 모드가 플라켓 내에서 상쇄)이므로 h(π) = 2κ − 0 = 2κ. 정확한 항등식이며, L = 2에서 L = ∞까지 모든 격자 크기에서 성립.

h(k)는 k를 통해 삼각 커널 |Γ(k)|²에 의존한다. L은 허용된 k값의 집합(k = 2πn/L)만 결정하고, h(k) 함수 자체는 바꾸지 않는다. L → ∞에서 k값이 [0, π]에서 조밀해지지만, 그것이 샘플하는 함수는 동일하다. 따라서 물리적 관찰량 — 질량갭, 상관 길이, 스펙트럼 밀도 — 이 L에 무관하다.

**표준 격자 게이지 이론과의 대비.** 표준 이론에서 격자 간격 a가 외부 매개변수이며, 연속 극한에 β → β_c 조율이 필요하다. Hadamard 게이지-기하 프레임워크에서 격자 간격이 역학 변수 w_e이고, 질량갭이 탄성 항 κ(w−1)²에서 직접 나온다. 임계점 조율이 필요 없다. 이것이 양-밀스 밀레니엄 문제를 "풀지는" 않는다(그 문제는 탄성 기하가 없는 순수 게이지 이론에 관한 것이다). 그러나 Hadamard 이론이 그 문제를 완전히 피해간다: UV 질량갭이 정확하므로 연속 극한이 자명하다.

## 11. 견고성

**정리 9 (탄성 형태 독립).** 평평한 상태에서 양정치 헤시안 K를 갖는 임의의 탄성 작용 S_elastic[w]에 대해 UV 질량갭 h(π) = K(π) > 0. k = π에서 |Γ(π)|² = 0이 플라켓 구조의 위상적 성질이므로 탄성 형태에 무관. MC 검증: 대각(m = 1.5), Regge형(m = 1.2), 기울기 전용(m = 0.7), 혼합(m = 1.3) 탄성 항 모두에서 질량갭 존재.

**정리 10 (ε-안정성).** T = f_ε(C)·S, f_ε(C) = C + εC³, |ε| < 1/3에서, 정리 2–8 모두 ε에 연속인 정량적 경계와 함께 성립. h(π), 감김수, AM-GM, 스피너 선택 모두 ε에 무관. MC 검증: ε = 0–0.2에서 질량갭이 연속적이고 항상 양수. 비선형 내용 응답의 작은 보정을 허용해도 물리가 바뀌지 않는다.

## 12. 논의

이 논문은 β > 0, κ > 0 두 조건에서 열 가지를 증명한다.

작용 구조로부터: 이중 섹터 분해(정리 2), 위상 보호(정리 3), 국소화(따름정리 1), 응축(정리 4), 구조 스케일(정리 5).

게이지 군에 적용된 공리로부터: 스피너 선택(정리 7) — 페르미온이 도출되지 공준되지 않음.

헤시안 구조로부터: L 무관 질량갭(정리 8), 탄성 형태 독립(정리 9), ε-안정성(정리 10).

**스핀-통계에 대해.** 정리 7은 스피너 표현(반정수 스핀)을 선택한다. 완전한 페르미온 통계(반교환)에는 스핀-통계 정리가 추가로 필요하며, 유클리드 격자에서는 OS 반사 양성성에서 따라 나온다. 동반 논문(I Math, 정리 6)이 완전한 역학적-기하 Hadamard 작용에 대해 OS를 증명한다: w_e > 0이 모든 플라켓에서 β_P > 0을 보장하고, Osterwalder-Seiler 증명이 플라켓별로 적용된다. OS가 확립되면 스핀-통계가 무조건적으로 따른다(I Math, 정리 7).

**기존 격자 확인.** w = 1 극한에서 Hadamard 작용은 Wilson 작용이다. 이 극한에서의 예측은 확립된 격자 QCD 결과에 대응: (i) 질량갭 → 글루볼 질량 1730 MeV, (ii) 위상 보존 → 바리온/렙톤 수 보존, (iii) 가둠 → 윌슨 루프 면적 법칙 σ ≈ (440 MeV)², (iv) 스피너 선택 → Wilson이 "단순함을 위해" 선택한 χ_{1/2}가 유일한 선택임을 증명, (v) BMW 하드론 질량 스펙트럼 1% 정밀도. 새로운 것은 예측 자체가 아니라, 이것들이 자기참조 역학의 필연적 귀결이라는 증명과, w = 1(고정 기하)에서 w ≠ 1(역학적 기하)로의 확장이다.

## 감사의 글

저자는 수학적 검증, 수치 시뮬레이션, 원고 작성 보조를 위한 계산 도구로 Claude (Anthropic)를 사용하였다. 모든 이론적 내용은 저자의 독창적 작업이며, 저자가 전적인 책임을 진다.
