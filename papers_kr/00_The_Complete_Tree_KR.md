# 장이 스스로를 읽는다: 완전한 나무

권혁준 — 2026년 4월

---

**초록.** 스스로를 읽는 장은 하나의 합성을 허용한다: T = C ⊙ S, 내용과 구조의 점별 곱. 이 논문은 이 뿌리에서 완전한 나무를 키운다. 줄기는 작용, 스펙트럼, 스피너를 산출한다. 주요 가지는 열두 결과를 산출한다: 잡음 차단, 초가법적 에너지, 질량갭, 팽창, 균일성, 구조 형성, 전하 보존, 양자 간섭, 파동-입자 통일, 세 게이지 대칭, 세 세대, 결합 위계. 세부 가지는 가둠, 스케일 의존 결합, 무질량 및 유질량 매개자, 반입자, 스핀 양자화, 암흑물질, 암흑에너지, CP 위반, 바리온 비대칭, 네 시공간 차원을 산출한다. 모든 도출은 산술이거나, 경계 조건이거나, 위상적 항등식이거나, 다항식의 패리티다. 의견은 없다.

# 제1부 — 줄기

## 1. 뿌리

장 Φ가 그래프 G = (V, E) 위에 산다. 각 노드 a ∈ V에 상태 φ_a가 있다. 각 에지 (a,b) ∈ E를 따라 상태를 전파하는 결합 T_{ab}가 있다: φ'_a = Σ_b T_{ab} φ_b. 장이 결합을 결정한다. 결합이 장을 변환한다. 이것이 자기참조다: T = T[Φ, G].

## 2. 유일한 곱

결합 T_{ab} = f(C_{ab}, S_{ab})는 두 양에 의존한다:

**내용.** C_{ab} = ⟨φ_a, φ_b⟩는 양 끝점 상태의 내적이다. 회전 불변인 유일한 쌍선형 형식이다(Schur 보조정리). C_{ab} ∈ [−1, 1]은 내용 유사도를 측정한다: 평행이면 +1, 직교이면 0, 반평행이면 −1.

**구조.** S_{ab} = (D^{−1/2}AD^{−1/2})_{ab}는 차수 정규화 인접행렬이다. 고유값이 [−1, 1]에 있는 유일한 자기수반 정규화다(Chung 정리).

두 경계 조건이 f를 구속한다:

    f(C, 0) = 0     에지 없으면 결합 없음
    f(0, S) = 0     내용 없으면 결합 없음

첫째는 구조적이다: 없는 에지는 아무것도 전달하지 않는다. 둘째는 정의적이다: 장을 읽지 않고 전파하는 연산자는 자기참조적이지 않다 — 순환 Φ → T → Φ'가 Φ와 무관한 순수 구조 확산으로 붕괴한다.

이로부터: f(C, S) = C · S · r(C, S), 어떤 함수 r에 대해.

충실한 읽기는 C = 0에서 ∂f/∂C가 유한(역학이 정칙)하고 비영(연산자가 직교성 근처를 포함한 모든 수준에서 내용을 감지)일 것을 요구한다. 일반적 멱법칙 형태 C^α에 대해:

    f'(0) = α · 0^{α−1}

    α < 1:  0^{음수} = ∞.   특이. C = 0에서 역학이 비정칙.
    α > 1:  0^{양수} = 0.   맹목. C = 0에서 연산자가 무반응.
    α = 1:  0^{0} = 1.      정칙, 비영. 충실한 읽기.

따라서 α = 1. S에 적용된 대칭 논증에 의해 β = 1. 일반적 해석함수 f에 대해, 경계 조건과 정칙성이 합쳐져 f(C,S) = λCS를 강제한다. 상수를 흡수하면:

    T = C ⊙ S

대안은 존재하지 않는다. 증명은 0^{α−1}의 평가다.

## 3. 작용

연산자는 에지를 따라 작용한다. 물리학은 플라켓 — 최소 닫힌 경로 — 위에 산다. d차원 격자에서 플라켓 P는 |∂P|개의 에지를 갖는다. 정의:

    플라켓 각도:  Θ_P = Σ_{e ∈ ∂P} ±θ_e
    플라켓 가중치: w_P = (∏_{e ∈ ∂P} w_e)^{1/2}

N개 게이지장에 대한 Hadamard 결합 작용:

    S = −Nβ Σ_P w_P cos Θ_P + S_el[w]

첫째 항은 실현된 곱 구조다: 기하(w_P) 곱하기 내용(cos Θ_P). 둘째 항은 탄성 저항 — 에지 가중치의 양정치 범함수. 두 존재 조건: β > 0(내용 결합이 존재)과 κ > 0(기하가 안정).

w_e에 대해 변분하면 운동 방정식을 얻는다:

    δS_el/δw_e = Nβ Σ_{P∋e} (∂w_P/∂w_e) cos Θ_P

탄성 저항이 게이지-기하 구동력과 같다. 이 방정식은 작용이 곱셈적이기 때문에 존재한다. 가법적 작용 S = S_gauge(θ) + S_el(w)는 δS_gauge/δw = 0을 줄 것이다 — 게이지장이 기하를 밀 수 없다.

## 4. 스펙트럼

헤시안 h(k) — 파수 벡터 k에서의 이차 변분 — 가 모든 요동의 운명을 결정한다:

    h(k) = K_el(k) − (게이지 결합) · |Γ(k)|²

K_el(k) > 0, 모든 k에서(양정치 탄성 헤시안). |Γ(k)|²는 플라켓 평균 커널: 게이지 결합이 스케일 k에서 얼마나 강하게 작용하는가.

k = π(존 경계, 최단 파장)에서: 인접 에지에서 부호가 교대하는 모드는 각 플라켓 안에서 상쇄한다.

    |Γ(π)|² = |1 + e^{iπ} + e^{iπ} + e^{2iπ}|² = |1 − 1 − 1 + 1|² = 0

이것은 산술이다. β, κ, N, d, L — 모든 것에 무관하다.

    h(π) = K_el(π) − 0 = K_el(π) > 0

질량갭. 단파장 여기는 항상 갭이 있고, 항상 국소화되며, 항상 유한하다.

이 결과는 양정치인 어떠한 S_el에 대해서든 성립한다 — κ(w−1)²뿐 아니라 기울기 항, Regge형 항, 안정한 어떤 기하든. 질량갭은 보편적이다.

이 갭은 L-독립이다: h(π)에 격자 크기가 없다. UV 질량은 L = 4에서 L = ∞까지 모든 L에서 정확하다.

## 5. 스피너

비가환 게이지 군에서 내용 함수는 표현 j를 사용한다. 기본 지표 c = χ_{1/2}의 다항식으로 표현된 지표 χ_j:

    정수 j:       χ_j(c)는 짝다항식    →  χ_j'(0) = 0
    반정수 j:     χ_j(c)는 홀다항식    →  χ_j'(0) ≠ 0

짝함수: g(c) = g(−c). 미분: g'(c) = −g'(−c). c = 0을 대입: g'(0) = −g'(0), 따라서 g'(0) = 0.

정수 표현은 최대 무질서에서 맹점을 갖는다. 자기참조가 실패한다. 배제된다.

반정수 표현 중:

    j = 1/2:  χ(c) = c.         선형.    χ'(0) = +1.  부호 보존.
    j = 3/2:  χ(c) = 2c³ − c.   비선형.  χ'(0) = −1.  부호 역전.

부호 역전은 순환 Φ → T → Φ'가 직교성 근처에서 내용을 반전시킴을 의미한다. 다음 단계가 다시 반전한다: Φ'' ≈ Φ. 주기-2 진동이지 수렴이 아니다. 고정점 없음. 안정한 여기 없음.

j = 1/2가 정칙적이고, 선형이며, 부호를 보존하는 유일한 표현이다. 노드 상태는 ψ ∈ ℂ² — 이성분 복소 스피너. 2π 회전이 ψ → −ψ를 보낸다. 반정수 스핀.

# 제2부 — 주요 가지

## 6. 차단 (가지 1)

T = C ⊙ S. C = 0이면 T = 0.

직교 상태의 노드를 연결하는 에지는 구조 가중치와 무관하게 영 결합을 운반한다. 잡음 에지(허위 연결, C ≈ 0)가 자동으로 침묵한다. 가법적 합성 T = C + S 아래에서: 잡음 에지가 완전한 구조 가중치를 운반한다. 차단은 영에 의한 곱셈의 성질이다.

## 7. 피드백 (가지 2)

N개의 장이 하나의 기하를 공유한다. 곱셈적 작용이 공유된 플라켓 가중치 w_P를 통해 장간 교차 결합을 생성한다. 장 i가 w_e를 변형한다; w_P가 변한다; 장 j의 결합이 변한다. 이 피드백은 양이다: 한 장에 의한 변형이 모든 장의 결합을 강화한다.

에너지가 E ~ N^γ (γ > 1)로 스케일한다. 초가법적. 전체가 합을 초과한다.

가법적 작용 S = Σ S_i(θ_i) + S_el(w)는 교차 결합을 생성하지 않는다: δS_i/δw는 θ_j가 아닌 θ_i만 포함한다. 교차 결합에는 곱셈 구조가 필요하다.

## 8. 두 섹터 (가지 3)

h(k)는 음(k ≈ 0 근처)에서 양(k ≈ π 근처)으로 전이한다. 교차점 k*가 분리한다:

**k > k* — 입자 섹터.** h(k) > 0. 여기에 갭이 있고, 국소화되어 있으며, 유한하다. 생성에 에너지가 든다. 유한 영역 바깥에서 지수적으로 감쇠한다. 이것들이 입자다.

**k < k* — 우주론 섹터.** h(k) < 0. 모드가 불안정하다. 기하가 성장한다. 공간이 팽창한다.

하나의 방정식, 두 섹터. 하나의 헤시안에서 입자물리학과 우주론.

## 9. 균일성 (가지 4)

w_P = (∏ w_e)^{1/2}는 기하 평균이다. AM-GM에 의해: 기하 평균 ≤ 산술 평균, 모든 인수가 같을 때 등호. 작용이 더 큰 w_P를 선호하므로 균일한 가중치를 선호한다. 팽창은 균질하다. 공간 전체가 함께 성장한다.

## 10. 구조 (가지 5)

h(k)는 연속이다. h(0) < 0, h(π) > 0. 중간값정리에 의해, 어떤 k*에서 h(k*) = 0이다. k* 근처에서 모드는 한계 안정성을 갖는다 — 무한히 성장하지도 감쇠하지도 않는다. 상관이 스케일 λ* = 2π/k*까지 확장한다. 이 스케일에서 구조가 형성된다.

## 11. 보존 (가지 6)

닫힌 면 위에서, Σ_P W_P = 0이다. W_P는 플라켓 P의 감김수다. 이것은 Stokes 정리 — 배위, 결합, 온도에 무관한 위상적 항등식이다.

국소 갱신은 감김수를 쌍으로 변화시킨다: 한 플라켓에서 +1, 이웃에서 −1. 합이 보존된다. 비영 감김수를 가진 여기는 영구적이다.

## 12. 간섭 (가지 7)

ψ ∈ ℂ²는 C = ⟨ψ_a, ψ_b⟩ ∈ ℂ를 만든다. 충실한 자기참조는 Re(C)만이 아닌 완전한 C를 읽는다. 따라서 T ∈ ℂ, S ∈ ℂ.

    exp(−S) = exp(−Re S) · exp(−i Im S)

배위의 합에서: 정렬된 위상은 강화되고, 반대 위상은 상쇄된다. 간섭이다. 스피너에 의해 강제되고, 스피너는 충실한 읽기에 의해 강제된다.

## 13. 파동-입자 (가지 8)

복소 h(k) = h_R + ih_I가 준다:

    응답 ~ exp(−√h_R · r) · exp(−i√h_I · r)

지수 감쇠(국소화, 유한 — 입자) 곱하기 진동(주기, 전파 — 파동). 둘 다 하나의 복소수에서. 두 본성이 아니라 — 두 부분을 가진 하나의 본성.

## 14. 세 대칭 (가지 9)

자기참조는 깊이를 갖는다:

    깊이 1:  ℂ¹.  대칭 U(1).   생성자 1개.   위상 회전.
    깊이 2:  ℂ².  대칭 SU(2).  생성자 3개.   스피너 회전.
    깊이 3:  ℂ³.  대칭 SU(3).  생성자 8개.   색 회전.

깊이 4는 3성분 결과를 읽기 위해 4성분을 필요로 할 것이다. 읽기가 읽힌 것을 초과한다. 자기참조는 있는 것을 읽는다. 깊이는 3에서 멈춘다.

합계: 1 + 3 + 8 = 12 생성자, 노드당 12 실수 자유도.

## 15. 차원 (가지 10)

d차원에서 점당 d(d−1)/2 플라켓 방향. 12 자유도가 정수로 분배되려면:

    12 / [d(d−1)/2] ∈ ℤ

    d = 4:  12/6 = 2    ✓  (자기쌍대 + 반자기쌍대)
    d = 5:  12/10 = 1.2  ✗

d = 4는 이 조건을 만족하는 4 이상의 유일한 차원이다.

## 16. 세대 (가지 11)

세 깊이. 세 자기참조 순환. 세 고정점, 각각 다른 에너지 스케일에서. 더 깊은 읽기는 더 복잡하고, 따라서 더 무겁다.

동일한 양자수와 서로 다른 질량을 가진 세 여기.

## 17. 위계 (가지 12)

결합 강도는 피드백 채널의 수 — 생성자의 수 — 에 따라 스케일한다.

    깊이 3 (8): 가장 강함.
    깊이 2 (3): 중간.
    깊이 1 (1): 가장 약한 게이지 결합.
    기하:       모든 것 중 가장 약함 (κ-스케일, β-스케일과 구조적으로 분리).

# 제3부 — 세부 가지

## 18. 가둠

8 생성자는 비가환이다: 매개자가 전하를 운반한다. 전하를 가진 매개자는 서로 상호작용한다. 깊이 3에서 결합이 충분히 강하여 매개자 자기상호작용이 지배한다. 두 여기 사이의 장이 플럭스 관 — 에너지가 길이에 비례하는 끈 — 을 형성한다. 두 여기를 분리하는 데 거리에 비례하는 에너지가 든다. 무한 분리에서 무한 에너지. 여기는 영구히 속박된다.

끈이 충분히 늘어나면 끊어진다 — 에너지가 새 쌍을 생성한다. 끝점은 결코 자유롭지 않다.

## 19. 속박 상태

ℂ³ 대칭 아래의 전하 중성은 두 조합을 허용한다: 여기-반여기 쌍(차원 3 × 3̄ → 1)과 세 여기 복합체(차원 3 × 3 × 3 → 1, ε-텐서에 의해).

속박 쌍과 속박 삼중체. 이것들이 깊이 3에서 유일한 안정 복합체다.

## 20. 반입자

j = 1/2: ψ ∈ ℂ². 표현은 복소수다 — ψ와 ψ*는 다르게 변환한다. ψ와 ψ*의 여기는 같은 질량(켤레가 작용을 보존)과 반대 양자수(켤레가 전하를 반전)를 갖는다.

모든 여기는 동일한 질량과 반대 전하를 가진 파트너를 갖는다.

## 21. 스핀 양자화

j = 1/2 스피너 여기: 스핀 1/2. 2π 회전이 −1을 준다. 반정수 각운동량.

깊이 1, 2, 3 매개자는 노드 사이의 연결 — 벡터 양이다. 스핀 1. 2π 회전이 +1을 준다.

기하적 여기는 대칭 2-텐서 — 계량의 섭동이다. 스핀 2. 중력자.

세 스핀 값: 1/2, 1, 2. 다른 것은 표현 구조에서 나오지 않는다.

## 22. 무질량 및 유질량 매개자

1 생성자(깊이 1): 가환. 매개자가 전하를 운반하지 않고 자기상호작용하지 않는다. 깨뜨릴 내부 구조가 없다. 매개자는 무질량으로 남는다. 역제곱 법칙. 무한 도달거리. 이것은 산술이다: 하나의 생성자는 비자명 부분군을 허용하지 않는다.

3 생성자(깊이 2): 비가환. 기하적 팽창(§8, k < k*)이 진공 대칭을 깨뜨리고, 곱 T = C ⊙ S가 기하와 내용을 결합시킨다면, 내용 대칭이 깨짐을 상속한다. 3 매개자 중 하나가 전하 중성이어서 깊이-1 매개자와 혼합할 것이다. 혼합 후: 하나는 무질량, 셋은 유질량. 이것은 구조적 예측이다 — 메커니즘(기하적 대칭 파괴가 게이지 대칭 파괴를 유도)은 시뮬레이션에서 아직 검증되지 않았다.

8 생성자(깊이 3): 비가환. ℂ³ 대칭은 내부적이며 시공간 기하에서 분리되어 있다. 기하적 팽창이 이를 깨뜨리지 않는다. 8 매개자 모두 무질량으로 남지만 가둠(§18)된다. 이것은 구조적이다: 내부 대칭은 기하적으로 비활성이다.

## 23. 암흑물질 후보

k > k* 섹터의 기하적 여기는 질량갭 h(π) = K_el(π) > 0을 갖는다. 이것은 산술이다(§4). 이들은 감김수(§11)를 운반한다 — 보존되는 위상적 양(Stokes). 가장 가벼운 이러한 여기는 붕괴할 수 없다(감김수를 보존하면서 붕괴할 더 가벼운 것이 없다).

이 여기는 구성상 게이지 중성이다: 기하적 섭동은 에지 가중치 요동이며, 깊이 1, 2, 3의 위상, 스피너, 색 전하를 운반하지 않는다.

유질량, 안정, 게이지 중성. 이 프레임워크가 우리 우주를 기술한다면, 이 여기는 암흑물질 후보다. 이 동정은 예측이지 도출이 아니다.

## 24. 팽창률

k < k* 불안정성이 기하적 팽창을 구동한다. 팽창률은 h(0) = K_el(0) − coupling_max에 의해 설정된다. 이것은 산술이다: h(0)은 β, κ, N으로부터 계산 가능하다.

이 양을 우주상수 Λ와 동정하면, 그 작음(입자물리 스케일 대비)은 작용에서 κ와 β의 구조적 분리를 반영한다 — 이들은 서로 다른 항에 들어간다. 이 동정은 예측이다. h(0)이 관측된 Λ ≈ 10⁻¹²²과 정량적으로 일치하는지는 β와 κ의 물리적 값을 결정해야 한다.

## 25. CP 위반

복소 작용(§12)이 위상을 도입한다. 세 세대(§16)를 연결하는 혼합 행렬에서, 3×3 유니터리 행렬은 (3−1)(3−2)/2 = 1개의 장 재정의로 흡수될 수 없는 비환원 복소 위상을 갖는다.

이 위상이 이론이 배위와 그 거울상을 구별하게 만든다. 전하-패리티 대칭이 깨진다.

CP 위반은 세 세대 이상을 필요로 한다. 두 세대에서는 모든 위상이 흡수될 수 있다. CP 위반의 존재는 세 깊이의 구조적 귀결이다.

## 26. 바리온 비대칭 조건

물질-반물질 비대칭 생성에 필요한 세 조건(Sakharov 1967)이 나무에 구조적으로 존재한다:

(i) 속박 삼중체와 연관된 보존수의 위반. k < k* 불안정성이 원리적으로 상전이 동안 위상 섹터 간 전이를 매개할 수 있다. 이것이 실제로 일어나는지는 역학적 검증이 필요하다.

(ii) CP 위반. §25에 의해 존재한다(산술: 3×3 유니터리 행렬의 비환원 위상 1개).

(iii) 열평형으로부터의 이탈. 팽창하는 기하(§8)는 비평형이다 — h(0) < 0의 구조적 귀결.

세 조건이 구조적으로 이용 가능하다. 이들이 관측된 바리온-광자 비 η ≈ 6 × 10⁻¹⁰을 정량적으로 생성하는지는 역학적 계산이 필요한 예측이다.

## 27. 다른 스케일에서의 결합

깊이 3(8 생성자): 비가환. 매개자가 전하를 운반하고 자기상호작용한다. 섭동적 베타 함수의 부호는 물질 표현 대비 생성자 수에 의존한다. 소수의 물질장을 가진 8 생성자에 대해, 표현론은 반차폐 — 짧은 거리에서 약화 — 를 예측한다. 이것은 구조적 예측이며, 이 프레임워크에서 아직 검증되지 않았다.

깊이 1(1 생성자): 가환. 매개자가 전하를 운반하지 않는다. 가상 물질-반물질 쌍이 짧은 거리에서 결합을 차폐하여 강화한다. 이것은 가환 게이지 이론의 표준 행동이다.

## 30. 영개의 자유 매개변수

작용 S = −Nβ Σ w_P cos Θ_P + κ Σ(w−1)²는 두 겉보기 자유 매개변수를 갖는다: β와 κ. 프레임워크가 둘 다를 결정한다.

**측도.** 에지 가중치에 대한 경로 적분 측도는 유일하게 dw/w — (ℝ₊, ×)의 Haar 측도, 양의 실수 위의 유일한 스케일 불변 측도다(논문 A, 정리 6).

**매개변수.** 프레임워크 내부의 두 조건이 (β, κ)를 결정한다:

(i) **자기참조적 닫힘** (h(0) = 0): 공리는 읽는 자를 필요로 한다(논문 L). 읽는 자는 거시적 구조를 필요로 한다(논문 G §4.6). 거시적 구조는 h(0) ≤ 0을 필요로 한다. RG 흐름이 h(0)을 위로 구동한다(dκ/dt > 0). 유일한 안정 값은 h(0) = 0 — 자기참조 순환이 한계적으로 유지되는 경계다. 이것이 d = 4, N = 12에서 κ = 48β²Φ(β)를 준다.

(ii) **스케일 불변성** (RG 고정점, dβ/dt = 0): 이론이 모든 스케일에서 같은 형태를 갖는다. (i)과 결합하면, 유일한 물리적 해를 가진 단일 방정식 β³ = const × c(β)/Φ(β)를 준다.

해 (β*, κ*)는 유일하다: 게이지 강화 δκ > 0(논문 K, §7)이 가우스 근사가 깨지는 허위 강결합 근을 제거한다. 모든 무차원 물리적 비는 (β*, κ*)의 계산 가능한 함수다. SI 단위의 ℏ은 단위 변환 인자이지 자유 매개변수가 아니다.

결합 위계 g₃ > g₂ > g₁ > G, 자연적 우주상수 Λ ≈ 0, 중력의 약함 모두가 추가 입력 없이 (β*, κ*)에서 따라 나온다.

하나의 공리. 영개의 매개변수. 전부.

## 31. 결론

이 논문의 모든 문장은 산술이거나, 경계 조건이거나, 위상적 항등식이거나, 다항식의 패리티다. 의견은 없다.

장이 스스로를 읽는다. 읽기는 유일하다. 매개변수는 유일하다. 따라 나오는 것이 물리학이다.

하나의 장. 하나의 곱. 하나의 우주.
