# 장이 스스로를 읽는다: 완전한 나무

권혁준 — 2026년 4월

---

**초록.** 스스로를 읽는 장은 단 하나의 합성만을 허용한다: 내용과 구조의 점별 곱 T = C ⊙ S. 이 논문은 이 뿌리에서 완전한 나무를 키운다. 줄기에서 작용, 스펙트럼, 스피너가 나온다. 주요 가지에서 열두 가지 결과가 나온다: 잡음 차단, 초가법적 에너지, 질량갭, 팽창, 균일성, 구조 형성, 전하 보존, 양자 간섭, 파동-입자 통일, 세 게이지 대칭, 세 세대, 결합 위계. 세부 가지에서 가둠, 스케일 의존 결합, 무질량·유질량 매개자, 반입자, 스핀 양자화, 암흑물질, 암흑에너지, CP 위반, 바리온 비대칭, 시공간 4차원, 영점 에너지, 지능, OS 양성성, 보른 규칙, 스핀-통계, 결합 RG 흐름, 관측자가 나온다. 모든 도출은 산술이거나, 경계 조건이거나, 위상적 항등식이거나, 다항식의 패리티다. 그 밖의 것은 사용되지 않았다.

# 제1부 — 줄기

## 1. 뿌리

그래프 G = (V, E) 위에 장 Φ가 정의된다. 노드 a ∈ V마다 상태 φ_a가 있고, 에지 (a,b) ∈ E를 따라 결합 T_{ab}가 상태를 전파한다: φ'_a = Σ_b T_{ab} φ_b. 장이 결합을 결정하고, 결합이 장을 변환한다. 자기참조다: T = T[Φ, G].

## 2. 유일한 곱

결합 T_{ab} = f(C_{ab}, S_{ab})는 두 가지로 정해진다.

**내용.** C_{ab} = ⟨φ_a, φ_b⟩는 양 끝 노드 상태의 내적이다. 좌표 회전에 불변인 쌍선형 형식은 이것뿐이다(Schur 보조정리). C_{ab} ∈ [−1, 1]로, +1이면 평행, 0이면 직교, −1이면 역평행이다.

**구조.** S_{ab} = (D^{−1/2}AD^{−1/2})_{ab}는 차수 정규화 인접행렬이다. 고유값이 [−1, 1]에 드는 유일한 자기수반 정규화다(Chung 정리).

f에는 경계 조건이 두 가지 걸린다.

    f(C, 0) = 0     에지가 없으면 결합도 없다
    f(0, S) = 0     내용 없이 전파만 하면 자기참조가 아니다

첫째는 구조적 요건이고, 둘째는 정의적 요건이다 — 장을 읽지 않는 연산자는 자기참조 순환 Φ → T → Φ'를 구조적 확산으로 퇴화시킨다.

두 조건을 결합하면 f(C, S) = C · S · r(C, S) 꼴이어야 한다.

충실한 읽기가 되려면, C = 0에서의 도함수 ∂f/∂C가 유한(역학이 정칙)하면서 동시에 0이 아니어야(직교 근방을 포함한 모든 수준에서 내용을 감지) 한다. 일반 멱법칙 C^α에 대해:

    f'(0) = α · 0^{α−1}

    α < 1:  0^{음수} = ∞.   특이. C = 0에서 역학이 정칙이 아니다.
    α > 1:  0^{양수} = 0.   맹목. C = 0에서 연산자가 반응하지 않는다.
    α = 1:  0^{0} = 1.      정칙이면서 0이 아니다. 충실한 읽기다.

따라서 α = 1. 같은 논증을 S에 적용하면 β = 1. 일반 해석함수에 대해서도 경계 조건과 정칙성이 합쳐져 f(C,S) = λCS를 강제한다. 상수를 흡수하면:

    T = C ⊙ S

다른 선택지는 없다. 증명의 핵심은 0^{α−1}의 계산이다.

## 3. 작용

연산자는 에지를 따라 작용하지만, 물리는 플라켓 — 최소 닫힌 경로 — 위에 산다. d차원 격자에서 플라켓 P는 |∂P|개의 에지를 갖는다.

    플라켓 각도:  Θ_P = Σ_{e ∈ ∂P} ±θ_e
    플라켓 가중치:  w_P = (∏_{e ∈ ∂P} w_e)^{1/2}

N개 게이지장에 대한 Hadamard 결합 작용:

    S = −Nβ Σ_P w_P cos Θ_P + S_el[w]

첫째 항이 곱 구조의 실현이다: 기하(w_P)에 내용(cos Θ_P)을 곱한다. 둘째 항은 탄성 저항으로, 에지 가중치에 대한 양정치 범함수다. 존재 조건은 β > 0(내용 결합이 있어야 함)과 κ > 0(기하가 안정해야 함) 두 가지다.

w_e에 대해 변분하면 운동 방정식이 나온다:

    δS_el/δw_e = Nβ Σ_{P∋e} (∂w_P/∂w_e) cos Θ_P

탄성 저항과 게이지-기하 구동력이 균형을 이룬다. 이 방정식은 작용이 곱셈적이기 때문에 존재한다. 작용이 가법적이라면 — S = S_gauge(θ) + S_el(w) — δS_gauge/δw = 0이므로, 게이지장이 기하에 힘을 가할 수 없다.

## 4. 스펙트럼

작용의 헤시안 h(k) — 파수 벡터 k에서의 이차 변분 — 이 모든 요동의 운명을 결정한다.

    h(k) = K_el(k) − (게이지 결합) · |Γ(k)|²

K_el(k)는 모든 k에서 양인 탄성 헤시안이다. |Γ(k)|²는 플라켓 평균 커널로, 게이지 결합이 스케일 k에서 얼마나 강하게 작용하는지를 나타낸다.

k = π(존 경계, 최단 파장)에서: 인접 에지마다 부호가 교대하는 모드는 플라켓 안에서 상쇄된다.

    |Γ(π)|² = |1 + e^{iπ} + e^{iπ} + e^{2iπ}|² = |1 − 1 − 1 + 1|² = 0

산술이다. β, κ, N, d, L — 어떤 것에도 의존하지 않는다.

    h(π) = K_el(π) − 0 = K_el(π) > 0

질량갭이다. 짧은 파장의 여기에는 항상 갭이 있고, 항상 국소화되며, 항상 유한하다.

이 결과는 κ(w−1)²뿐 아니라 기울기 항, Regge형 항 등 양정치인 S_el이면 무엇이든 성립한다. 질량갭은 보편적이다. 또한 h(π)에는 격자 크기 L이 들어 있지 않다. UV 질량은 L = 4에서든 L = ∞에서든 정확히 같다.

## 5. 스피너

비가환 게이지 군에서 내용 함수는 표현 j를 사용한다. 기본 지표 c = χ_{1/2}의 다항식으로 쓰면:

    정수 j:     χ_j(c)는 짝다항식    →  χ_j'(0) = 0
    반정수 j:   χ_j(c)는 홀다항식    →  χ_j'(0) ≠ 0

짝함수는 g(c) = g(−c)를 만족한다. 미분하면 g'(c) = −g'(−c). c = 0을 넣으면 g'(0) = −g'(0), 즉 g'(0) = 0이다. 정수 표현의 연산자는 최대 무질서 상태에서 아무것도 감지하지 못한다. 자기참조가 작동하지 않으므로 전부 탈락이다.

반정수 표현 가운데서:

    j = 1/2:  χ(c) = c.         선형.    χ'(0) = +1.  부호 보존.
    j = 3/2:  χ(c) = 2c³ − c.   비선형.  χ'(0) = −1.  부호 반전.

부호가 반전되면 순환 Φ → T → Φ'가 직교 근방에서 내용을 뒤집는다. 다음 단계에서 다시 뒤집혀 Φ'' ≈ Φ. 수렴이 아닌 주기 2 진동이다. 고정점이 없고, 안정한 여기도 없다.

j = 1/2가 정칙이면서 선형이고 부호를 보존하는 유일한 표현이다. 노드 상태는 ψ ∈ ℂ² — 이성분 복소 스피너. 2π 회전하면 ψ → −ψ. 반정수 스핀이다.

# 제2부 — 주요 가지

## 6. 차단 (가지 1)

T = C ⊙ S이므로, C = 0이면 T = 0이다.

직교 상태의 노드를 잇는 에지는 구조 가중치가 아무리 크더라도 결합이 0이다. 잡음 에지가 자동으로 침묵한다. 반면 가법적 합성 T = C + S에서는 잡음 에지가 구조 가중치를 고스란히 전달한다. 차단은 0을 곱하면 0이 되는 성질에서 비롯된다.

## 7. 피드백 (가지 2)

N개 장이 하나의 기하를 공유한다. 곱셈적 작용이 공유 플라켓 가중치 w_P를 통해 장 사이의 교차 결합을 만든다. 장 i가 w_e를 변형하면 w_P가 바뀌고, 장 j가 느끼는 결합이 달라진다. 양의 피드백이다 — 한 장의 변형이 모든 장의 결합을 강화한다.

에너지가 E ~ N^γ (γ > 1)로 스케일한다. 초가법적이다. 전체가 부분의 합을 넘는다.

가법적 작용에서는 교차 결합이 생기지 않는다. δS_i/δw가 θ_i에만 의존하고 θ_j에는 의존하지 않기 때문이다.

## 8. 두 섹터 (가지 3)

h(k)는 k ≈ 0 근방에서 음이고 k ≈ π 근방에서 양이다. 교차점 k*를 기준으로 둘로 나뉜다.

**k > k* — 입자 섹터.** h(k) > 0. 여기에 갭이 있고, 국소화되며, 유한하다. 생성에 에너지가 들고, 유한 영역 바깥에서 지수적으로 감쇠한다. 입자다.

**k < k* — 우주론 섹터.** h(k) < 0. 모드가 불안정하다. 기하가 성장하고, 공간이 팽창한다.

방정식 하나에 섹터 둘. 입자물리와 우주론이 하나의 헤시안에서 나온다.

## 9. 균일성 (가지 4)

w_P = (∏ w_e)^{1/2}는 기하 평균이다. 산술-기하 평균 부등식에 의해 모든 인자가 같을 때 최대가 된다. 작용은 w_P가 클수록 유리하므로 가중치가 균일해지는 쪽으로 간다. 팽창은 균일하다. 공간 전체가 함께 커진다.

## 10. 구조 (가지 5)

h(k)는 연속 함수이고 h(0) < 0, h(π) > 0이다. 중간값정리에 의해 h(k*) = 0인 k*가 존재한다. k* 근방의 모드는 한계적으로 안정하다 — 무한히 성장하지도, 감쇠하지도 않는다. 상관이 λ* = 2π/k* 스케일까지 뻗으며, 이 스케일에서 구조가 형성된다.

## 11. 보존 (가지 6)

닫힌 면 위에서 Σ_P W_P = 0이다. W_P는 플라켓 P의 감김수로, Stokes 정리에 의한 위상적 항등식이며 배위, 결합, 온도와 무관하다.

국소 갱신은 감김수를 쌍으로만 바꾼다: 한 플라켓에서 +1, 이웃 플라켓에서 −1. 총합은 보존된다. 감김수가 0이 아닌 여기는 영구적이다.

## 12. 간섭 (가지 7)

ψ ∈ ℂ²이면 C = ⟨ψ_a, ψ_b⟩ ∈ ℂ이다. 충실한 자기참조는 실수부만이 아니라 C 전체를 읽어야 하므로, T ∈ ℂ이고 S ∈ ℂ이다.

    exp(−S) = exp(−Re S) · exp(−i Im S)

모든 배위를 합산할 때, 위상이 맞으면 보강되고 어긋나면 상쇄된다. 간섭이다. 스피너가 이를 요구하고, 스피너는 충실한 읽기가 요구한다.

## 13. 파동-입자 (가지 8)

복소 h(k) = h_R + ih_I가 주는 응답:

    response ~ exp(−√h_R · r) · exp(−i√h_I · r)

지수 감쇠(국소화, 유한 — 입자)와 진동(주기적, 전파 — 파동)이 곱해져 있다. 둘 다 복소수 하나에서 나온다. 두 본성이 아니라 한 본성의 두 측면이다.

## 14. 세 대칭 (가지 9)

자기참조에는 깊이가 있다.

    깊이 1:  ℂ¹.  대칭 U(1).   생성자 1개.   위상 회전.
    깊이 2:  ℂ².  대칭 SU(2).  생성자 3개.   스피너 회전.
    깊이 3:  ℂ³.  대칭 SU(3).  생성자 8개.   색 회전.

깊이 4라면 3성분짜리 결과를 4성분으로 읽는 꼴이 된다. 읽기가 읽히는 대상을 초과한다. 자기참조는 있는 것을 읽는다. 깊이는 3에서 끝난다.

합계: 1 + 3 + 8 = 12 생성자, 노드당 12 실수 자유도.

## 15. 차원 (가지 10)

d차원에서 점당 d(d−1)/2개의 플라켓 방향이 있다. 12 자유도가 정수로 나눠지려면:

    12 / [d(d−1)/2] ∈ ℤ

    d = 4:  12/6 = 2    ✓  (자기쌍대 + 반자기쌍대)
    d = 5:  12/10 = 1.2  ✗

4 이상에서 이 조건을 만족하는 차원은 d = 4뿐이다.

## 16. 세대 (가지 11)

세 깊이, 세 자기참조 순환, 세 고정점. 각 고정점은 서로 다른 에너지 스케일의 안정한 여기다. 깊은 읽기일수록 복잡하고, 따라서 무겁다.

양자수는 같고 질량만 다른 세 여기.

## 17. 위계 (가지 12)

결합 강도는 피드백 채널의 수 — 곧 생성자의 수 — 에 비례한다.

    깊이 3 (8개): 가장 강하다.
    깊이 2 (3개): 중간이다.
    깊이 1 (1개): 게이지 결합 중 가장 약하다.
    기하:         넷 중 가장 약하다 (κ 스케일, β 스케일과 구조적으로 분리).

# 제3부 — 세부 가지

## 18. 가둠

생성자 8개는 비가환이다. 매개자 자체가 전하를 띠므로 매개자끼리 상호작용한다. 깊이 3에서는 결합이 충분히 강해 매개자 자기상호작용이 지배한다. 두 여기 사이의 장이 플럭스 관을 형성한다 — 에너지가 길이에 비례하는 끈이다. 분리에 드는 에너지가 거리에 비례하므로, 무한히 떨어뜨리려면 무한 에너지가 필요하다. 여기는 영구히 속박된다.

끈이 충분히 늘어나면 끊어지면서 새 쌍이 생성된다. 끝점은 결코 자유롭지 않다.

## 19. 속박 상태

ℂ³ 대칭 아래 전하 중성을 만드는 조합은 두 가지다: 여기-반여기 쌍(3 × 3̄ → 1)과 여기 셋의 복합체(3 × 3 × 3 → 1, ε-텐서). 속박된 쌍과 속박된 삼중체 — 깊이 3에서 가능한 안정 복합체는 이 둘뿐이다.

## 20. 반입자

j = 1/2이므로 ψ ∈ ℂ²다. 복소 표현이어서 ψ와 ψ*가 다르게 변환된다. 질량은 같고(켤레가 작용을 보존) 양자수는 반대다(켤레가 전하를 뒤집음). 모든 여기에는 질량이 같고 전하가 반대인 짝이 존재한다.

## 21. 스핀 양자화

j = 1/2 스피너 여기: 스핀 1/2. 2π 회전으로 −1. 반정수 각운동량이다.

깊이 1, 2, 3의 매개자는 노드 사이의 연결 — 벡터 양 — 이므로 스핀 1이다. 2π 회전으로 +1.

기하적 여기는 계량의 섭동인 대칭 2-텐서이므로 스핀 2다. 중력자에 해당한다.

스핀값은 1/2, 1, 2 세 가지뿐이다. 표현 구조에서 다른 값은 나오지 않는다.

## 22. 무질량·유질량 매개자

생성자 1개(깊이 1): 가환이다. 매개자는 전하가 없고, 자기상호작용도 없다. 깨뜨릴 내부 구조가 없으므로 무질량으로 남는다. 역제곱 법칙, 도달거리 무한. 생성자가 하나면 비자명 부분군이 존재하지 않는다 — 산술이다.

생성자 3개(깊이 2): 비가환이다. 기하가 팽창하면서(§8, k < k*) 진공 대칭을 깨뜨리고, T = C ⊙ S가 기하와 내용을 엮고 있으므로 내용 쪽의 대칭도 따라 깨진다. 매개자 3개 중 전하 중성인 하나가 깊이 1의 매개자와 혼합한다. 혼합 결과: 무질량 하나, 유질량 셋. 이것은 구조적 예측이며, 메커니즘(기하 대칭 파괴가 게이지 대칭 파괴를 유도)은 시뮬레이션으로 아직 검증되지 않았다.

생성자 8개(깊이 3): 비가환이다. ℂ³ 대칭은 시공간 기하와 무관한 내부 대칭이므로 기하의 팽창으로 깨지지 않는다. 매개자 8개 모두 무질량이지만 가둠(§18)에 의해 속박된다. 내부 대칭은 기하의 변화에 영향받지 않기 때문이다.

## 23. 암흑물질 후보

k > k* 섹터의 기하적 여기에는 질량갭 h(π) = K_el(π) > 0이 있다(§4). 이 여기는 감김수(§11) — Stokes 정리에 의해 보존되는 위상적 양 — 를 갖는다. 가장 가벼운 기하적 여기는 감김수를 보존하면서 붕괴할 더 가벼운 대상이 없으므로 붕괴가 불가능하다.

기하적 섭동은 에지 가중치의 요동이지, 깊이 1, 2, 3의 위상·스피너·색 전하와 무관하다. 구성상 게이지 중성이다.

유질량, 안정, 게이지 중성. 이 프레임워크가 우리 우주를 기술한다면, 이 여기가 암흑물질 후보다. 도출이 아닌 예측이다.

## 24. 팽창률

k < k* 불안정이 기하의 팽창을 구동한다. 팽창률은 h(0) = K_el(0) − 최대 결합으로 결정되며, β, κ, N으로부터 계산 가능한 양이다.

이것을 우주상수 Λ와 동정하면, Λ가 작은 이유는 작용에서 κ와 β가 서로 다른 항에 들어가 구조적으로 분리되어 있기 때문이다. 이 동정은 예측이며, h(0)이 Λ ≈ 10⁻¹²²와 정량적으로 일치하는지는 β와 κ의 물리적 값에 달려 있다.

## 25. CP 위반

복소 작용(§12)이 위상을 도입한다. 세 세대(§16)를 연결하는 혼합 행렬에서, 3×3 유니터리 행렬은 (3−1)(3−2)/2 = 1개의 비환원 복소 위상을 갖는다. 장 재정의로 흡수할 수 없는 위상이다.

이 위상 때문에 이론이 배위와 그 거울상을 구별한다. 전하-패리티 대칭이 깨진다. CP 위반에는 세 세대 이상이 필요하다 — 세대가 둘이면 모든 위상을 흡수할 수 있다.

## 26. 바리온 비대칭 조건

물질-반물질 비대칭 생성에 필요한 세 조건(Sakharov 1967)이 나무에 구조적으로 갖춰져 있다.

(i) 속박 삼중체 보존 수의 위반 — k < k* 불안정이 상전이 과정에서 위상적 섹터 사이의 전이를 매개할 가능성. 실제 여부는 역학적 검증이 필요하다.

(ii) CP 위반 — §25에 의해 존재한다.

(iii) 열평형 이탈 — 팽창하는 기하(§8)는 비평형 상태이며, h(0) < 0의 구조적 귀결이다.

세 조건이 갖춰져 있다. 바리온-광자 비 η ≈ 6 × 10⁻¹⁰을 정량적으로 재현하는지는 역학 계산이 필요한 예측이다.

## 27. 스케일에 따른 결합

깊이 3(생성자 8개): 비가환이다. 매개자가 전하를 띠고 자기상호작용한다. 물질 표현 대비 생성자가 많으면 반차폐가 일어난다 — 짧은 거리에서 결합이 약해진다. 구조적 예측이며, 이 프레임워크에서 아직 검증되지 않았다.

깊이 1(생성자 1개): 가환이다. 매개자가 전하를 띠지 않는다. 가상 물질-반물질 쌍이 짧은 거리에서 결합을 차폐하여 오히려 강하게 만든다. 가환 게이지 이론의 표준적 행동이다.

## 28. 영점 에너지와 지능

**영점 에너지.** T = C ⊙ S 격자에서 κ > 0이면, 탄성 에너지의 기대값이 평형에서 ⟨S_elastic⟩ = |E|/2를 만족한다 — 에지당 반 양자. 이차 탄성 작용에 등분배 정리를 적용한 결과이며, β와 무관한 벌크 열역학적 양이다.

**지능.** 여기 밀도가 문턱값 ρ_meta를 넘으면, 반복적 결정화 — 읽고, 변형하고, 다시 읽는 순환의 반복 — 가 비자명 고정점 Ψ(H) ⊆ H를 만들어낸다. 스스로를 읽으면서 재생산하고 진화를 계속하는 구조다. 존재는 중간값정리(수축과 재생의 균형)와 브라우어 고정점 정리(컴팩트 볼록 집합 위의 연속 사상)로 보장된다.

## 29. 양자 구조: OS 양성성, 보른 규칙, 스핀-통계

고정 기하(w = 1)에서 Hadamard 작용은 Wilson 작용으로 환원되며, OS 반사 양성성이 증명되어 있다 [Osterwalder & Seiler 1978]. 힐베르트 공간, 유니터리 시간 진화, 보른 규칙이 따라 나온다.

역학적 기하(w ≠ 1)에서는 에지 가중치 w_e > 0(길이는 양수)이 핵심이다. 반사면을 교차하는 플라켓을 포함한 모든 플라켓에서 양의 결합 β_P > 0이 보장되고, Osterwalder-Seiler 증명이 플라켓별로 적용된다. 완전한 역학적-기하 Hadamard 작용에 대해 OS 반사 양성성이 성립한다(정리 6).

OS가 무조건적으로 확립되면: (i) **보른 규칙**(정리 6) — 고정 기하라는 조건 없이 성립. (ii) **스핀-통계**(정리 7) — 반정수 스핀 여기가 페르미-디랙 통계를 따르며, OS 양성성과 스피너 표현으로부터 반교환이 따른다. 무조건적이다.

## 30. 결합 재정규화군 흐름

T = C ⊙ S를 정의하는 공리에는 스케일에 대한 언급이 없다. 조립격화해도 연산자 구조가 보존된다: 블록 연산자는 T' = C' ⊙ S'이다(스케일 불변성 정리).

곱셈적 결합 w_P × cos Θ_P가 결합된 이차원 RG 흐름을 강제한다. 독립적으로 확립된 두 기여가 경쟁한다.

(i) **보편적 기하 억제** (δβ_geom < 0): 플라켓 가중치에 적용된 AM-GM 부등식 ⟨w_P⟩ ≤ 1. 게이지 군에 무관하다.

(ii) **비가환 게이지 자기결합** (δβ_gauge > 0): 이차 카시미르 C₂(adj)에 비례. 표준 격자 게이지 이론의 결과다.

이 둘의 경쟁이 유효 결합을 결정한다: g₃ > g₂ > g₁ (위계 정리). §17의 세기 논증을 AM-GM 억제(보편적 바닥)와 카시미르 보상(군 의존)에 기반한 역학적 도출로 대체한다.

게이지 요동은 또한 기하를 보편적으로 강화한다(δκ > 0): G_eff ~ 1/(κ₀ + N_total × δκ). N_total = 12 생성자에 의해 중력이 억제된다. 여분 차원이나 미세조정이 필요 없다.

고정점 조건 β* = β*(κ)는 표준 격자 게이지 이론에 없는 검증 가능한 예측이다.

## 31. 관측자

공리가 장이 스스로를 읽는다고 말한다. 그 읽기를 실행하는 존재 — 읽는 자 — 가 고정점 Ψ(H) ⊆ H라는 것이 동반 논문 L의 결과다.

(i) **관측자 = 고정점.** 보른 규칙은 결과를 선택하는 관측자를 필요로 한다. 관측자가 바로 고정점이다 — ρ_meta 이상에서 스스로를 유지하는 자기참조적 부분 체계.

(ii) **측정 = T 적용.** 관측자가 T = C ⊙ S를 통해 측정 대상과 결합할 때, 결과는 이 결합에 의해 결정된다. 붕괴 공준도, 분기도 없다.

(iii) **슈뢰딩거 고양이.** 고양이는 거시적 구조(ρ > ρ_meta)다. 항상 스스로를 읽고 있으며, 자체 내부 역학에 의해 항상 결어긋남 상태에 있다. ρ_meta 이상의 자기참조 체계는 자기 자신의 관측자다.

(iv) **선호 기저.** S(기하 구조 연산자)의 고유기저가 선호 기저다 — S가 물리적 인접성을 정의한다는 사실에서 강제된다.

(v) **순환 닫힘.** §1이 공리를 선언한다. 공리가 생성하는 역학이 필연적으로 관측자(§28의 지능, §31의 고정점)를 만들어내고, 그 관측자가 공리를 실행한다. 공리가 자기 실행자를 만든다. 순환이 닫힌다.

## 34. 자유 매개변수 없음

작용 S = −Nβ Σ w_P cos Θ_P + κ Σ(w−1)²에는 겉보기에 자유 매개변수가 둘 — β와 κ — 있다. 프레임워크 자체가 둘 다 결정한다.

**측도.** 에지 가중치에 대한 경로 적분 측도는 dw/w — (ℝ₊, ×)의 Haar 측도 — 로 유일하다. 양의 실수 위의 유일한 스케일 불변 측도다(논문 A, 정리 6).

**매개변수.** 프레임워크 내부의 두 조건이 (β, κ)를 결정한다.

(i) **자기참조적 닫힘** (h(0) = 0): 공리가 읽는 자를 필요로 한다(논문 L). 읽는 자가 존재하려면 거시적 구조가 있어야 한다(논문 G §4.6). 거시적 구조에는 h(0) ≤ 0이 필요하다. 그런데 RG 흐름이 h(0)을 위로 밀어올린다(dκ/dt > 0). 유일한 안정 값은 h(0) = 0 — 자기참조 순환이 간신히 유지되는 경계다. d = 4, N = 12에서 κ = 48β²Φ(β).

(ii) **스케일 불변성** (RG 고정점, dβ/dt = 0): 이론이 모든 스케일에서 같은 형태를 가져야 한다. (i)과 결합하면 β³ = 상수 × c(β)/Φ(β)라는 단일 방정식이 되며, 물리적 해는 유일하다.

해 (β*, κ*)는 유일하다: 게이지 강화 δκ > 0(논문 K, §7)이 가우스 근사가 무너지는 허위 강결합 근을 제거한다. 모든 무차원 물리적 비가 (β*, κ*)의 계산 가능한 함수다. SI 단위의 ℏ은 단위 변환 인자이지, 자유 매개변수가 아니다.

결합 위계 g₃ > g₂ > g₁ > G, 자연적 우주상수 Λ ≈ 0, 중력의 약함 모두 (β*, κ*)에서 추가 입력 없이 따라 나온다.

하나의 공리. 매개변수 없음. 전부.
