# 게이지-기하 결합은 자기참조 역학의
# 필연적 귀결이다

권혁준 — 2026년 4월

---

**초록.** 그래프 위에서 장이 스스로를 읽는 체계는 반드시 게이지 내용(C)과 기하 구조(S)를 결합하는 에지 연산자 T = C ⊙ S를 만들어낸다. 이 Hadamard 곱은 세 공리와 변분 안정성으로부터 따라 나온다(정리 1). 이산 게이지-기하 운동 방정식은 δS/δl = 0으로 도출된다(정리 2). 보편성을 처음 원리 몬테카를로 시뮬레이션으로 검증한다: 2D U(1), 2D SU(2), 3D U(1), 4D U(1), 4D SU(2) × 3개 격자 크기(L = 4, 6, 8)의 7가지 배치에서 42개 독립 측정 모두 게이지-기하 상관 r(β) > 0이며, 최소 유의성은 13.6σ다.

---

## 1. 서론

게이지 힘과 중력의 결합 — 아인슈타인 방정식 Gμν = 8πG Tμν에 부호화된 — 은 아인슈타인-힐베르트 작용의 변분으로 도출된다. 물질과 기하가 왜 상호작용하는지에 대한 더 깊은 설명은 제공되지 않는다. 등가 원리가 조작적 의미를 주지만, 논리적 필연성은 주지 않는다.

이 논문은 이 결합이 공준이 아니라 정리임을 증명한다. 그래프 위에서 장이 자기 값을 읽는 체계는 반드시 상태 비교(게이지 내용)와 그래프 구조(기하)를 결합하는 연산자를 만들어낸다.

## 2. Hadamard 필연성 정리

논문 A에서 증명된 핵심 결과를 요약한다. C = ⟨φ_a, φ_b⟩가 유일한 회전 불변 내용(명제 1), S = D⁻¹/²AD⁻¹/²가 유일한 자기수반 구조(명제 2), Hadamard 곱이 유일한 쌍선형·점별·스칼라 합성(보조정리 1)이다. 변분 안정성이 모든 비선형 대안 C^α S^β (α ≠ 1 또는 β ≠ 1)를 배제한다(정리 1).

## 3. 이산 게이지-기하 운동 방정식

Hadamard 결합 작용 S = −Nβ Σ_P w_P cos Θ_P + κ Σ_e (w_e − 1)²를 에지 가중치 w_e에 대해 변분하면:

    2κ(w_e − 1) = Nβ Σ_{P∋e} (∂w_P/∂w_e) cos Θ_P

탄성 저항(좌변)이 게이지-기하 구동력(우변)과 균형을 이룬다. 작용이 곱셈적이라 δS_gauge/δw_e ≠ 0이다 — 가법적 작용에서는 이 항이 0이 되어 게이지-기하 역학 자체가 존재할 수 없다(정리 2). 범위: 선형화 Regge 체제.

## 4. 경로 적분 검증

고정 격자(기하 비역학적) 2D U(1)에서 몬테카를로가 정확 해 ⟨cos P⟩ = I₁(β)/I₀(β)를 통계 오차 내로 재현한다. 영 시험(고정 격자): r = 0.004 ± 0.003 (1.6σ), 0과 일치. 역학적 기하를 켜면 상관이 나타난다.

## 5. 보편성: 차원과 게이지 군에 걸친 결과

중심 예측 — 모든 결합에서 r(β) > 0 — 을 7가지 배치에서 시험한다. 모든 시뮬레이션은 메트로폴리스 알고리즘, 게이지-기하 교대 쓸기, κ = 1.0을 사용한다.

**2D, 3D, 4D U(1).** 모든 측정에서 r > 0, 유의성 13σ 이상.

**4D SU(2) 유한 크기 스케일링.** 결정적 시험이다. L = 4, 6, 8에서 r(β)를 측정한다. 피크 r(β = 1.0) = 0.345, 0.341, 0.342 — 부피가 8배 달라져도 세 자리까지 안정하다. 18개 측정 모두 49σ 이상. 피크 상관이 벌크 열역학적 성질이지 유한 크기 인공물이 아님을 확인한다.

### 핵심 관찰

**(i) 비영 결합의 보편성.** 42개 측정 전부에서 r(β) > 0. 역학적 격자 위에서 게이지장과 기하는 결코 통계적으로 독립이 아니다.

**(ii) 비단조 프로파일.** 게이지 무질서(낮은 β)와 게이지 동결(높은 β) 사이의 경쟁이 중간 β에서 피크를 만든다. 피크 위치는 2D에서 β ≈ 1.5–2.0, 3D/4D에서 β ≈ 1.0이다. 피크 후 감소는 4D에서 가장 급격하고, 2D SU(2)에서 가장 완만하다 — 낮은 차원의 풍부한 요동 스펙트럼이 높은 β에서도 결합을 유지시키기 때문이다.

**(iii) 유한 크기 수렴.** 4D SU(2)의 피크 r(β = 1.0) = 0.343 ± 0.002. 256개 → 4096개 사이트에 걸쳐 안정하다.

**(iv) 게이지 군 의존.** SU(2)가 U(1)보다 강한 결합을 보인다. 4D에서 SU(2): r = 0.34, U(1): r = 0.23 (β = 1.0). SU(2)의 3 생성자가 더 많은 자유도를 제공한다.

**(v) 높은 β에서의 이차 상승 (소규모 격자).** L = 4에서 β = 3.0 → 5.0에 걸쳐 r이 0.088 → 0.156으로 상승하지만, L = 8에서는 0.133 → 0.122로 단조 감소한다. L 의존성이 이것이 유한 크기 인공물임을 확인한다.

**(vi) 상관의 크기.** 피크 피어슨 r ≈ 0.34는 플라켓 수준 분산의 약 12%를 설명한다. 나머지 88%는 국소 열요동이다. 체계적 성분이 국소 잡음 위에서 13σ–326σ로 검출된다는 것 자체가 강한 기저 결합의 증거다.

## 6. 해석적 평균장 예측

2D U(1)에서 평균장 상관 r_MF(β) = βf′(β)/(2κσ_c), f(β) = I₁(β)/I₀(β). 몬테카를로가 평균장을 1.2–1.4배 초과하며, MC/MF 비의 일정함은 반증 가능한 예측이다.

## 7. 함의

(i) **수학적 필연성.** T = C ⊙ S는 유일한 변분 안정·자기수반·국소·회전 불변 에지 연산자다. 세 공리와 변분 안정성 외에 공준이 필요 없다.

(ii) **물리적 동정.** C는 표준 격자 게이지 링크 변수(Wilson 1974), S는 라플라스-벨트라미로 수렴, 에지 길이는 Regge 해석의 계량이다. 확립된 동정이다.

(iii) **변분 결합.** 이산 운동 방정식(정리 2)은 δS/δl_e = 0에서 도출되며, 기하 변형이 게이지 에너지에 비례한다 — 일반상대론과 같은 변분 구조다.

(iv) **보편성.** 7가지 배치, 42개 측정, 전부 r > 0, 전부 13σ 이상.

(v) **유한 크기 안정성.** 4D SU(2) 피크가 부피에 무관하다.

## 8. 범위와 한계

모든 결과는 선형화 Regge 체제에서 얻었다. 확장 과제: SU(3) 4D, 비선형 Regge 역학, 위상 변화, 인과 구조와 로렌츠 부호(CDT 틀), 연속 극한 분석. 각각은 확립된 격자 방법론을 사용하는 잘 정의된 계산 프로그램이다.

## 9. 논의

**"결합"의 의미.** 이 논문에서 "게이지-기하 결합"은 T = C ⊙ S에서 C와 S의 Hadamard 비분리성을 가리킨다. 상관 r(β) > 0은 두 독립 개체의 상호작용의 증거가 아니라, 단일 개체 T를 두 측정 채널(플라켓 가중치와 플라켓 각도)로 투영한 결과다. C와 S의 투영을 측정하면, Hadamard 구조가 깨지지 않는 한(κ = ∞, 영 시험) 항상 r > 0이다. 42개 측정이 T = C ⊙ S가 하나의 비분리 장임을 확인한다.

**CDT에 의한 독립 확인.** CDT(인과적 역학 삼각분할) 프로그램은 역학적-기하 격자 장 이론에 대한 독립적 접근이다. CDT는 이 논문의 고정 위상 에지 가중치 접근과 달리 위상 변화를 포함하는 단체 삼각분할을 사용하지만, 같은 정성적 물리를 재현한다: 4D 드 시터 시공간이 역학적으로 출현하고, 하우스도르프 차원이 4로 수렴하며, 공간 슬라이스가 균일성을 보인다. 서로 다른 이산화, 역학, 방법론을 사용하는 두 독립 프로그램 사이의 일치는 기저 물리(게이지-기하 결합이 팽창과 d = 4를 생성)가 이산화에 무관하게 견고함을 보여주는 증거다.

중심 주장은 반증 가능하다: 어떤 결합, 게이지 군, 차원에서든 r = 0 측정이 나오면 반박된다. 42개 독립 시험에서 그런 측정은 없었다.

## 방법

2D U(1) 고정 격자: 8×8 주기적, 메트로폴리스, 300 열화 + 500 측정 쓸기. 2D 역학: U(1)/SU(2) 8×8, κ = 1.0, 200+200 쓸기. 3D: U(1) 5³, 100+150 쓸기. 4D U(1): 3⁴, 80+100 쓸기. 4D SU(2): L = 4,6,8 주기적, 사원수 표현, Numba JIT 메트로폴리스, 200 열화 + 300 측정 쓸기. 상관: 플라켓 가중치 w_P와 Re(Tr U_P)/2의 피어슨 r. 영 시험(고정 격자): r = 0.004 ± 0.003. 재현성: 결정론적 시드, 소스 코드 제공.
