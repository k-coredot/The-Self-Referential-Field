# 자기참조적 그래프의 Hadamard 구조:
# 공리적 도출, 유일성, 변분 구조

권혁준 — 2026년 4월

---

**초록.** 그래프 위의 자기참조적 역학에 대한 공리적 틀을 세우고, 에지 진화 연산자가 T = C ⊙ S로 유일하게 결정됨을 증명한다(정리 1). C는 상태 내적, S는 자기수반 정규화 인접행렬이다. 세 공리(국소 자기참조, 회전 불변성, 자기수반 스펙트럼 정합성)와 변분 안정성(약장 극한에서의 유한하고 0이 아닌 응답)으로부터, (α,β) ≠ (1,1)인 멱법칙 C^α S^β뿐 아니라 exp(CS)나 C·log(S) 같은 비다항식 합성까지 배제된다. Hadamard 곱이 공리를 만족하는 유일한 국소 합성임을 보이고(보조정리 1), 이산 게이지-기하 운동 방정식(정리 2), 섭동 하의 스펙트럼 수렴(정리 3), 스케일 불변성(정리 4), 비대칭 단조감소(정리 5), 경로 적분 측도의 유일성(정리 6)을 증명한다.

## 자기참조란 무엇인가

이 논문이 다루는 체계는 간단하게 설명할 수 있다. 그래프 G = (V, E)가 있고, 각 노드에 상태가 있으며, 에지를 따라 상태를 전파하는 연산자 T가 있다. 핵심은 T가 고정된 것이 아니라 노드 상태와 그래프 구조에 의해 결정된다는 것이다: T = T[Φ, G]. 연산자가 장을 읽고, 장이 연산자를 결정하는 피드백 루프 — 이것이 자기참조의 정의다.

이 논문의 질문은: 이 피드백 루프를 만족하는 연산자가 얼마나 있는가? 답은 하나뿐이다.

## 세 가지 공리

연산자 T에 세 가지 조건을 건다.

**공리 1 — 국소성.** T_{ab}는 에지 양 끝의 상태 φ_a, φ_b와 주변 그래프 구조에만 의존한다. 먼 곳의 정보를 참조하지 않는다.

**공리 2 — 회전 불변성.** 좌표를 회전해도 T가 달라지지 않는다. 물리는 좌표 선택에 의존하지 않는다.

**공리 3 — 자기수반 스펙트럼 정합성.** 구조 전파 행렬이 대칭이고, 차수 불균형 아래에서도 그래프 라플라시안의 스펙트럼 성질을 보존한다. 이것은 실수 고유값(관측 가능량), 직교 고유벡터(분해 가능성), 가역적 역학(에너지 보존)을 보장한다.

## 성분의 유일성

공리 2에 의해 내용 쪽이 결정된다. Schur 보조정리에 의하면, SO(d) 표준 표현이 기약일 때, 회전 불변 쌍선형 형식은 스칼라 배를 제외하면 내적 ⟨φ_a, φ_b⟩뿐이다(명제 1). RBF 커널 같은 비쌍선형 불변량도 존재하지만, 이들은 뒤에 나오는 변분 안정성에 의해 배제된다 — 쌍선형성이 독립적 가정이 아니라 공리의 귀결이라는 점이 중요하다.

    C_{ab} = ⟨φ_a, φ_b⟩

공리 3에 의해 구조 쪽이 결정된다. 차수 의존 정규화 S = D^α A D^β에서 자기수반성은 α = β를 요구하고, 정규화 라플라시안의 스펙트럼이 [0, 2]에 들려면 α = −1/2여야 한다(Chung 정리). 확률보행 행렬 D⁻¹A는 같은 고유값을 갖지만 자기수반이 아니므로 탈락한다(명제 2).

    S_{ab} = (D^{−1/2}AD^{−1/2})_{ab}

## 합성의 유일성: 왜 T = C ⊙ S인가

C와 S가 결정되었으니, 둘을 어떻게 결합하는지가 남는다. 공리 1(국소성)에 의해 T_{ab} = h(C_{ab}, S_{ab}), 즉 에지당 스칼라 두 개의 함수다. 합성이 점별적(에지마다 독립)이어야 한다는 점이 대안을 크게 좁힌다. 행렬 곱 (CS)_{ab} = Σ_c C_{ac}S_{cb}는 모든 노드 c를 참조하므로 국소성에 위배된다. 텐서 곱 C ⊗ S는 에지당 스칼라가 아닌 행렬을 만든다. 직접 합 T = C + S는 δT/δC = 1이어서 내용과 구조 사이에 곱셈적 결합이 생기지 않는다. 따라서 점별적·쌍선형 합성은 Hadamard 곱 T = λ(C ⊙ S)뿐이다(보조정리 1).

**경계 조건 두 가지.** 에지가 없으면(S = 0) 결합도 없다: h(C, 0) = 0. 내용이 없으면(C = 0) 전파도 없다: h(0, S) = 0. 장을 읽지 않고 전파만 하는 연산자는 자기참조가 아니기 때문이다. 이 두 조건으로부터 h(C, S) = C · S · r(C, S) 꼴이 따르며, r이 상수임을 보이면 된다.

**멱법칙의 경우(경우 1).** f(C) = C^α를 넣으면 도함수는 α · 0^{α−1}이다.

    α < 1:  도함수 발산. C = 0에서 힘이 무한대 — 역학이 특이적이다. 피카르-린델뢰프 정리의 리프시츠 조건에 실패한다. 배제.

    α > 1:  도함수가 0. 장이 C ≈ 0 근방에서 미세하게 바뀌어도 연산자가 반응하지 않는다. 자기참조의 정의에 위배된다 — 장이 바뀌면 연산자가 그것을 감지해야 한다. 배제.

    α = 1:  도함수가 1. C = 0에서 차단(T = 0, 잡음 면역)이 성립하면서도, 미세한 변화는 감지한다. 자기참조 순환이 온전하다.

같은 논증을 S에 적용하면 β = 1. 따라서 멱법칙에 대해 T = C ⊙ S.

**일반 해석함수의 경우(경우 2).** C + εC³이나 sin(C) 같은 비선형 대안을 배제해야 한다.

자기참조 순환 Φ → T → Φ'가 안정한 고정점에 수렴하려면, C = 0 근방에서 내용 응답 f'(0)의 부호가 양이어야 한다. f'(0) < 0이면 양의 섭동이 음의 응답을 만들고, 다음 단계에서 다시 양으로 돌아온다 — 주기 2 진동이지 수렴이 아니다. 이 논증은 C = 0뿐 아니라 모든 C에 적용되므로, f'(C) > 0이 (−1, 1) 전체에서 성립해야 한다.

f(C) = C + εC³를 보자. f'(C) = 1 + 3εC²인데, ε > 0이면 C = 0 바깥에서 이득이 증폭되고, ε < 0이면 어딘가에서 f' = 0이 되어 맹목이 생긴다. 어느 쪽이든 내용에 따라 읽기의 감도가 달라진다. 유일하게 내용 독립적인 이득은 f'(C) = 상수이며, 이것은 f(C) = λC를 준다. 상수를 흡수하면:

    T = C ⊙ S

다른 가능성은 없다.

*비고.* 경우 1(멱법칙)은 C = 0에서의 정칙성과 자기참조 정의만으로 성립하는 수학 정리다. 경우 2(일반 해석함수)는 부호 보존과 이득 균일성을 추가로 사용하는데, 둘 다 자기참조 순환에서 도출된다. 경우 2의 논증을 받아들이지 않더라도, 멱법칙 유일성은 그대로 성립한다. 남는 가능성은 f(C) = C + εC³ 같은 작은 비선형 보정인데, 이 경우에도 물리적 결론은 ε에 연속이어서 정성적으로 달라지지 않는다(동반 논문 H Math, 정리 10).

## 게이지-기하 운동 방정식

연산자에서 작용으로 넘어간다. N개 게이지장에 대한 Hadamard 결합 작용은:

    S = −Nβ Σ_P w_P cos Θ_P + κ Σ_e (w_e − 1)²

에지 가중치 w_e에 대해 변분하면 이산 운동 방정식이 나온다(정리 2):

    2κ(w_e − 1) = Nβ Σ_{P∋e} (∂w_P/∂w_e) cos Θ_P

좌변은 탄성 저항, 우변은 게이지-기하 구동력이다. 이 방정식이 존재하는 이유는 작용에서 기하(w_P)와 내용(cos Θ_P)이 곱해져 있기 때문이다. 작용이 가법적이라면 — S = S_gauge(θ) + S_el(w) — 게이지 항을 w로 미분하면 0이므로, 게이지장이 기하에 힘을 가할 수 없다.

게이지와 기하가 역학적으로 분리 불가능하다는 것(따름정리 1)은 T = C ⊙ S의 직접적 귀결이다.

## 스펙트럼 수렴

에지 가중치가 역학적으로 변할 때, 정규화 라플라시안의 스펙트럼이 망가지지 않는지가 문제다. 정리 3이 이를 보장한다: Belkin-Niyogi 조건 아래, 에지 가중치 섭동이 유계이면 고유값이 연속 라플라스-벨트라미 연산자의 고유값으로 수렴한다. Weyl 섭동 정리에 의한 결과다. 역학이 스펙트럼 구조를 파괴하지 않으므로 프레임워크의 연속 극한이 정합적이다.

## 스케일 불변성과 비대칭 역학

**스케일 불변성(정리 4).** 공리에는 길이나 에너지 스케일에 대한 언급이 없다. 블록 스핀 변환으로 격자를 조립격화해도 같은 공리가 성립하고, 블록 격자의 연산자도 T' = C' ⊙ S'가 된다. 이론이 모든 스케일에서 같은 형태를 갖는다.

이것은 결합 RG 흐름의 출발점이다 — 조립격화가 유효 결합 β'(β, κ)와 κ'(β, κ)를 정의하며, Hadamard 구조가 이 둘의 결합을 강제한다. 상세한 전개는 동반 논문 K에 있다.

**비대칭 단조감소(정리 5).** 과감쇠 역학 dw_e/dt = −dS/dw_e 아래, 에지 가중치의 비대칭 A = Σ(w_e − w̄)²이 단조 감소한다: dA/dt ≤ 0. 역학 자체가 기하를 균일하게 만드는 경향이 있다.

## 경로 적분 측도의 유일성

에지 가중치 w_e는 길이를 나타낸다. 길이는 양의 실수이고, 양의 실수는 곱셈으로 군을 이룬다: (ℝ₊, ×). 이 군의 Haar 측도 — 유일한 스케일 불변 측도 — 는 dw/w다(정리 6). 르베그 측도 dw는 w → λw 아래 불변이 아니지만, dw/w는 불변이다.

이 측도는 Regge 해석에서의 Misner 측도 ∏ dl_e/l_e와 정확히 일치한다. 새로운 선택이 아니라 확립된 동정이다.

이로써 분배 함수가 완전히 지정된다:

    Z = ∫ [dU] ∏_e (dw_e/w_e) exp(−S[U, w])

게이지 군의 컴팩트 Haar 측도 [dU]와 ℝ₊의 Haar 측도 dw/w. 이론의 정의에 자유 선택이 남지 않으며, 유일하게 남는 것은 작용에 나타나는 β와 κ인데, 이 둘은 자기정합 조건에 의해 결정된다(동반 논문 K).

## 물리적 동정

C = cos θ = Re(e^{iθ})는 Wilson(1974)이 정의한 U(1) 격자 게이지 변수와 동일하다. 유사가 아니라 같은 수학적 대상이 같은 역할을 한다. S는 Regge 해석에서 에지 길이가 계량을 부호화하는 것에 대응한다(정리 3에 의해 연속 라플라스-벨트라미로 수렴).

따라서 T = C ⊙ S는 게이지 내용과 기하 구조를 에지마다 결합시키는 연산자이며, 이것은 아인슈타인 장 방정식이 공준하는 게이지-기하 결합의 이산 버전이다. 차이는 여기서는 증명이라는 것이다.

## 여섯 결과와 그 귀결

이 논문은 여섯 가지를 수학적으로 증명한다.

(1) T = C ⊙ S의 유일성(정리 1). 멱법칙뿐 아니라 모든 해석함수에 대해 성립한다.

(2) Hadamard 곱이 유일한 쌍선형 국소 합성이며(보조정리 1), 쌍선형성 자체가 변분 안정성의 귀결이다.

(3) 변분으로부터 이산 게이지-기하 운동 방정식이 도출된다(정리 2). 연산자 T에서 작용 S로, 그리고 모든 역학적 예측으로 이어지는 다리다.

(4) 스펙트럼 수렴이 섭동을 견딘다(정리 3). 연속 극한이 정합적이다.

(5) 이론이 모든 스케일에서 같은 형태를 갖는다(정리 4). 결합 RG 흐름의 출발점이다.

(6) 경로 적분 측도가 dw/w로 유일하다(정리 6). 분배 함수의 정의에 자유 선택이 없다.

T = C ⊙ S의 유일성(정리 1)이 Hadamard 결합 작용의 유일성을 함축한다. 이 단일 작용으로부터 동반 논문들이 도출하는 것은: 게이지-기하 상관 r > 0 [B], 곱셈적 잡음 차단 [C], 초가법적 에너지 E ~ N^γ [D], 힘 정렬 [F], 팽창·균일성·구조 형성 [G], 유질량 입자와 우주론적 역학의 이중섹터 스펙트럼 분해 [H]. 이 모든 것이 세 공리와 변분 안정성만으로 강제된다. 공리에서 우주론과 물질까지의 연쇄가 하나의 병목 — Hadamard 곱 — 을 지나가며, 대안은 없다.

**범위.** 정리 2는 선형화 Regge 체제에서 도출되었다. 3+1차원 중력의 완전한 텐서 구조, SU(2) 너머의 비가환 게이지 군, 위상 변화 역학은 열린 문제로 남는다.
