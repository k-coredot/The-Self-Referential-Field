# 자기참조적 그래프의 Hadamard 구조:
# 공리적 도출, 유일성, 변분 구조

권혁준 — 2026년 4월

---

**초록.** 그래프 위의 자기참조적 역학에 대한 공리적 틀을 수립하고, 에지 진화 연산자가 T = C ⊙ S로 유일하게 결정됨을 증명한다(정리 1). 여기서 C는 상태 내적이고 S는 자기수반 정규화 인접행렬이다. 증명은 세 공리(국소 자기참조, 회전 불변성, 자기수반 스펙트럼 정합성)와 한 물리적 조건(변분 안정성: 약장 극한에서 유한하고 비영인 응답)으로부터 진행된다. 변분 안정성은 쌍선형 경우 f(C,S) = CS를 넘는 모든 대안 T = f(C, S)를 배제한다. 여기에는 (α,β) ≠ (1,1)인 멱법칙 형태 C^α S^β와 exp(CS) 또는 C·log(S) 같은 비다항식 합성이 포함된다. Hadamard 곱이 공리를 만족하는 유일한 국소 합성임을 증명하고(보조정리 1), 이산 게이지-기하 운동 방정식이 δS/δl = 0으로부터 도출됨을 증명하며(정리 2), 스펙트럼 수렴이 에지 가중치 섭동 아래 존속함을 증명한다(정리 3). 확률보행 정규화 D⁻¹A는 자기수반성 요건에 의해 배제된다.

## 0. 용어

**정의 (자기참조적 역학).** 그래프 G = (V, E) 위의 역학계가 *자기참조적*이란, 에지를 따라 노드 상태를 전파하는 진화 연산자 T 자체가 그 노드 상태와 그들이 점유하는 그래프 구조에 의해 결정됨을 뜻한다. 형식적으로: T = T[Φ, G], 여기서 Φ = {φ(v)}_{v∈V}는 장 배위이고 G는 인접 및 계량 데이터를 제공하며, 갱신 규칙 Φ' = TΦ는 T의 다음 평가에 피드백된다. 이 피드백 순환 — 연산자가 상태를 읽고, 상태가 연산자를 결정한다 — 이 정의적 성질이다. 공리 1–3이 이 자기참조적 구조 아래 T의 형태를 구속한다.

## 1. 공리

**공리 1 (국소 자기참조).** T_{ab}는 국소 데이터에만 의존한다: φ(a), φ(b), 그리고 G|_{N(a,b)}.

**공리 2 (회전 불변성).** 모든 R ∈ SO(d)에 대해 T_{ab}[Rφ] = T_{ab}[φ].

**공리 3 (자기수반 스펙트럼 정합성).** 구조적 전파 행렬은 ℓ²(V) 위의 표준 내적에 대해 자기수반(대칭)이며, 차수 이질성 아래 그래프 라플라시안의 스펙트럼 성질을 보존한다. 자기수반성은 실수 고유값(물리적 관측량), 직교 고유벡터(분해 가능성), 가역적 역학(에너지 보존)을 보장한다.


## 2. 성분의 유일성

**명제 1.** 공리 2 아래, 쌍선형 형식 중 C_{ab} = ⟨φ_a, φ_b⟩(스칼라 배를 제외하고 유일). 더욱이 쌍선형성은 변분 안정성과 양립하는 유일한 부류다.

*증명.* Schur 보조정리에 의해, ℝ^d 위의 SO(d) 표준 표현이 기약(d ≥ 2)이므로, 임의의 SO(d)-등변 쌍선형 형식 B: ℝ^d × ℝ^d → ℝ는 어떤 λ ∈ ℝ에 대해 B = λ⟨·,·⟩를 만족한다. 이것이 쌍선형 부류 내에서의 유일성을 확립한다. □

*비고 (비쌍선형 불변량의 배제).* (φ_a, φ_b)의 비쌍선형 SO(d)-불변 함수가 존재한다: 예를 들어 RBF 커널 k(φ_a, φ_b) = exp(−‖φ_a − φ_b‖²/2σ²), 또는 ‖φ_a − φ_b‖²의 임의의 함수. 이들은 공리 2만으로는 배제되지 않고 변분 안정성(정리 1)에 의해 배제된다. 구체적으로, 임의의 해석적 SO(d)-불변 함수 f(⟨φ_a, φ_b⟩)에 대해, 테일러 전개 f(c) = f(0) + f'(0)c + ½f''(0)c² + ⋯는 변분 안정성을 위해 f''(0) = 0을 만족해야 한다(정리 1의 일반 경우 참조). f(0) = 0(분리 배제: 내용 없이 전파를 피하려면 C = 0일 때 T가 소멸해야 함)과 결합하면, f(c) = λc가 강제된다. 따라서 C의 쌍선형성은 공리와 변분 안정성의 *귀결*이지 독립적 가정이 아니다.


**명제 2.** 공리 3 아래, S = D^{-1/2}AD^{-1/2} (유일).

*증명.* 두 단계로 진행한다.

*단계 1 (D^α A D^α로의 환원).* 일반적 차수 의존 정규화를 고려한다: S = D^α A D^β. 자기수반성은 S = S^T를 요구한다. 계산: S_{ij} = d_i^α A_{ij} d_j^β, S^T_{ij} = d_j^α A_{ij} d_i^β. 모든 그래프에서 S = S^T이려면: d_i^α d_j^β = d_j^α d_i^β. 이것은 (d_i/d_j)^{α−β} = 1을 준다. 서로 다른 차수의 노드를 연결하는 에지가 하나라도 있는 그래프에서(d_i ≠ d_j) α = β가 요구된다. 따라서 S = D^α A D^α.

*단계 2 (α = −1/2가 유일).* 정규화 라플라시안은 L_norm = I − D^α A D^α로 정의된다. 이것이 모든 그래프에서 [0, 2] 고유값을 갖는 양반정치가 되려면 — 연속 Laplace-Beltrami 연산자의 스펙트럼 성질과 일치하도록 — α = −1/2가 요구된다 [Chung 1997, 정리 1.7]. α = −1/2에서 행렬 D^{-1/2}AD^{-1/2}는 [−1, 1] 고유값을 갖는 자기수반이고, L_norm = I − D^{-1/2}AD^{-1/2}는 [0, 2] 고유값을 갖는다. 확률보행 행렬 D^{-1}A는 같은 고유값을 공유하지만 자기수반이 아니다: 좌 고유벡터는 D^{1/2}v_k이고 우 고유벡터는 D^{-1/2}v_k여서 일관된 변분 분해를 방해한다. □


## 3. 합성의 유일성

**보조정리 1.** 공리 1 아래, (i) 점별 의존적이고 (ii) 에지당 스칼라 출력을 생성하는 임의의 합성 •: M_E × M_E → M_E는, 합성이 쌍선형이면 • = λ(⊙)를 만족한다.

*증명.* 점별 의존(공리 1로부터)은 (A • B)_{ab} = f(A_{ab}, B_{ab})를 준다. f가 두 스칼라 인수에 대해 쌍선형이면: f(x, y) = λxy. 따라서 • = λ(⊙).

배제된 대안: 행렬 곱 (CS)_{ab} = Σ_c C_{ac}S_{cb}는 국소성을 위반한다(모든 노드 c에 의존). 텐서 곱 C ⊗ S ∈ ℝ^{|E|×|E|}는 에지당 스칼라가 아닌 행렬을 생성한다. 직접 합 T = C + S는 곱셈적 결합을 생성하지 않는다(S와 무관하게 δT/δC = 1). □

*비고.* 보조정리 1은 합성의 쌍선형성을 가정한다. 비쌍선형 점별 합성 f(C_{ab}, S_{ab})의 배제를 포함한 완전한 유일성 결과는 아래 정리 1을 참조.


## 4. 주정리

**정리 1.** 공리 1–3과 자기참조적 정의(§0) 아래, T_{ab} = C_{ab} · S_{ab}가 유일한 에지 연산자다.

*증명.* 명제 1–2가 C와 S를 (스칼라 배까지) 유일하게 결정한다. 합성을 결정하는 것이 남는다. 공리 1(국소성)에 의해 T_{ab} = h(C_{ab}, S_{ab}), 어떤 두 스칼라의 함수 h에 대해.

**단계 0: 인수분해.** 두 경계 조건이 h를 구속한다:
(a) 모든 C에 대해 h(C, 0) = 0: S_{ab} = 0이면 a와 b 사이에 에지가 없으므로 결합이 없다. 따라서 S = 0에서 h가 소멸.
(b) 모든 S에 대해 h(0, S) = 0: C_{ab} = 0(직교 노드 상태)이면 연산자가 내용 정보를 전달하지 않는다. 영 내용 유사도를 가진 에지는 영 결합을 기여한다 — 논문 C에서 경험적으로 검증된 잡음 차단 성질.

(a)로부터: h(C, S) = S · q(C, S). (b)로부터: q(0, S) = 0이므로 q(C, S) = C · r(C, S). 따라서 h(C, S) = C · S · r(C, S)이며, r = 상수를 보이면 충분하다.

**경우 1: 멱법칙 f(C) = C^α (α > 0).**

도함수 f'(C) = α C^{α−1}. C = 0(직교 노드 상태 — 비자명 그래프에 존재하는 배위)에서 평가:

    f'(0) = α · 0^{α−1}

(a) α < 1이면: f'(0) = ∞. 오일러-라그랑주 방정식 δS/δθ_e가 힘 항에 f'(C)를 포함한다. C = 0에서 힘이 발산 — 역학에 특이점. 초기값 문제가 비정칙: Picard-Lindelöf가 리프시츠 벡터장을 요구하며 C = 0에서 실패. 따라서 α < 1은 정칙성에 의해 배제.

(b) α > 1이면: f'(0) = 0. C ≈ 0에서의 자기참조 순환을 고려: 작은 회전 δφ가 C를 0에서 δ로 변화시킨다. 연산자 변화 δT = f'(0) · S · δ = 0. 연산자가 변화를 감지하지 못한다. 장은 변했지만 읽기가 알아차리지 못했다. 이것은 자기참조적 정의를 위반한다: T = T[Φ, G]는 T가 Φ의 변화에 응답할 것을 요구한다. C = 0에서 일차적으로 맹목인 연산자는 장을 완전히 읽지 않는다. 따라서 α > 1은 자기참조성에 의해 배제.

(c) α = 1이면: f'(0) = 1. C = 0에서 차단 성질이 성립(T = 0, 잡음 면역)하지만, f'(0) = 1은 연산자가 완전한 일차 감도를 가짐을 의미 — 직교성을 포함한 모든 내용 수준에서 미소 내용 변화를 감지한다. 자기참조 순환이 온전하다.

따라서 α = 1이 유일한 멱법칙 지수다. S 의존성에 적용된 대칭 논증(동일한 정칙성 및 비퇴화 조건)에 의해 β = 1. 따라서 모든 멱법칙 형태에 대해 T = C ⊙ S. □(경우 1)

**경우 2: 일반 해석함수 f(C).**

f(C) = C + εC³ 또는 f(C) = sin(C) 같은 비멱법칙 대안의 배제가 남는다. 자기참조 순환에서 도출된 두 조건이 증명을 완성한다.

**조건 (부호 보존).** 자기참조 순환 Φ → T[Φ] → Φ'는 고정점(안정 여기)으로 수렴해야 한다. C = 0 근처의 선형화된 내용 응답은 δT ∝ f'(0) · δC다. f'(0) < 0이면 양의 내용 섭동이 음의 응답을 생성하고, 다음 단계에서 양의 응답을 생성: 주기-2 진동. 고정점 없음. 안정 여기 없음. 따라서 f'(0) > 0.

경우 1과 결합(f'(0)이 유한, 비영): f'(0) > 0이고 유한.

**조건 (전역 부호 보존).** 같은 논증이 C = 0에서뿐 아니라 모든 내용 수준에 적용된다. 어떤 C₀ ∈ (−1, 1)에서든 f'(C₀) < 0이면, 순환이 내용 C₀ 근처 배위 주위에서 진동한다. 따라서 모든 C ∈ (−1, 1)에서 f'(C) > 0.

이제: f(0) = 0, 모든 C에서 f'(C) > 0, f'(0)이 유한. 문제는 f'(C)가 C에 따라 변할 수 있는가다.

f(C) = C + εC³를 고려: f'(C) = 1 + 3εC². ε > 0이면 모든 C ≠ 0에서 f'(C) > 1 — 연산자가 영에서 먼 내용 변동을 증폭한다. ε < 0이면 f'(C) < 1이고 결국 C₀ = 1/√(3|ε|)에서 f'(C₀) = 0 — 유한 내용에서 맹점. 두 경우 모두 내용 의존적 이득을 도입한다: 읽기가 어떤 영역에서는 더 크고 다른 영역에서는 더 작다. 유일한 내용 독립적 이득은 f'(C) = 상수이며, f(C) = λC를 준다.

이 논증에는 매개변수론적 보강이 있다: 공리 1–3은 연속적으로 조율 가능한 매개변수를 포함하지 않는다. f(C) = C + εC³인 연산자는 공리적 기원이 없는 ε를 도입한다. 유일한 매개변수-자유 내용 응답은 f(C) = C다.

f(C) = λC가 확립되었으므로, λ를 흡수: T = C ⊙ S. □(경우 2)

*비고.* 경우 1(멱법칙)은 추가 조건 없이 수학적 정리다. 경우 2(일반 해석)는 부호 보존(고정점 존재로부터의 역학적 논증)과 이득 균일성을 사용한다. 둘 다 자기참조 순환에서 도출된 것이지 부과된 것이 아니다. 논문 H(Math)의 정리 10은 모든 물리적 결론이 ε에 연속임을 더 보여, 작은 비선형 수정이 허용되더라도 물리학이 변하지 않음을 확인한다.

잔존 항: f(C, S) = a_{00} + a_{10}C + a_{01}S + a_{11}CS.

**따름정리 1 (게이지와 기하의 역학적 불가분성).** 공리 1–3을 만족하는 그래프에서 Φ와 G가 모두 역학적이면, C와 S의 진화를 독립적으로 최적화할 수 없다.

*증명.* d차원 격자 위의 N개 게이지장에 대한 Hadamard 결합 작용을 정의한다:

S = −Nβ Σ_P w_P cos Θ_P + κ Σ_e (w_e − 1)²

여기서 w_e는 에지 가중치(기하적 자유도), w_P = (∏_{e∈∂P} w_e)^{1/2}는 플라켓 넓이 측도, Θ_P는 플라켓 각도다. 이 작용은 C(cos Θ_P를 통해)와 S(w_P를 통해)를 곱셈적으로 결합한다 — T = C ⊙ S의 직접적 귀결. 오일러-라그랑주 방정식 δS/δw_e = 0(정리 2)은 각 에지에서 게이지 에너지에 비례하는 기하적 변형을 산출한다. Φ의 최적화(cos Θ_P를 결정)는 w의 에너지 지형을 변화시키고, 역도 마찬가지다. 어느 쪽도 독립적으로 극값에 도달하지 못한다. □


## 5. 이산 게이지-기하 운동 방정식

**정리 2.** Hadamard 결합 작용 S = −Nβ Σ_P w_P cos Θ_P + κ Σ_e (w_e − 1)²를 에지 가중치 w_e에 대해 변분하면:

    2κ(w_e − 1) = Nβ Σ_{P∋e} (∂w_P/∂w_e) cos Θ_P

탄성 저항(좌변)이 게이지-기하 구동력(우변)과 같다. Hadamard 구조가 필수적이다: 작용 내의 곱셈적 결합 w_P × cos Θ_P가 δS_{gauge}/δw_e ≠ 0을 보장한다. 가법적 작용 S = S_gauge(C) + S_elastic(w)는 δS_{gauge}/δw_e = 0을 산출하여, 게이지-기하 운동 방정식을 배제할 것이다. □

*범위와 한계.* 정리 2는 고정 격자 위상의 약장(선형화 Regge) 체제에서 도출된다. 두 확장이 열려 있다: (i) 3+1D에서 스핀-2 중력의 완전한 텐서 구조는 비선형화 Regge 역학을 요구하며 이 논문의 범위 밖이다; (ii) 위상 변화(에지 생성/삭제)는 매끄러운 변분 δl에 포착되지 않는다.

*비고 (비대칭성이 구동력).* 오일러-라그랑주 방정식(정리 2)은 다음과 같이 읽을 수 있다: 내용이 요구하는 것과 구조가 제공하는 것 사이의 불일치 Delta_e = dS/dw_e가 모든 역학의 구동력이다. 평형은 불일치의 부재다. 역학은 비대칭을 해소하는 것 이외에 아무것도 하지 않는다.

## 6. 섭동 아래 스펙트럼 수렴

**정리 3.** 유계 에지 가중치 섭동 ‖w−1‖ ≤ δ_n → 0인 Belkin-Niyogi 조건 아래, 정규화 라플라시안 고유값이 수렴한다: λ_k(L_n(w)) → λ_k(Δ_g).

*증명 개요.* 비섭동 수렴 λ_k(L_n(1)) → λ_k(Δ_g)는 Belkin-Niyogi [2007]와 García Trillos et al. [2020]에 의해 성립한다. 섭동 경우에 대해 L_n(w) = L_n(1) + E_n으로 쓴다. 여기서 E_n은 w ≠ 1에 의해 유도된 섭동 행렬이다. Weyl 섭동 정리 [Weyl 1912; Kato 1966, 정리 V.4.10]에 의해 |λ_k(L_n(w)) − λ_k(L_n(1))| ≤ ‖E_n‖₂. ‖E_n‖₂ = O(δ_n) → 0이므로, 삼각 부등식이 λ_k(L_n(w)) → λ_k(Δ_g)를 준다. □

*의미.* 이 정리는 에지 가중치의 역학적 진화(정리 2의 오일러-라그랑주 방정식에 의해 구동)가 프레임워크를 정칙으로 만드는 스펙트럼 구조를 파괴하지 않음을 보장한다. 정규화 라플라시안의 고유값은 에지 가중치가 단위에서 벗어나더라도 기저 다양체 위의 Laplace-Beltrami 연산자의 고유값을 근사하기를 계속한다.

## 6b. 스케일 불변성과 비대칭 역학

**정리 4 (스케일 불변성).** 네 공리 VS1-VS4는 길이 스케일, 에너지 스케일, 격자 간격에 대한 참조를 포함하지 않는다. 블록-스핀 변환에 의해 얻어진 임의의 조립격자는 같은 공리를 만족하며, 블록 격자의 연산자는 T' = C' ⊙ S'이다.

*증명.* 격자를 크기 b의 블록으로 분할한다. 블록 임베딩 p_alpha = normalize(sum a_i phi_i)을 정의한다. 블록 내용 C' = ⟨p_alpha, p_beta⟩는 SO(d)-불변; 명제 1에 의해 유일. 블록 구조 S' = D'^{-1/2}A'D'^{-1/2}; 명제 2에 의해 유일. 정리 1에 의해 T' = C' ⊙ S'. □

*귀결.* 이론이 모든 스케일에서 동일하다. 조립격화는 유효 결합 beta'(beta, kappa)와 kappa'(beta, kappa)를 정의한다. Hadamard 구조가 이들이 결합되도록 강제한다: beta'는 kappa에 의존(기하 요동이 게이지 결합을 수정)하고 kappa'는 beta에 의존(게이지 요동이 탄성 강성을 수정)한다. 완전한 전개는 동반 논문(논문 K)에 있다.

**정리 5 (비대칭의 단조 감소).** kappa > 0인 T = C ⊙ S 격자 위의 과감쇠 역학 dw_e/dt = -dS/dw_e에 대해:

dS/dt = -sum_e (dS/dw_e)^2 ≤ 0

작용이 단조 감소한다. 등호는 평형(모든 e에서 dS/dw_e = 0)에서만 성립한다.

*증명.* dS/dt = sum (dS/dw_e)(dw_e/dt) = -sum (dS/dw_e)^2 ≤ 0. □

*따름정리.* 총 제곱 비대칭 sum Delta_e^2도 단조 감소한다: d/dt sum Delta^2 = -2 Delta^T H Delta ≤ 0, 여기서 H = d^2S/dwdw는 양정치(kappa > 0). □

## 6c. 경로 적분 측도

**정리 6 (측도 유일성).** 에지 가중치 w_e ∈ (0, ∞)에 대한 경로 적분 측도는 유일하게 dμ(w_e) = dw_e/w_e (곱셈군 ℝ₊의 Haar 측도)이다. 적분 영역은 w_e ∈ [w_min, ∞), w_min > 0이며, 길이가 엄밀히 양이라는 물리적 요건을 반영한다. 분배 함수의 수렴: ∫_{w_min}^∞ (1/w) exp(−κ(w−1)²) dw ≤ (1/w_min) ∫₀^∞ exp(−κ(w−1)²) dw = √(π/κ)/w_min < ∞.

*증명.* 에지 가중치 w_e는 길이를 나타낸다(Regge 해석). 길이는 재스케일 w → λw(단위 변경)에 의해 작용받는 양의 실수다. 이것은 곱셈군 (ℝ₊, ×)을 정의한다.

(ℝ₊, ×)의 Haar 측도는 dw/w — 군 위의 (정규화까지) 유일한 좌-우-불변 측도다 [Haar 1933]. w → λw 아래: dw/w → dw/w (불변), 르베그 측도는 dw → λdw (불변이 아님).

유일성은 Haar 측도의 분류로부터 따른다: 모든 국소 컴팩트 군은 (스칼라까지) 유일한 좌-불변 라돈 측도를 허용한다. (ℝ₊, ×)에 대해 이 측도는 dw/w다. □

*비고 1 (측도와 작용의 분리).* 작용의 탄성 항 κ(w−1)²가 w = 1에서의 평형을 설정한다. 측도까지 w = 1을 특권화하면(르베그가 그러하듯, [0,1]과 [1,2]에 같은 가중치를 부여하여) 평형이 이중으로 부과된다. 측도 dw/w는 평형에 대해 불가지다 — [1/2, 1]과 [1, 2]에 같은 가중치를 부여하여 "길이의 절반"과 "길이의 배"를 대칭적으로 취급한다. 측도는 적분 영역을 제공하고; 작용이 물리학을 제공한다.

*비고 2 (Regge 해석과의 정합성).* 양자 Regge 중력에서, Misner 측도 ∏ dl_e/l_e [Misner 1957]는 에지 길이에 대한 표준 스케일 불변 선택이며, 중력 경로 적분의 등각 불변성과 정합적이다. Hadamard 프레임워크의 w_e가 l_e의 역할을 한다. 측도 dw/w는 Misner 측도다 — 새로운 선택이 아닌 확립된 동정.

*비고 3 (새 공리 없음).* 정리 6은 에지 가중치의 양실수 본성 이외의 공리를 사용하지 않는다. 길이는 양의 실수; 곱셈 아래 양의 실수는 군을 형성; 이 군의 유일한 불변 측도는 dw/w. 이것은 추가 물리적 원리가 아니라 — 변수의 수학적 구조다. 공리(VS1–VS4)가 연산자 T와 작용 S를 결정한다. 측도는 적분 변수의 본성에 의해 결정된다. 합쳐서, 분배 함수 Z = ∫ [dU] ∏_e (dw_e/w_e) exp(−S[U, w])가 완전히 지정된다: [dU]는 게이지 군의 컴팩트 Haar 측도(컴팩트성에 의해 유일)이고, dw_e/w_e는 ℝ₊의 Haar 측도(정리 6에 의해 유일)다. 남은 자유 양은 β와 κ뿐이며, 이들은 자기정합성에 의해 결정된다(논문 K, §10).

## 7. 물리적 동정

C = cosθ = Re(e^{iθ})는 Wilson [1974]이 정의한 U(1) 게이지 링크 변수와 형식적으로 동일하다: 에지 (a,b)의 격자 게이지 변수는 U_{ab} = exp(iθ_{ab})이고, 플라켓 작용은 Re(U_{ab}) = cosθ_{ab}를 사용한다. 이것은 유비가 아닌 정의의 일치 — 같은 역할의 같은 수학적 대상이다.

S는 Laplace-Beltrami로 수렴한다(정리 3). Regge 해석에서 에지 길이는 계량을 부호화한다 [Regge 1961].

Hadamard 결합 T = C ⊙ S(정리 1)는 각 에지에서 게이지 내용과 기하를 결합시킨다. 이산 게이지-기하 운동 방정식(정리 2)은 이 결합이 변분시 게이지 에너지에 비례하는 기하적 변형을 생성함을 보여준다. 역학적 검증(몬테카를로, 2D–4D의 7 배위에 걸쳐 42 측정, U(1)과 SU(2))은 동반 논문 [Kwon 2026b]에 있다.


## 8. 논의

수학적 증명에 의한 다섯(+하나) 결과:

(1) T = C ⊙ S가 공리 아래 유일한 변분 안정 에지 연산자이며(정리 1), 유일성이 멱법칙 형태뿐 아닌 모든 해석적 합성으로 확장된다. C만의 항(a₁₀ = 0)의 배제는 스펙트럼 정합성(공리 3)을 사용하고; S만의 항(a₀₁ = 0)의 배제는 자기참조성(정의 §0)을 사용한다. 둘 다 필요하다: 유일성에는 세 공리, 자기참조적 정의, 변분 안정성 모두가 함께 작동해야 한다.

(2) Hadamard 곱이 유일한 쌍선형 국소 합성이며(보조정리 1), 쌍선형성 자체가 독립적 가정이 아닌 변분 안정성의 귀결이다(명제 1, 비고).

(3) 변분이 이산 게이지-기하 운동 방정식을 산출하며(정리 2), 이것이 연산자 T에서 작용 S로, 따라서 모든 역학적 예측으로의 다리다.

(4) 스펙트럼 수렴이 섭동을 견디며(정리 3), 프레임워크의 연속 극한이 정칙임을 보장한다.

(5) 게이지와 기하가 역학적으로 불가분이다(따름정리 1): 단일 작용을 공유하며 독립적으로 최적화될 수 없다.

(6) 에지 가중치에 대한 경로 적분 측도가 유일하게 dw/w이다(정리 6): 곱셈군 ℝ₊의 Haar 측도이며 Regge 해석의 Misner 측도와 정합적이다. 게이지 군의 컴팩트 Haar 측도와 결합하면, 분배 함수 Z = ∫ [dU] ∏(dw/w) exp(−S)가 이론 정의에 자유 선택 없이 완전히 지정된다.

**이 결과들로부터 따라 나오는 것.** T = C ⊙ S의 유일성(정리 1)은 Hadamard 결합 작용의 유일성을 함축한다(따름정리 1). 이 단일 작용으로부터 동반 논문들이 도출한다: 모든 결합과 차원에 걸친 게이지-기하 상관 r > 0 [B]; 그래프 신경망에서의 곱셈적 잡음 차단 [C]; N개 장이 기하를 공유할 때 초가법적 에너지 E ~ N^γ [D]; 공유 플라켓 구조로부터의 정합적 힘 정렬 [F]; 작용의 k-의존 구조의 필연적 귀결로서의 팽창, 균일성, 구조 형성 [G]; 같은 헤시안에서 유질량 입자(단파장)와 우주론적 역학(장파장) 모두를 산출하는 이중섹터 스펙트럼 분해 [H].

이 모든 귀결은 강제된다: 세 공리와 변분 안정성이 주어지면, 연산자는 T = C ⊙ S이고, 작용이 결정되며, 나열된 현상이 따른다. 공리에서 우주론과 물질로의 연쇄는 단일 병목 — Hadamard 곱 — 을 통과하며, 대안은 없다.

**범위.** 프레임워크는 선형화 Regge 체제(정리 2 범위 주)에서 작동한다. SU(2)를 넘는 비가환 게이지 군, 완전한 3+1D 중력, 위상 변환 역학으로의 확장이 열려 있다.
