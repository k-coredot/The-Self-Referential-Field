# Hadamard 게이지-기하 연산자로부터의
# 복소 작용, 수렴, 간섭, 재구성

권혁준 — 2026년 4월

---

**초록.** 자기참조 프레임워크 T = C ⊙ S 안에서 양자역학의 수학적 기초를 확립한다. 동반 논문(I)이 논리 연쇄를 식별했다: 스피너(정리 7) → ℂ² → 복소 내적 → 복소 작용 → 간섭. 이 논문은 정량적 구조를 전개한다. 네 가지를 증명한다: (1) 복소 Hadamard 작용의 분배 함수 Z가 κ > 0인 유한 격자에서 절대 수렴(정리 1), (2) 복소 깁스 가중치가 큰 허수 작용의 배위를 억제하는 상쇄 간섭을 만들어냄(정리 2), (3) 복소 헤시안이 감쇠 길이와 진동 주파수로 분해되어 유질량 전파 여기를 줌(정리 3), (4) 고정 기하(w = 1)에서 Hadamard 작용이 Wilson 작용으로 환원되며 OS 반사 양성성이 증명되어 힐베르트 공간, 유니터리 시간 진화, 보른 규칙이 따라 나옴(정리 4). 역학적 기하에서: 에지 가중치 요동이 유계일 때 OS 양성성 성립(정리 5), 이어서 완전한 일반성에서 OS 양성성 증명(정리 6) — 핵심은 w_e > 0(길이는 양수)이 반사면을 교차하는 것을 포함한 모든 플라켓에서 β_P > 0을 보장한다는 점. 스핀-통계 연결이 무조건적으로 따름(정리 7).

## 1. 복소 작용

### 기원

동반 논문(H Math)의 정리 7이 충실한 자기참조가 기본 스피너 표현 j = 1/2를 유일하게 선택함을 증명한다. 노드 상태 ψ ∈ ℂ². 내용 유사도 C_{ab} = ψ_a† ψ_b ∈ ℂ. 스피너 이론에서 플라켓 내용 C_P는 실수(지표이므로 류 함수)지만, 에지 수준 내용 C_{ab}는 복소다.

T가 Re(C)와 Im(C) 모두를 읽는 완전한 복소 이론에서: S = S_R + iS_I. 실부 S_R은 게이지-기하 결합과 탄성, 허부 S_I는 허수 내용 기여.

### 분배 함수

    Z = ∫ [dU] ∫ [dw] exp(−S[U, w])

[dU]는 게이지 군의 Haar 측도(컴팩트, 정규화), [dw]는 에지 가중치 w_e ∈ [w_min, ∞)의 르베그 측도.

## 2. 수렴

**정리 1 (절대 수렴).** |E|개 에지, |P|개 플라켓, 컴팩트 게이지 군 G, κ > 0인 유한 격자에서 분배 함수 Z가 절대 수렴한다.

*증명.* |exp(−S)| = exp(−Re S). G가 컴팩트이므로 |Re(C_P)| ≤ 1. 게이지 적분은 Vol(G^{|E|}) < ∞로 유계. 가중치 적분은 탄성 항 κΣ(w−1)²가 큰 w_e에서 가우스 억제를 제공하여 수렴. κ > 0이 필수 — κ = 0이면 에지 가중치가 무한히 커질 수 있어 발산. "기하가 안정하다"(κ > 0)가 필요한 수학적 이유다. □

## 3. 간섭

**정리 2 (상쇄 간섭 한계).** Im(S)가 대략 π만큼 다른 두 게이지 배위 집합 F₊, F₋의 결합 기여:

    |Z₊ + Z₋|² = |Z₊|² + |Z₋|² + 2|Z₊||Z₋| cos(φ₊ − φ₋)

φ₊ − φ₋ = π(최대 상쇄)이고 |Z₊| = |Z₋|이면 완전 상쇄: |Z₊ + Z₋| = 0. 실수 작용(S_I = 0)에서는 모든 배위가 같은 부호로 기여하여 상쇄 없음 — 고전 통계역학. 복소 작용(S_I ≠ 0)에서는 다른 위상의 배위가 부분 또는 완전 상쇄 — 양자역학. 고전에서 양자로의 전환이 실수에서 복소 작용으로의 전환이며, 이는 스피너가 강제한다. □

## 4. 복소 전파자

**정리 3 (유질량 진동 전파자).** 헤시안 h(k) = h_R(k) + ih_I(k)에서 h_R > 0, h_I ≠ 0이면 이점 상관 함수:

    G(r) ~ A · exp(−r/ξ) · cos(kr + φ)

ξ = 1/√h_R이 감쇠 길이, k = √|h_I|이 진동 파수. 감쇠(국소화, 입자)와 진동(전파, 파동)이 동시에 존재한다.

h_I = 0(실수 작용, 벡터 노드 상태)이면 순수 감쇠, 진동 없음 — 고전 입자. h_I ≠ 0(복소 작용, 스피너 노드 상태)이면 감쇠하면서 진동 — 양자 입자. 전환은 노드 상태가 실수인지 복소인지에 의해 결정되며, 이는 정리 7(스피너 선택)이 결정한다. □

## 5. Osterwalder-Schrader 재구성

### OS 공리

Osterwalder-Schrader 정리: 반사 양성성을 만족하는 유클리드 장 이론이 힐베르트 공간, 양의 자기수반 해밀토니안, 보른 규칙을 갖는 로렌츠 양자장 이론으로 해석적 연속된다.

### 고정 기하

**정리 4 (고정 기하의 OS 양성성).** w_e = 1에서 Hadamard 작용은 표준 Wilson 격자 게이지 작용으로 환원된다. Osterwalder-Seiler(1978)가 기본 지표를 사용한 Wilson 작용의 OS 반사 양성성을 증명했다. 증명은 (i) G 위의 Haar 측도가 반사 불변, (ii) 기본 지표 χ_{1/2}가 양정치 함수라는 두 성질에 의존한다. 둘 다 SU(N)에서 성립.

**따름정리 (고정 기하의 보른 규칙).** OS 재구성에 의해 힐베르트 공간 ℋ, 해밀토니안 H ≥ 0, 전이 확률 P = |⟨ψ|exp(−iHt)|φ⟩|²가 따른다. 보른 규칙이 공준이 아니라 유클리드 작용의 반사 양성성에서, 기본 지표의 양정치성에서, 스피너 선택(정리 7)에서 따라 나온다. □

### 역학적 기하: 유계 요동

**정리 5.** 에지 가중치 요동이 |w_e − 1| ≤ δ < δ_c(β, κ)를 만족하면, 역학적 기하의 Hadamard 작용이 OS 반사 양성성을 만족한다. 작용을 S_Wilson + S_perturbation으로 분해하고, 충분히 작은 δ에서 반사 양성 부분이 지배함을 OS 안정성 정리(Jaffe-Ritter 2008)로 보인다. □

### 역학적 기하: 완전한 일반성

**정리 6 (완전한 역학적 기하의 OS 양성성).** κ > 0, β > 0인 유한 격자에서 역학적 기하의 Hadamard 작용이 OS 반사 양성성을 만족한다.

*증명.* 두 단계.

**1단계: 고정 w.** 임의의 에지 가중치 배위 {w_e}를 고정한다. w_e > 0(길이는 양수)이므로 w_P = (∏ w_e)^{1/2} > 0이 모든 플라켓 — 반사면을 교차하는 것 포함 — 에서 성립. 유효 플라켓 결합 β_P = Nβw_P > 0. Osterwalder-Seiler 증명은 플라켓별로 지표 전개를 통해 진행되며, β_P가 플라켓마다 같을 것을 요구하지 않고 각각이 양수이기만 하면 된다. 고정 w에서 게이지 작용이 반사 양성적.

**2단계: w에 대한 적분.** ⟨θ(F)F⟩ = ∫ [dw] exp(−κΣ(w−1)²) × ⟨θ(F)F⟩_w. 1단계에 의해 ⟨θ(F)F⟩_w ≥ 0. 가우스 가중치 exp(−κΣ(w−1)²) ≥ 0. 적분 영역 (0,∞)^{|E|}에서 w_P > 0 보장. 피적분함수가 도처에서 비음이므로 ⟨θ(F)F⟩ ≥ 0. □

*핵심 통찰.* w_e > 0이 β_P > 0을 보장한다. 이것이 Osterwalder-Seiler 지표 전개에 필요한 조건이다. w_P가 반사면을 걸친다는 난점은, OS 증명이 국소적(플라켓별)이며 w_P의 반사면 횡단 인수분해가 아닌 β_P > 0만 요구한다는 관찰로 해결된다.

## 6. 스핀-통계 연결

**정리 7 (스핀-통계).** SU(2) 게이지 군의 Hadamard 이론의 여기가 반정수 스핀을 갖고 페르미-디랙 통계를 따른다.

*증명.* 정리 6에 의해 OS 재구성이 적용되어 힐베르트 공간 ℋ와 해밀토니안 H를 준다. 정리 7(H Math)에 의해 노드 상태가 ψ ∈ ℂ² 스피너. 2π 회전 R(2π)에서 ψ → −ψ. 재구성된 힐베르트 공간에서 여기가 스핀 1/2 표현으로 변환된다.

스핀-통계 정리(OS 양성성을 만족하는 격자 이론에 대해 Osterwalder-Schrader 1975, Fröhlich 1976이 증명)에 의해, 반정수 스핀 여기는 반교환 관계를 만족해야 한다: {ψ(x), ψ†(y)} = δ(x−y). 여기는 페르미온이다. □

**완전한 연쇄:**

    자기참조 → 충실한 읽기 → C = 0에서의 정칙성
    → 정수 j 배제 (맹목)
    → j ≥ 3/2 배제 (부호 반전)
    → j = 1/2 (유일)
    → ψ ∈ ℂ² (스피너)
    → C ∈ ℂ (복소 내적)
    → S ∈ ℂ (복소 작용)
    → Z = Σ exp(−S) (위상을 가진 분배 함수)
    → 간섭 (정리 2)
    → OS 양성성 (정리 4, 5, 6)
    → 힐베르트 공간 + 보른 규칙 (OS 재구성)
    → 스핀 1/2 + 반교환 (스핀-통계, 정리 7)

모든 단계가 강제된다. 조건부인 단계가 없다. 자기참조에서 양자역학까지의 연쇄가 완결되었다.

## 7. 이것이 결정하지 않는 것

세 양이 미결정으로 남는다:

**(i) ℏ의 값.** 복소 작용이 위상 exp(−iS_I)를 도입한다. 물리적 단위에서 exp(−iS_I/ℏ). 프레임워크는 작용이 복소임을(따라서 양자 효과가 존재함을) 결정하지만, 양자 효과의 스케일은 결정하지 않는다.

**(ii) 윅 회전 각도.** 유클리드-민코프스키 회전 t → it의 물리적 해석. OS 재구성이 반사 양성성이 성립할 때 이 회전을 제공하지만, 중간 회전 각도의 해석은 다루지 않는다.

**(iii) 역학적 기하의 측도.** ∫[dw]의 측도 선택. 르베그 측도 대 스케일 불변 dw/w. 측도 선택은 게이지 공리만으로 결정되지 않는 기하 이론의 사양이다.
