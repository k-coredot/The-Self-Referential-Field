# Hadamard 게이지-기하 연산자로부터의
# 복소 작용, 수렴, 간섭, 재구성

권혁준 — 2026년 4월

---

**초록.** 자기참조적 프레임워크 T = C ⊙ S 내에서 양자역학의 수학적 기초를 확립한다. 동반 논문(논문 I)이 논리적 연쇄를 식별했다: 스피너(정리 7) → ℂ² → 복소 내적 → 복소 작용 → 간섭. 이 논문은 정량적 구조를 전개한다. 네 결과를 증명한다: (1) κ > 0인 임의의 유한 격자에서 복소 Hadamard 작용의 분배 함수 Z가 절대 수렴(정리 1); (2) 복소 깁스 가중치가 큰 허수 작용을 가진 배위를 억제하는 파괴적 간섭을 생성하며, 억제 인자에 대한 계산 가능한 상한(정리 2); (3) 복소 헤시안이 감쇠 길이와 진동 주파수로 분해되어 유질량 전파 여기를 줌(정리 3); (4) 고정 기하(w = 1)에서 Hadamard 작용이 Wilson 작용으로 환원되며, 이에 대해 Osterwalder-Schrader 반사 양성성이 증명되어 힐베르트 공간, 유니터리 시간 진화, 보른 규칙을 산출(정리 4). 역학적 기하에 대해 부분적 결과를 증명: 에지 가중치 요동이 유계일 때 OS 양성성 성립(정리 5). 이어서 완전한 일반성에서 OS 양성성을 증명(정리 6): 핵심은 에지 가중치 w_e > 0(길이는 양수)이므로 반사면을 교차하는 플라켓을 포함한 모든 플라켓에서 양의 플라켓 결합 β_P > 0이 보장됨. 스핀-통계 연결이 무조건적으로 따른다(정리 7): 여기는 반정수 스핀과 반교환 장을 가진 페르미온.

## 1. 복소 작용

### 1.1 기원

동반 논문(H Math)의 정리 7은 충실한 자기참조가 기본 스피너 표현 j = 1/2를 유일하게 선택함을 증명한다. 노드 상태는 ψ ∈ ℂ². 내용 유사도는:

    C_{ab} = ⟨ψ_a, ψ_b⟩ = ψ_a† ψ_b ∈ ℂ

복소 내용을 가진 Hadamard 작용:

    S = −Nβ Σ_P w_P C_P + κ Σ_e (w_e − 1)²

게이지 군 SU(2)에서 C_P = Re(Tr U_P)/2, U(1)에서 C_P = cos Θ_P. 스피너 이론에서 플라켓 내용 C_P는 실수(지표이므로 류 함수)이지만, 에지 수준 내용 C_{ab}는 복소수.

T가 Re(C)와 Im(C) 모두를 읽는 완전한 복소 이론에서:

    S = S_R + iS_I

S_R = −Nβ Σ w_P Re(C_P) + κ Σ(w−1)², S_I = −Nβ Σ w_P Im(C_P).

### 1.2 분배 함수

**정의.** 분배 함수:

    Z = ∫ [dU] ∫ [dw] exp(−S[U, w])

[dU]는 게이지 군의 Haar 측도(컴팩트, 정규화)이고 [dw]는 w_e ∈ [w_min, ∞) (w_min > 0) 위의 르베그 측도.

---

## 2. 수렴

**정리 1 (절대 수렴).** |E|개 에지, |P|개 플라켓, 컴팩트 게이지 군 G, 탄성 강성 κ > 0인 임의의 유한 격자에서 분배 함수 Z는 절대 수렴한다.

*증명.* |exp(−S)|를 상한으로 잡는다:

    |exp(−S)| = exp(−Re S) = exp(Nβ Σ w_P Re(C_P) − κ Σ(w−1)²)

G가 컴팩트이고 C_P가 지표이므로: |Re(C_P)| ≤ 1.

게이지 적분: ∫ |exp(−S)| [dU] ≤ exp(Nβ Σ w_P) · Vol(G^{|E|}). G가 컴팩트이므로 Vol(G^{|E|}) < ∞.

가중치 적분: 탄성 항 κ Σ(w−1)²가 큰 w_e에서 가우스 억제를 제공. 이차 항이 W → ∞에서 지배하여 지수적 억제를 줌. w_e ∈ [w_min, ∞) 위의 적분이 각 에지에 대해 독립적으로 수렴.

따라서: |Z| ≤ ∫ [dw] exp(Nβ Σ w_P − κ Σ(w−1)²) · Vol(G^{|E|}) < ∞ □

*비고.* 수렴에는 κ > 0이 필요. κ = 0이면 탄성 저항이 없고, 에지 가중치가 무한히 성장 가능하며, 분배 함수가 발산. 이것이 공리 "기하가 안정하다"(κ > 0)가 필요한 수학적 이유.

## 3. 간섭

**정리 2 (파괴적 간섭 상한).** Im(S)가 약 π만큼 다른 두 게이지 배위 집합 F_+와 F_−의 Z에 대한 합산 기여:

    |Z_{+} + Z_{−}| ≤ |Z_{+}| + |Z_{−}| − 2 min(|Z_{+}|, |Z_{−}|) cos(ΔS_I)

ΔS_I = |Im(S_+) − Im(S_−)|.

*증명.* Z_± = |Z_±| exp(iφ_±)로 쓴다. φ_+ − φ_− = π(최대 파괴적)일 때: |Z_+ + Z_−|² = (|Z_+| − |Z_−|)². |Z_+| = |Z_−|이면 간섭이 완전.

일반적으로, 비결맞음 합 대비 억제 인자: η = 1 + 2r cos Δφ / (1 + r²). r ≈ 1, Δφ ≈ π에서: η ≈ 0. 완전 상쇄. □

*물리적 내용.* 실수 이론(S_I = 0)에서 모든 배위가 같은 부호로 기여. 상쇄 없음. 간섭 없음. 고전 통계역학. 복소 이론(S_I ≠ 0)에서 다른 위상의 배위가 부분적 또는 완전히 상쇄. 양자역학. 고전에서 양자로의 전이는 실수에서 복소 작용으로의 전이이며, 스피너에 의해 강제된다.

## 4. 복소 전파자

**정리 3 (유질량 진동 전파자).** 헤시안 h(k) = h_R(k) + ih_I(k)가 h_R > 0, h_I ≠ 0일 때, 이점 상관 함수:

    G(r) = ⟨δw(0) δw(r)⟩ ~ A · exp(−r/ξ) · cos(kr + φ)

ξ = 1/√h_R이 감쇠 길이, k = √|h_I|이 진동 파수, A와 φ는 상수.

*증명.* 푸리에 공간에서 G̃(k) = 1/h(k). 1/(h_R + ih_I)의 역 푸리에 변환이 감쇠와 진동을 동시에 준다. 격자 분산 h(k) = 2κ − Nβc|Γ(k)|²에 대해, h_R의 최솟값에서 안장점 근사:

    ξ = [h_R(k₀)]^{−1/2},  진동 주기 = 2π / k₀

감쇠와 진동이 동시에 존재. 여기가 국소화(입자)되고 진동(파동)한다. □

*비고.* h_I = 0(실수 작용, 벡터 노드 상태): G(r) ~ exp(−r/ξ). 순수 감쇠. 진동 없음. 고전 입자. h_I ≠ 0(복소 작용, 스피너 노드 상태): G(r)이 감쇠하면서 진동. 양자 입자. 전이는 노드 상태가 실수인지 복소수인지에 의해 제어되며, 정리 7(스피너 선택)이 이를 결정한다.

## 5. Osterwalder-Schrader 재구성

### 5.1 OS 공리

Osterwalder-Schrader 정리(1973, 1975)는 특정 공리 — 특히 **반사 양성성** — 을 만족하는 유클리드 장론이 다음을 가진 로렌츠 양자장론으로 해석적으로 연속될 수 있음을 말한다:

(i) 상태의 힐베르트 공간 ℋ
(ii) 시간 진화를 생성하는 양의 자기수반 해밀토니안 H
(iii) 유클리드 대칭군의 유니터리 표현
(iv) 보른 규칙: 전이 확률 = |⟨ψ|φ⟩|²

### 5.2 반사 양성성

**정의.** θ를 시간 좌표 t → −t를 사상하는 반사로 두자. 격자 작용 S가 반사 양성적이란, 양의 시간 반격자(t > 0) 위의 장의 임의의 함수 F에 대해:

    ⟨(θF)* · F⟩ ≥ 0

### 5.3 고정 기하

**정리 4 (고정 기하에서의 OS 양성성).** 게이지 군 SU(N)의 기본 표현에서 고정 기하(모든 e에 대해 w_e = 1)의 Hadamard 작용은 Osterwalder-Schrader 반사 양성성을 만족한다.

*증명.* w_e = 1에서 Hadamard 작용은 다음으로 환원:

    S = −Nβ Σ_P cos Θ_P + κ · const

이것은 (가법 상수까지) 표준 Wilson 격자 게이지 작용. 기본 지표에 대한 Wilson 작용의 OS 반사 양성성은 Osterwalder와 Seiler(1978)에 의해 증명되었다. 증명은 두 성질에 의존: (i) G 위의 Haar 측도가 반사 불변, (ii) 지표 χ_{1/2}가 G 위의 양정치 함수. 두 성질이 기본 지표의 SU(N)에서 성립. □

**따름정리 (고정 기하에서의 보른 규칙).** OS 재구성 정리에 의해 고정 기하 Hadamard 이론은 힐베르트 공간 ℋ, 해밀토니안 H ≥ 0, 전이 진폭 ⟨ψ|exp(−Ht)|φ⟩ = ⟨ψ(0)φ(t)⟩_E를 정의한다. 전이 확률:

    P(ψ → φ, t) = |⟨ψ|exp(−iHt)|φ⟩|²

이것이 보른 규칙. 공준이 아니라 — 유클리드 작용의 반사 양성성으로부터, 이것은 기본 지표의 양정치성으로부터, 이것은 정리 7(스피너 선택)에 의해 강제된 것으로부터 따른다. □

### 5.4 역학적 기하

역학적 기하(w_e 변수)에서 작용:

    S = −Nβ Σ_P w_P χ(U_P) + κ Σ(w−1)²

플라켓 가중치 w_P = (∏_{e ∈ ∂P} w_e)^{1/2}가 반사면 양쪽의 에지를 포함하여, OS 양성성에 필요한 인수분해를 복잡하게 한다.

**정리 5 (유계 기하 요동에 대한 OS 양성성).** 에지 가중치 요동이 어떤 δ < δ_c(β, κ)에 대해 |w_e − 1| ≤ δ를 만족하면, 역학적 기하 Hadamard 작용은 OS 반사 양성성을 만족한다.

*증명 개요.* w_P = 1 + ε_P로 쓴다. 작용이 S = S_Wilson + S_perturbation이 된다. S_Wilson은 반사 양성적(정리 4). 섭동이 유계: |S_perturbation| ≤ Nβ|P|δ^{d/2} + κ|E|δ². 충분히 작은 δ에서 반사 양성 부분이 지배. OS 양성성의 안정성 정리(Jaffe와 Ritter 2008)에 의해 보존. □

**정리 6 (완전한 역학적 기하에 대한 OS 양성성).** 탄성 항 κ > 0인 역학적 기하 Hadamard 작용은 모든 κ > 0, β > 0에서 임의의 유한 격자에서 OS 반사 양성성을 만족한다.

*증명.* 두 단계로 진행: (i) 각 고정 w-배위에서 OS 양성성을 보이고, (ii) w에 대한 적분이 OS 양성성을 보존함을 보인다.

*단계 1: 고정 w.* 임의의 에지 가중치 배위 {w_e}를 고정. 모든 e에서 w_e > 0(에지 가중치는 길이)이므로, 반사면을 교차하는 플라켓을 포함한 모든 플라켓 P에서 플라켓 가중치 w_P = (∏_{e∈∂P} w_e)^{1/2} > 0. 유효 플라켓 결합 β_P = Nβ w_P > 0.

Osterwalder-Seiler(1978)의 Wilson 작용 반사 양성성 증명은 지표 전개를 통해 플라켓별로 진행:

    exp(β_P Re χ(g · θ(g)⁻¹)) = Σ_R d_R a_R(β_P) χ_R(g) χ_R(θ(g))*

β_P > 0에서 계수 a_R(β_P) > 0(U(1)에서 수정 베셀 함수, 컴팩트군의 유사체). 각 교차 플라켓이 독립적으로 반사 양성 인자를 기여. 증명은 비균일 β_P에 대해 수정 없이 적용 — β_P가 모든 플라켓에서 같을 것을 요구하지 않고 각각이 양수이기만 하면 된다. 따라서 고정 w에서 게이지 작용은 반사 양성적.

*단계 2: w에 대한 적분.* 완전한 기대값:

    ⟨θ(F)F⟩ = ∫_{(0,∞)^|E|} [dw] exp(−κΣ(w−1)²) × ⟨θ(F)F⟩_w

단계 1에 의해 각 w-배위에서 ⟨θ(F)F⟩_w ≥ 0. 가우스 가중치 exp(−κΣ(w−1)²) ≥ 0. 적분 영역 (0,∞)^|E|}가 전체에서 w_P > 0을 보장. 따라서 피적분함수가 모든 곳에서 비음수이고, ⟨θ(F)F⟩ ≥ 0. □

*비고.* 핵심 통찰은 w_e > 0(에지 가중치는 길이)이 반사면을 교차하는 플라켓을 포함한 모든 플라켓에서 β_P > 0을 보장한다는 것. 이것이 Osterwalder-Seiler 지표 전개에 요구되는 조건. 이전에 식별된 난점 — w_P가 반사면에 걸침 — 은 OS 증명이 국소적(플라켓별)이고 β_P > 0만 요구하며 반사면에 걸친 w_P의 인수분해 가능성은 요구하지 않음을 관찰하여 해소.

## 6. 스핀-통계 연결

**정리 7 (스핀-통계).** SU(2) 게이지 군을 가진 Hadamard 이론의 여기는 반정수 스핀을 운반하고 페르미-디랙 통계를 따른다.

*증명.* 정리 6에 의해 OS 재구성이 적용되어 힐베르트 공간 ℋ와 해밀토니안 H를 준다.

정리 7(H Math)에 의해 노드 상태는 스피너 ψ ∈ ℂ². 공간 좌표의 2π 회전 R(2π) 아래 ψ → −ψ. 표현이 이중값. 재구성된 힐베르트 공간에서 회전군이 보편 피복 SU(2)를 통해 작용하고, 여기가 스핀-1/2 표현으로 변환.

스핀-통계 정리(OS 양성성을 만족하는 격자 이론에 대해 Osterwalder와 Schrader 1975에 의해 증명, Fröhlich 1976에 의해 확장)는 반사 양성 이론의 반정수 스핀 여기가 반교환 관계를 만족해야 함을 말한다. 따라서 여기의 장 연산자:

    {ψ(x), ψ†(y)} = δ(x − y)

여기는 페르미온이다. □

*비고.* 이것이 연쇄를 완성한다:

    자기참조 → 충실한 읽기 → C = 0에서 정칙성
    → 정수 j 배제 (맹점)
    → j ≥ 3/2 배제 (부호 역전)
    → j = 1/2 (유일)
    → ψ ∈ ℂ² (스피너)
    → C ∈ ℂ (복소 내적)
    → S ∈ ℂ (복소 작용)
    → Z = Σ exp(−S) (위상을 가진 분배 함수)
    → 간섭 (정리 2)
    → OS 양성성 (정리 4, 5, 6)
    → 힐베르트 공간 + 보른 규칙 (OS 재구성)
    → 스핀-1/2 + 반교환 (스핀-통계, 정리 7)

모든 단계가 강제된다. 조건부 단계가 없다. 자기참조에서 양자역학으로의 연쇄가 완전하다.

## 7. 결정하지 않는 것

세 양이 미결정으로 남는다:

**(i) ℏ의 값.** 복소 작용이 위상 exp(−iS_I)를 도입. 물리 단위에서 이것은 exp(−iS_I/ℏ). 프레임워크는 작용이 복소수임(따라서 양자 효과가 존재)을 결정하지만, 양자 효과의 스케일은 결정하지 않는다. ℏ이 자기참조적 고정점에 의해 고정되는지 자유 매개변수인지는 열려 있다.

**(ii) 위크 회전각.** 유클리드 이론은 S_E ∈ ℝ(실수 게이지장) 또는 S_E ∈ ℂ(스피너장). 민코프스키 이론은 t → it 회전으로 얻어진다. 반사 양성성이 성립할 때 OS 재구성이 이 회전을 제공한다.

**(iii) 역학적 기하의 측도.** 에지 가중치에 대한 적분 ∫[dw]에 측도가 필요. 르베그 측도를 사용했다. 대안 측도(예: 스케일 불변 dw/w)는 게이지 구조에 영향 주지 않고 기하적 배위의 가중치를 변화시킬 것이다.
