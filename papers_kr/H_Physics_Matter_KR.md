# 자기참조적 장으로부터 창발하는 물질

권혁준 — 2026년 4월

---

**초록.** 스스로를 읽는 장 — 유일한 Hadamard 연산자 T = C ⊙ S로 형식화 — 이 필연적으로 물질을 만들어낸다. 자기읽기 순환이 잘 정의되려면 필요한 두 구조적 조건(탄성 복원력 κ > 0, 게이지 내용 β > 0)이 함께: (i) 스펙트럼 질량갭 m ∈ (0, ∞), (ii) 위상적으로 보호되는 여기, (iii) 교란의 지수적 국소화, (iv) 구조 형성의 특성 스케일을 보장한다. 이 넷이 입자의 재료다. 2D 역학적 격자의 몬테카를로 시뮬레이션으로 각 예측을 확인한다.

---

## 1. 하나의 공리

장이 존재한다. 스스로를 읽는다. 읽기가 장을 변형한다. 변형된 장이 다시 읽힌다.

이 공리가 — 프레임워크의 전체 내용 — 유일한 에지 연산자 T = C ⊙ S를 강제한다. C = ⟨φ_a, φ_b⟩(상태 비교, 게이지 내용)이고 S = (D^{−1/2}AD^{−1/2})_{ab}(구조 전파, 기하). T는 Hadamard 곱이며, 각 에지에서 하나의 수 — 게이지 내용과 기하 구조의 곱. 이 곱은 유일하게 인수분해할 수 없다. 장은 결합된 두 섹터가 아니라 하나의 개체다.

## 2. 두 가지 조건

자기읽기 순환이 안정하려면 변형에 복원력이 있어야 한다. 없으면 읽기 → 변형 → 더 큰 변형 → 발산. κ라 부른다. 읽기가 정보를 담으려면 장에 다양한 내용이 있어야 한다. 없으면 모든 위치가 동일해 읽을 것이 없다. β라 부른다.

이것은 조절 가능한 매개변수가 아니다. 공리가 잘 정의된 비자명 역학을 만들기 위한 최소 조건이다. κ = 0이면 분배 함수 발산, β = 0이면 모든 상관 소멸.

    S = −Nβ Σ_P w_P cos Θ_P + κ Σ_e (w_e − 1)²

## 3. 네 가지 필연

### 3.1 질량

κ > 0이 각 에지 가중치에 복원력을 준다. 바닥 상태로부터의 이탈 — 여기 — 은 κ에 비례하는 에너지가 든다. 최소 에너지 m > 0. 여기는 유질량이다. β > 0이 플라켓 항을 통해 이웃 에지를 결합한다. 여기가 전파하고, 상관 길이는 유한: m < ∞.

따라서 m ∈ (0, ∞): 장의 모든 여기가 유한하고 0이 아닌 질량을 갖는다. 힉스장이 도입되지 않는다. 질량은 장 자체의 요동에 대한 탄성 응답에서 나온다.

**확인.** 기하 상관 G(r)이 β ≥ 5.0에서 지수 감쇠. m_∞ ≈ 0.2–0.4.

### 3.2 보호

주기 격자에서 총 플라켓 감김수 Q = Σ W_P = 0이 모든 배위에서 성립한다(Stokes 정리). 단일 링크 갱신이 정확히 두 플라켓의 감김수를 동등하고 반대로 바꿔 총합을 보존한다. 위상적이지 역학적이 아니다 — β, κ, 온도와 무관. 비영 감김수를 가진 여기는 국소 갱신의 어떤 순서로도 진공으로 붕괴할 수 없다. 전하 보존의 유비다.

**확인.** 총 위상 전하 변동 std < 1. 소용돌이 수 변화율 1.3%. 소용돌이 수명이 모든 시뮬레이션 시간을 초과.

### 3.3 국소화

m > 0이면, 크기 R의 국소 섭동이 R + ξ 바깥에서 exp(−mr)로 감쇠하는 응답을 만든다. 교란이 무한히 퍼지지 않는다. 국소화되어 에너지를 천천히 잃는다. 스펙트럼 갭의 직접적 수학적 귀결이다.

**확인.** L = 16 열화 격자에서 3×3 섭동이 60 쓸기 후 에너지 집중 15.9배 유지. 질량 중심 이동 < 0.3 격자점.

### 3.4 구조

플라켓 가중치 w_P = (∏ w_e)^{1/2}가 AM-GM에 의해 균일 가중치에서 최대. 게이지 작용이 더 큰 w_P를 선호. 따라서 장이 k = 0 모드로 응축한다.

N ≥ 2 장이 기하를 공유하면, 곱셈적 결합이 양의 피드백을 만든다: 한 장의 변형이 w_P를 키우고, 모든 장의 결합을 강화한다. 총 결합이 탄성 저항을 넘으면 균일 모드가 성장한다 — 장이 팽창한다.

팽창은 k = 0 성장을 구동하고, 탄성이 짧은 파장을 억제한다. 중간값정리에 의해 성장률이 부호를 바꾸는 교차 파수 k*가 존재한다. k* 아래에서 구조로 성장하고, k* 위에서 감쇠된다.

k*가 이론의 두 섹터를 분리한다: k* 아래는 팽창과 구조(우주론), k* 위는 유질량·국소화 여기(입자). 같은 작용에서 양쪽이 나오며, 별도 이론이 아닌 하나의 스펙트럼의 두 섹터다.

**확인.** k = 0 모드 96.5%. ⟨w⟩: 1.0 → 2.58. 초가법성 R(5) = 4.48. 교차 k ≈ 3 근방.

## 4. 하나의 귀결

질량(§3.1) + 보호(§3.2) + 국소화(§3.3) + 구조(§3.4). 구조화된 기하 안의 유질량·안정·유한 크기 여기 — 이것이 물질의 정의적 성질이다. 네 가지 모두 T = C ⊙ S + κ > 0 + β > 0의 필연적 귀결이다.

장이 탄성 응답과 함께 스스로를 읽는 어떤 그래프에서든 — 격자든, 네트워크든, 이산화된 다양체든 — 이 네 성질이 성립한다. 여기가 입자이고, 질량은 κ와 β가 정하고, 안정성은 위상이 정하며, 크기는 ξ = 1/m이고, 공간적 배열은 진스 스케일 k*가 정한다.

이것은 모형이 아니다. 도출이다.

## 5. 남은 것

존재의 도출은 완결되었다. 남은 질문은 정량적이다:

(i) **구체적 질량값.** 질량갭은 임의 차원에서 존재하지만, 수치 m(β, κ, d)는 결합 상수에 의존한다. 알려진 입자 질량에 대응하려면 β/κ를 고정해야 한다.

(ii) **그래스만 통계.** 정리 7이 스피너 표현(반정수 스핀)을 선택한다. 완전한 페르미온 통계(반교환)는 OS 반사 양성성에서 따라 나온다. 동반 논문(I Math, 정리 6)이 완전한 역학적-기하 작용에 대해 OS를 증명하며, 스핀-통계가 무조건적으로 따른다.

(iii) **ℏ의 값.** 스피너가 강제한 복소 작용이 위상 exp(−iS)를 도입한다. ℏ이 자기참조 고정점에 의해 결정되는지 자유 매개변수인지가 가장 깊은 열린 문제다.

이것들은 어떤 우주인지에 대한 질문이지, 우주가 존재하는지에 대한 질문이 아니다.
