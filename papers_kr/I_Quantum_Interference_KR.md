# 자기참조적 스피너 구조로부터의 양자 간섭

권혁준 — 2026년 4월

---

**초록.** 자기참조적 장 역학의 공리(Kwon 2026a)는 비가환 게이지 군의 기본 표현 j = 1/2를 강제한다(Kwon 2026h, 정리 7). 노드 상태는 스피너 ψ ∈ ℂ²가 된다. 이것은 예견되지 않은 귀결을 갖는다: 내용 유사도 C = ⟨ψ_a, ψ_b⟩가 복소값이 되어 연산자 T = C · S, 작용, 깁스 가중치를 복소 영역으로 강제한다. 복소 깁스 가중치 exp(−S) = exp(−Re S) · exp(−i Im S)는 간섭을 생성한다: 정렬된 위상을 가진 배위는 강화되고, 반대 위상은 상쇄된다. 복소 헤시안 h(k) = h_R + ih_I는 동시에 감쇠(국소화, 입자형)하고 진동(전파, 파동형)하는 여기를 준다. 양자 간섭과 파동-입자 통일 모두가 공준이 아닌 도출이다. 논리적 연쇄: 자기참조 → 충실한 읽기 → 스피너(정리 7) → ℂ² → 복소 내적 → 복소 작용 → 간섭 → 파동-입자 통일. 모든 단계가 강제된다.

---

## 1. 연쇄

동반 논문(Kwon 2026h)의 정리 7은 SU(2)의 모든 표현 중 공리가 유일하게 j = 1/2를 선택함을 증명한다. 증명은 두 단계다: 정수 표현은 χ'(0) = 0(맹점, 정칙성에 의해 배제)이고, 더 높은 반정수 표현은 χ'(0) < 0(부호 역전, 고정점 존재에 의해 배제)이다. 잔존하는 표현은 노드 상태 ψ ∈ ℂ²를 갖는다.

이 논문은 ψ ∈ ℂ²의 이론 구조에 대한 귀결을 전개한다.

## 2. 복소 내적

실수 상태 φ ∈ ℝ^d에서 내적은 실수: C = ⟨φ_a, φ_b⟩ ∈ ℝ. 연산자 T = C · S는 실수. 작용은 실수. 깁스 가중치 exp(−S)는 양의 실수.

스피너 ψ ∈ ℂ²에서 내적은 복소수: C = ⟨ψ_a, ψ_b⟩ = ψ_a† ψ_b ∈ ℂ.

충실한 자기참조(정리 1, Kwon 2026a)는 연산자가 C의 완전한 정보를 읽을 것을 요구한다 — 실수부만이 아닌 완전한 복소값을. Re(C)만 읽는 연산자는 위상 정보를 버려, 자기참조 순환에서 정보가 손실된다.

따라서: T = C · S ∈ ℂ. 작용 S = −Nβ Σ w_P C_P ∈ ℂ. 깁스 가중치 exp(−S) ∈ ℂ.

## 3. 간섭

복소 깁스 가중치를 쓴다:

    exp(−S) = exp(−Re S) · exp(−i Im S)

첫 인수는 크기(양의 실수). 둘째 인수는 위상(단위 원 위의 복소수).

배위의 합에서(경로 적분):

    Z = Σ_config exp(−S[config])

각 항은 크기와 위상을 갖는다. 두 배위의 위상이 정렬(Im S₁ ≈ Im S₂)되면 그들의 기여가 강화(건설적 간섭)된다. 위상이 반대(Im S₁ ≈ Im S₂ + π)이면 기여가 상쇄(파괴적 간섭)된다.

이것이 양자 간섭이다. 공준이 아니라 스피너의 귀결이며, 스피너는 충실한 읽기의 귀결이다.

## 4. 파동-입자 통일

복소 헤시안 h(k) = h_R(k) + i h_I(k)는 복소 질량을 가진 여기를 생성한다:

    응답 ~ exp(−√h_R · r) · exp(−i √h_I · r)

**실수부** √h_R: 지수 감쇠. 여기는 유한 영역에 국소화된다. 영역 바깥에서 지수적으로 사라진다. 이것이 입자 — 유한하고, 국소화되고, 에너지가 든다.

**허수부** √h_I: 진동. 여기가 공간에서 전파하며 주기적 패턴을 만든다. 이것이 파동 — 주기적이고, 전파하며, 간섭할 수 있다.

둘 다 하나의 복소수에서. 두 본성이 아니라 — 두 부분을 가진 하나의 본성. 복소 지수가 감쇠와 진동을 한 함수에 통일한다.

## 5. 상보성이 아닌 통일

코펜하겐 해석에서 파동과 입자는 "상보적" 측면 — 동시에 관측될 수 없는 양립 불가능한 기술. 이것은 공준이다.

이 프레임워크에서 파동과 입자는 하나의 복소 함수의 실수부와 허수부다. 이들은 항상 동시에 존재한다. "관측"은 Re 또는 Im을 사영하는 것이다 — 그러나 기저 함수는 항상 둘 다를 포함한다.

상보성은 불완전한 기술의 귀결이지 자연의 근본 성질이 아니다. 완전한 기술 — 복소 h(k) — 은 통일을 포함한다.

## 6. 논리적 연쇄 요약

    자기참조(§0.1)
    → 충실한 읽기(정리 1): T = C · S
    → 정수 표현 배제(χ'(0) = 0)
    → 반정수 표현 선택(χ'(0) ≠ 0)
    → j = 1/2 유일(선형, 부호 보존)
    → ψ ∈ ℂ²
    → C = ⟨ψ_a, ψ_b⟩ ∈ ℂ
    → T ∈ ℂ, S ∈ ℂ
    → exp(−S) = 크기 × 위상
    → 간섭(위상 정렬/상쇄)
    → h(k) ∈ ℂ
    → 감쇠 + 진동 = 입자 + 파동

모든 단계가 수학적으로 강제된다. 선택의 여지가 없다. 양자역학은 자기참조적 스피너 구조의 귀결이다.

## 7. 이전 이론과의 관계

**Feynman 경로 적분(1948).** Feynman은 양자 역학이 경로에 대한 합에 의해 기술될 수 있으며, 각 경로에 위상 exp(iS/ℏ)이 부여됨을 보였다. 이 프레임워크에서 경로 적분은 가정이 아닌 도출이다: 복소 작용이 스피너로부터 나오고, 스피너는 자기참조로부터 나온다.

**OS 양성성(Osterwalder & Schrader 1973).** 유클리드 격자 장론에서 OS 반사 양성성은 힐베르트 공간과 유니터리 시간 진화의 존재를 보장한다. 동반 논문(I Math, 정리 6)은 완전한 역학적 기하 Hadamard 작용에 대해 OS 양성성이 성립함을 증명한다.

## 8. 결론

스피너(정리 7)가 복소 내적을 강제한다. 복소 내적이 복소 작용을 강제한다. 복소 작용이 간섭을 강제한다. 복소 헤시안이 파동-입자 통일을 강제한다. 양자역학은 자기참조의 귀결이다.
