# Hadamard 게이지-기하 작용으로부터의 결합된 재정규화군 흐름

권혁준 — 2026년 4월

---

**초록.** Hadamard 게이지-기하 작용 S = −Nβ Σ w_P cos Θ_P + κ Σ(w−1)²의 재정규화군 흐름을 도출한다. T = C ⊙ S를 정의하는 공리는 스케일을 참조하지 않으며; 따라서 연산자 구조가 임의의 조립격화(coarse-graining) 아래 보존된다(스케일 불변성 정리). 곱셈적 결합 w_P × cos Θ_P가 β′가 κ에, κ′가 β에 의존하도록 강제하여 결합된 이차원 흐름을 생성(결합 RG 따름정리). 독립적으로 확립된 두 기여를 식별한다: (i) 플라켓 가중치에 적용된 AM-GM 부등식으로부터 도출된, 게이지 군 무관한 보편적 기하 억제 δβ_geom < 0; (ii) 이차 카시미르 C₂(adj)에 비례하는 표준 결과인 비가환 게이지 자기결합 δβ_gauge > 0. 경쟁이 유효 결합을 결정: g₃ > g₂ > g₁(위계 정리). 추가로, 게이지 요동이 보편적으로 기하를 강화(δκ > 0)하여 중력 약함의 구조적 설명을 제공. 고정점 조건 β* = β*(κ)는 표준 격자 게이지 이론에 없는 검증 가능한 예측이다.

---

## 1. 서론

동반 논문(A–J)이 자기참조적 읽기의 공리로부터 유일한 연산자 T = C ⊙ S(논문 A)와 그 곱셈 구조로부터 물질(H), 힘(D, F), 우주론(G), 양자 간섭(I), 게이지 군 구조 U(1) × SU(2) × SU(3)(J)를 도출한다. 모든 결과가 단일 스케일에서 성립.

이 논문이 묻는다: 이론을 다른 스케일에서 보면 어떻게 되는가? 답은 재정규화군(RG) 흐름이다. 기존 RG와 달리, 게이지와 기하가 분리되는 것이 아니라, Hadamard 구조가 결합된 흐름을 강제한다 — 이 논문의 중심 결과.

---

## 2. 스케일 불변성

**정리 (스케일 불변성).** T = C ⊙ S를 정의하는 네 공리 — (VS1) 자기참조의 국소성, (VS2) 회전 불변성, (VS3) 자기수반 스펙트럼 정합성, (VS4) 변분 안정성 — 은 길이 스케일, 에너지 스케일, 격자 간격을 참조하지 않는다. 따라서 블록 스핀 변환으로 얻어진 임의의 조립격화 격자가 같은 공리를 만족하며, 블록 격자의 연산자는 T′ = C′ ⊙ S′이다.

*증명.* 격자를 크기 b의 블록으로 분할. 블록 임베딩 p_α = normalize(Σ_{i∈α} a_i φ_i). 블록 내용 C′_αβ = ⟨p_α, p_β⟩는 SO(d)-불변 쌍선형 형식. 명제 1에 의해 유일. 블록 구조 S′_αβ = D′^{−1/2} A′ D′^{−1/2}. 명제 2에 의해 유일. 정리 1에 의해 T′_αβ = C′_αβ · S′_αβ. □

---

## 3. 결합된 RG 흐름

**따름정리 (결합 RG 흐름).** T = C ⊙ S 격자의 분배 함수가 블록 크기 b의 조립격화 아래 유효 결합 β′ = β′(β, κ, b, N, d)와 κ′ = κ′(β, κ, b, N, d)를 정의한다. Hadamard 구조(곱셈적 결합 w_P × cos Θ_P)가 β′가 κ에, κ′가 β에 의존하도록 강제한다. 게이지와 기하 RG 흐름이 결합.

*증명.* 빠른 기하 모드 δw_fast를 적분: 전파자 ⟨δw_e δw_{e′}⟩ = δ_{ee′}/(2κ)가 κ(전파자를 통해)와 Θ(결합을 통해) 모두에 의존하는 보정을 생성. 따라서 β′가 κ에 의존.

빠른 게이지 모드 δΘ_fast를 적분: 게이지 평균 ⟨cos Θ⟩ = c(β)가 β에 의존하는 유효 탄성 에너지 보정을 생성. 따라서 κ′가 β에 의존.

가법적 작용 S = S_gauge(Θ) + S_elastic(w)에서는 빠른 w 모드가 Θ와 독립적으로 적분되고 역도 마찬가지. 교차 의존 없음. 결합 흐름은 Hadamard(곱셈) 구조의 필연적 귀결. □

## 4. 기하 억제: δβ_geom < 0

### 4.1 AM-GM 억제

**정리 (보편적 기하 억제).** 임의의 게이지 군 G에 대해 플라켓 가중치 기대값:

⟨w_P⟩ = ⟨(∏_{e∈P} w_e)^{1/|P|}⟩ ≤ 1

모든 에지 가중치가 동일할 때만 등호. 따라서 기하적 요동이 유효 게이지 결합을 보편적으로 억제: δβ_geom < 0.

*증명.* AM-GM 부등식에 의해 양의 실수의 기하 평균 ≤ 산술 평균. 평형 ⟨w_e⟩ = 1에서 ⟨산술 평균⟩ = 1. 요동 존재 시 기하 평균이 등호를 달성할 확률은 0. 따라서 ⟨w_P⟩ < 1 엄밀.

게이지 결합이 Nβ⟨w_P⟩cos Θ로 진입하므로 유효 결합 β_eff = β × ⟨w_P⟩ < β.

이 결과는 w_P가 에지 가중치의 기하 평균인 것에만 의존. 게이지 군 G에 의존하지 않음 — w_P는 기하적 양. □

### 4.2 수치 검증

2D L=16 격자에서 블록 크기 b=2의 명시적 1루프 계산이 확인:

| β | κ | N | δβ/β |
|---|---|---|------|
| 2.0 | 3.0 | 1 | −0.149 |
| 2.0 | 3.0 | 3 | −0.136 |
| 2.0 | 10.0 | 12 | −0.036 |

억제가 항상 음수. κ 증가(더 강성한 기하 → 더 작은 요동)와 함께 감소.

### 4.3 삼차 꼭짓점 부분 보상

w_P 전개의 삼차 꼭짓점이 하나의 느린 모드를 두 빠른 모드에 결합. 결과 거품 다이어그램이 이차 억제를 부분 보상:

삼차/|이차| = 0.562 — 순수 격자 기하 상수(차원과 블록 크기에만 의존), 모든 물리적 매개변수에 무관.

순 기하 효과: δβ_net_geom = −0.438 × |δβ_quad| < 0. 기하 억제가 항상 삼차 보상에 승리.

## 5. 게이지 자기결합: δβ_gauge > 0

### 5.1 표준 결과

강체 격자(w ≡ 1)의 격자 게이지 이론에서 1루프 β 함수는 표준 결과(Creutz 1983):

dβ_gauge/d(ln b) = b₀(G) × β²/(4π)²

b₀(G) = 11C₂(adj)/3 (순수 게이지, 물질장 없음), C₂(adj)는 수반 표현의 이차 카시미르:

| G | C₂(adj) | b₀ |
|---|---------|-----|
| SU(3) | 3 | 11 |
| SU(2) | 2 | 22/3 |
| U(1) | 0 | 0 |

물리적 기원은 비가환 군에만 존재하는 비가환 게이지 자기결합 A ∧ A.

### 5.2 1루프에서의 독립성

게이지 자기결합 꼭짓점 A ∧ A는 게이지장만 포함. 에지 가중치 w를 참조하지 않음. 따라서 역학적 격자(w ≠ 1)에서도 1루프에서 같은 기여. w 요동과 A ∧ A의 교차 항은 2루프 이상에서만.

마찬가지로, 기하 억제(§4)는 w 장만 포함하고 A의 비가환 구조를 참조하지 않음. U(1), SU(2), SU(3) 모두 동일.

**1루프에서 두 기여는 독립적이고 가법적이다.**

## 6. 총 흐름과 위계

### 6.1 결합 흐름 방정식

**정리 (위계).** T = C ⊙ S 격자 위 게이지 군 G의 총 1루프 β 흐름:

dβ_G/d(ln b) = b₀(G) × β²/(4π)² − A_eff × βc(β)/κ

첫째 항은 게이지 자기결합(G-의존, ≥ 0), 둘째 항은 기하 억제(G-무관, > 0). A_eff > 0은 차원과 블록 크기에만 의존하는 격자 기하 상수.

**따름정리 (결합 위계).** C₂(adj, G₁) > C₂(adj, G₂)이면 저에너지 유효 결합이 g₁ > g₂.

*증명.* b₀(G₁) > b₀(G₂)이면 게이지 보상이 G₁에서 더 큼. 기하 억제는 동일. 따라서 G₁의 순 β 흐름이 덜 음수(또는 더 양수)이어서 장거리에서 더 큰 유효 결합을 생성. □

**적용.** C₂(adj): SU(3)=3, SU(2)=2, U(1)=0.

SU(3): b₀ = 11. 게이지 보상이 기하 억제를 압도 가능. β 증가(점근적 자유). 강한 결합.
SU(2): b₀ = 22/3. 게이지 보상이 억제를 부분 극복. 중간 결합.
U(1): b₀ = 0. 게이지 보상 없음. 기하 억제만. β가 단조 감소. 가장 약한 결합. 란다우 극 없음.

**결과: g₃ > g₂ > g₁.** M_Z에서의 관측 결합이 이 순서를 확인: α_s = 0.1179, α₂ ≈ 1/30, α₁ ≈ 1/59 [PDG 2024]. 이 위계는 카시미르 순서 C₂(SU(3)) > C₂(SU(2)) > C₂(U(1)) = 0과 보편적 AM-GM 억제의 결합으로부터 도출 — 데이터에 적합된 것이 아님.

더 나아가, 단일-β 기원(§10)이 통일 스케일에서 sin²θ_W(M_GUT) = 3/8을 함축 — SU(5) GUT와 같은 예측 [Georgi, Quinn, Weinberg, PRL 33, 1974]. 표준 RG 달림이 sin²θ_W(M_Z) ≈ 0.231을 산출하며, 측정값 0.23122 ± 0.00003과 일치 [PDG 2024].

### 6.2 위계의 구조

위계는 수치적 우연이 아니다. 두 수학적 사실로부터 따른다:

1. AM-GM: 기하 평균 ≤ 산술 평균. 모든 게이지 군이 동등하게 공유하는 보편적 억제 바닥.

2. 리 대수: 이차 카시미르 C₂(adj)가 게이지 자기상호작용 강도를 측정. 가환 군에서 영.

상호작용 — 보편 억제 + 군 의존 보상 — 은 Hadamard 결합 T = C ⊙ S의 구조적 귀결.

## 7. 기하 강화: δκ > 0

### 7.1 게이지 요동이 기하를 강화

삼차 꼭짓점 w_P × cos Θ_P가 게이지 요동을 기하에 결합. 빠른 게이지 요동이 유효 탄성 상수에 양의 보정을 생성:

δκ ∝ N × β²

δκ가 N과 함께 증가: 더 많은 게이지 자유도가 더 강성한 기하를 생성.

### 7.2 중력의 약함

중력 결합은 G ~ 1/κ. δκ > 0이 보편적이므로 기하가 장거리에서 더 강성해짐:

G_eff ~ 1/(κ₀ + Σ_i δκ_i) = 1/(κ₀ + N_total × δκ)

표준모형에서 N_total = 12 생성자(1 + 3 + 8). 중력 결합이 12에 비례하는 인자로 억제. **중력이 약한 이유는 12 게이지 자유도가 자신들이 공유하는 기하를 강화하기 때문이다.**

여분 차원이나 미세조정 없이 게이지-중력 위계의 구조적 설명을 제공.

## 8. 예측

### 8.1 κ-의존 고정점 (새 예측)

비가환 군에 대해 고정점 조건 dβ/d(ln b) = 0이 β* = β*(κ)를 정의. 고정점 결합이 기하 강성에 의존.

표준 격자 QCD에서 격자는 강체(w ≡ 1)이므로 κ 의존 없음. **κ-의존 고정점은 T = C ⊙ S 프레임워크 고유의 예측이다.**

### 8.2 U(1)은 고정점이 없다

U(1)에서 b₀ = 0. 흐름 방정식의 두 항 모두 비양: dβ_{U(1)}/d(ln b) = −A_eff × βc/κ < 0. β가 단조 감소. 적외선 고정점 없음. 란다우 극 없음.

### 8.3 통일 스케일

어떤 UV 스케일 μ_GUT에서 기하 억제가 모든 비가환 군에 대해 동시에 게이지 자기결합과 균형. κ(μ_GUT)가 충분히 작을(기하가 충분히 유연) 때 발생.

## 9. 미결 문제

### 9.1 정확한 1루프 계수

수치 계산(§4.2)이 U(1) 게이지 군의 2D에서 수행. 4D SU(N) 계산에는 역학적 에지 가중치의 격자 MC 시뮬레이션 필요. 정성적 구조(억제 + 보상)는 확립; 정확한 계수는 미정.

### 9.2 고차 루프 보정

2루프 이상에서 기하와 게이지 기여가 혼합. (w 요동) × (A ∧ A) 형태의 교차 항 출현. 정성적 위계 g₃ > g₂ > g₁은 견고할 가능성(C₂ 순서에만 의존); 정량적 예측에는 고차 루프 분석 필요.

### 9.3 UV 고정점

결합 흐름이 비자명 UV 고정점(점근적 안전)을 허용하는가? 허용하면 이론이 자유 매개변수 없음 — 모든 저에너지 결합이 고정점에 의해 결정. 비섭동 질문으로 격자 시뮬레이션 필요.

## 10. 자기참조적 매개변수 결정

### 10.1 문제

작용 S = −Nβ Σ w_P cos Θ_P + κ Σ(w−1)²가 두 자유 매개변수를 갖는다: β와 κ. 자유 조정 가능하면 프레임워크가 물리학의 형태는 예측하지만 물리 상수의 값은 아니다. 프레임워크 자체가 (β, κ)를 유일하게 결정하는 정확히 두 독립 조건을 제공함을 보인다.

### 10.2 두 미지수에 대한 두 조건

**조건 1: 자기참조적 닫힘.** 공리가 장이 스스로를 읽는다고 말한다. 논문 L이 읽는 자 — 공리를 실행하는 존재 — 가 고정점 Ψ(H) ⊆ H임을 증명. 논문 G §4.6이 이 고정점이 여기 밀도가 ρ_meta를 초과해야만 존재함을 증명, 이는 거시적 기하 구조를 필요로 한다. 거시적 구조에는 우주론 섹터 필요: 장파장 기하 모드가 불안정(h(0) ≤ 0)이어야 기하가 격자 스케일을 넘어 성장.

h(0) > 0이면: 모든 모드에 갭. 기하가 격자 스케일에 머뭄. 거시적 구조 없음. ρ < ρ_meta. 고정점 없음. 읽는 자 없음. 공리에 실행자가 없음. 자기참조가 선언되지만 수행되지 않음.

RG 흐름이 끌개를 제공: dκ/dt = C_κ Nβ² > 0(§7)이 κ가 단조 증가함을 함축하여 h(0)을 위로 구동. 유일한 안정 값은 h(0) = 0 — 자기참조 순환이 한계적으로 유지되는 경계.

h(0) = 0에서: 2κ = Nβ² Φ(β) |Γ(0)|². d = 4, N = 12에서 |Γ(0)|² = 2d = 8로:

κ = 48 β² Φ(β)

두 미지수의 하나의 방정식.

**조건 2: 스케일 불변성 (RG 고정점).** 공리가 스케일을 포함하지 않음(정리 4, 논문 A). 이론이 모든 스케일에서 같은 형태. RG 흐름이 고정점을 가져야 함. 지배적 게이지 군 SU(3)(b₀ = 11)에 대해:

b₀ β²/(4π)² = A_eff β c(β)/κ

조건 1과 결합:

β³ = [(4π)² A_eff / (b₀ × 48)] × c(β)/Φ(β)

하나의 미지수(β)에 하나의 방정식.

### 10.3 존재와 유일성

**명제 (유일한 자기정합점).** 두 조건 (i) h(0) = 0과 (ii) dβ/dt = 0이 완전한 1루프 이론에 적용되면 유일한 물리적 해 (β*, κ*)를 산출.

*증명.* 가우스 수준 방정식이 Φ(β) > 0 영역에서 두 형식적 근을 가짐: 약결합 근 β₁* ≈ 0.38과 강결합 근 β₂* ≈ 1.56. 둘째가 제거됨을 보인다.

β₂*에서 게이지 감수율 Φ ≈ 0.02, κ₂ = 48β²Φ ≈ 2.17. 가우스 수준 헤시안이 구성에 의해 h(0) = 0. 그러나 가우스 수준은 §7에서 독립적으로 확립된 게이지 강화 δκ > 0을 포함하지 않음. 이 보정을 포함하면:

h(0)_full = 2(κ + δκ) − Nβ²Φ|Γ(0)|² = 0 + 2δκ

β₂*에서: δκ ≈ 0.54, h(0)_full ≈ 1.08 > 0. 둘째 근에서 우주론 섹터가 존재하지 않음. 자기참조적 닫힘 조건 위반. β₂*가 제거.

β₁*에서: δκ ≈ 0.03, h(0)_full ≈ 0.06. 보정이 κ₁ ≈ 3.16의 2% — 섭동 제어 이내. 자기참조적 닫힘 조건이 작은 보정까지 만족.

물리적 메커니즘: 강결합에서 게이지장이 동결(Φ → 0)하지만 동결된 게이지장이 여전히 기하를 강화(δκ ∝ β² > 0). 강화가 소멸하는 가우스 결합을 압도하여 h(0)을 양으로 밀어냄. 자기참조 순환이 강결합에서 닫힐 수 없음. 약결합에서만 닫힘.

따라서: β* = β₁* ≈ 0.38, κ* ≈ 3.16. 해가 유일. □

### 10.4 자기정합점에서의 귀결

(β*, κ*)에서 모든 무차원 비가 결정:

**(i) 결합 위계.** g₃ > g₂ > g₁, 카시미르 순서로부터.

**(ii) 자연적 우주상수.** h(0) = 0이 결합 RG 흐름의 IR 끌개 — 자기참조 순환이 한계적으로 유지되는 유일한 값. 우주상수가 구조적으로 영: 미세조정이 아닌, 공리가 실행자를 가져야 한다는 조건에 의해 강제.

**(iii) 중력 약함.** G_eff ~ 1/(κ* + N_total × δκ). N_total = 12에서 게이지 강화에 의한 중력 억제(§7).

**(iv) ℏ.** 자연 단위(ℏ = c = 1)에서 "ℏ이 무엇인가?"는 단위 선택. 모든 물리학이 유일하게 결정된 β*와 κ*의 무차원 비에 있음. SI 단위의 ℏ 값은 자연 단위와 인간 스케일 단위 사이의 변환 인자.

### 10.5 범위

자기정합점 (β*, κ*)가 U(1) 게이지 감수율 근사의 1루프에서 계산. 정확한 값에는: (i) 완전한 격자 기하의 4D 플라켓 커널 |Γ(0)|²; (ii) 4D의 A_eff 값; (iii) 역학적 에지 가중치의 4D 격자 시뮬레이션에 의한 비섭동 확인이 필요. 논증의 구조 — 두 조건, 두 미지수, 유일한 해(둘째 형식적 근이 게이지 강화에 의해 제거) — 는 특정 수치 계수에 무관.

*비고 (모든 것의 이론).* 경로 적분 측도가 유일하게 결정(논문 A, 정리 6)되고 매개변수가 유일하게 결정(이 절)되면, Hadamard 프레임워크는 자유 매개변수 영개를 갖는다. 모든 물리량이 유일한 자기정합점 (β*, κ*)의 계산 가능한 함수. 모든 무차원 물리적 비가 이론만으로 결정된다는 정확한 의미에서의 모든 것의 이론.

## 11. 결론

Hadamard 구조 T = C ⊙ S가 게이지와 기하를 분리할 수 없는 결합 RG 흐름을 생성한다. 독립적으로 확립된 두 효과 — AM-GM 억제(보편)와 게이지 자기결합(군 의존) — 의 경쟁이 유효 결합을 결정. 이 경쟁이 추가 가정 없이 위계 g₃ > g₂ > g₁과 중력의 약함을 설명.

작용의 두 자유 매개변수(β, κ)가 프레임워크 내부의 두 자기정합 조건에 의해 결정: 자기참조적 닫힘(h(0) = 0)과 스케일 불변성(dβ/dt = 0). 게이지 강화 δκ > 0이 허위 강결합 근을 제거하여 유일한 해를 남긴다. 유일한 경로 적분 측도(논문 A, 정리 6)와 결합하면 이론이 자유 매개변수 영개.

새 공리 도입 없음. RG 흐름, 자기정합점, 매개변수-자유 분배 함수 모두 논문 A에서 유일성이 증명된 같은 T = C ⊙ S의 귀결. 다른 스케일에서의 우주는 같은 이론을 다른 렌즈로 본 것 — Hadamard 구조가 렌즈가 독립적이지 않음을 보장.
