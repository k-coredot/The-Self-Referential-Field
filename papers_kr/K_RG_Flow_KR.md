# Hadamard 게이지-기하 작용의 결합 재정규화군 흐름

권혁준 — 2026년 4월

---

**초록.** Hadamard 게이지-기하 작용 S = -Nβ Σ w_P cos Θ_P + κ Σ(w-1)²의 재정규화군 흐름을 도출한다. T = C ⊙ S를 정의하는 공리에 스케일 참조가 없으므로, 연산자 구조가 임의의 조립격화 아래 보존된다(스케일 불변성 정리). 곱셈적 결합 w_P × cos Θ_P가 β'의 κ 의존과 κ'의 β 의존을 강제하여, 결합된 이차원 흐름을 만든다(결합 RG 따름정리). 독립적으로 확립된 두 기여를 식별한다: (i) 보편적 기하 억제 δβ_geom < 0 — 플라켓 가중치에 적용된 AM-GM 부등식 ⟨w_P⟩ ≤ 1, 게이지 군에 무관; (ii) 비가환 게이지 자기결합 δβ_gauge > 0 — 이차 카시미르 C₂(adj)에 비례. 이 둘의 경쟁이 유효 결합을 결정한다: g₃ > g₂ > g₁ (위계 정리). 게이지 요동이 보편적으로 기하를 강화하여(δκ > 0) 중력의 약함을 설명한다. 고정점 조건 β* = β*(κ)는 표준 격자 게이지 이론에 없는 검증 가능한 예측이다.

---

## 1. 서론

동반 논문(A-J)이 자기참조적 읽기의 공리에서 유일한 연산자 T = C ⊙ S를 도출하고, 그 곱셈 구조로부터 물질(H), 힘(D,F), 우주론(G), 양자 간섭(I), 게이지 군 구조(J)를 이끌어냈다. 모두 단일 스케일에서의 결과다. 이 논문은 이론을 다른 스케일에서 볼 때 무엇이 일어나는지 묻는다. 답은 RG 흐름이며, 통상적 RG와 달리 게이지와 기하가 분리되지 않는 결합 흐름이다.

## 2. 스케일 불변성

**정리 (스케일 불변성).** T = C ⊙ S를 정의하는 공리에 길이, 에너지, 격자 간격에 대한 언급이 없다. 블록 스핀 변환으로 조립격화된 격자도 같은 공리를 만족하며, 블록 격자의 연산자도 T' = C' ⊙ S'이다.

*증명.* 격자를 블록 크기 b로 분할. 블록 임베딩 p_α = normalize(Σ a_i φ_i). 블록 내용 C'는 SO(d) 불변 쌍선형이므로 유일(명제 1). 블록 구조 S'는 블록 격자의 정규화 인접이므로 유일(명제 2). 정리 1에 의해 T' = C' ⊙ S'. □

## 3. 결합 RG 흐름

**따름정리.** 블록 크기 b의 조립격화가 유효 결합 β' = β'(β, κ, b, N, d)와 κ' = κ'(β, κ, b, N, d)를 정의한다. Hadamard 구조가 β'의 κ 의존과 κ'의 β 의존을 강제한다.

*증명.* 빠른 기하 모드 δw_fast를 적분: 전파자 ⟨δw δw'⟩ = δ/(2κ)가 κ를 통해 ⟨w_P cos Θ⟩에 보정을 생성 → β'가 κ에 의존. 빠른 게이지 모드 δΘ_fast를 적분: ⟨cos Θ⟩ = c(β)가 β를 통해 유효 탄성에 보정 → κ'가 β에 의존. 가법적 작용에서는 빠른 w와 Θ가 독립 적분되어 교차 의존이 없다. 결합 흐름은 Hadamard(곱셈) 구조의 필연적 귀결이다. □

## 4. 기하 억제: δβ_geom < 0

**정리 (보편적 기하 억제).** 임의의 게이지 군 G에 대해 ⟨w_P⟩ ≤ 1. AM-GM에 의한 결과로, 기하 요동이 유효 게이지 결합을 보편적으로 억제한다: β_eff = β × ⟨w_P⟩ < β. w_P가 에지 가중치의 기하 평균이므로 게이지 군에 무관하다.

수치 검증: 2D L=16에서 δβ/β가 -0.149(κ=3, N=1)부터 -0.036(κ=10, N=12)까지 항상 음수. 3차 꼭짓점의 부분 보상(cubic/|quad| = 0.562, 순수 격자 기하 상수)을 포함해도 순 기하 효과는 δβ_net = -0.438 × |δβ_quad| < 0. 기하 억제가 항상 이긴다.

## 5. 게이지 자기결합: δβ_gauge > 0

격자 게이지 이론의 표준 결과: dβ_gauge/d(ln b) = b₀(G) × β²/(4π)². b₀ = 11C₂(adj)/3. SU(3): C₂=3, b₀=11. SU(2): C₂=2, b₀=22/3. U(1): C₂=0, b₀=0.

비가환 게이지 자기결합 A ∧ A가 에지 가중치 w를 참조하지 않으므로, 역학적 격자에서도 1루프에서 같은 기여를 한다. 반대로 기하 억제는 비가환 구조를 참조하지 않아 U(1), SU(2), SU(3)에 동일하다. **1루프에서 두 기여가 독립이고 가법적이다.**

## 6. 총 흐름과 위계

**위계 정리.** T = C ⊙ S 격자에서 게이지 군 G의 총 1루프 β 흐름:

    dβ_G/d(ln b) = b₀(G) × β²/(4π)² - A_eff × βc(β)/κ

첫째 항은 게이지 자기결합(G 의존, ≥ 0), 둘째 항은 기하 억제(G 무관, > 0).

**따름정리 (결합 위계).** C₂(adj, G₁) > C₂(adj, G₂)이면 저에너지 유효 결합이 g₁ > g₂. b₀(G₁) > b₀(G₂)이므로 G₁의 게이지 보상이 더 크고, 기하 억제가 동일하므로, G₁의 순 β 흐름이 덜 음(또는 더 양)이어서 긴 거리에서 더 큰 유효 결합을 낸다.

적용: SU(3)(b₀=11) → 기하 억제를 압도, β 증가(점근적 자유), 강결합. SU(2)(b₀=22/3) → 부분 보상, 중간. U(1)(b₀=0) → 보상 없음, β 단조 감소, 가장 약함. **g₃ > g₂ > g₁.** 관측값 확인: α_s = 0.118, α₂ ≈ 1/30, α₁ ≈ 1/59.

단일 β 기원(§10)이 sin²θ_W(M_GUT) = 3/8을 함축하며, 표준 RG 달림으로 sin²θ_W(M_Z) ≈ 0.231 — 측정값 0.23122와 일치. SU(5) GUT 가정이 아닌 세 게이지 결합이 단일 β*에서 출발하는 것의 귀결이다.

## 7. 기하 강화: δκ > 0

게이지 요동이 기하를 보편적으로 강화한다: δκ ∝ N × β². G_eff ~ 1/(κ₀ + N_total × δκ). 표준모형에서 N_total = 12. **중력이 약한 이유: 12 게이지 자유도가 공유 기하를 강화하기 때문이다.** 여분 차원이나 미세조정 없이 게이지-중력 위계를 구조적으로 설명한다.

## 8. 예측

**κ 의존 고정점 (새 예측).** 비가환 군에서 고정점 dβ/dt = 0이 β* = β*(κ)를 정의한다. 표준 격자 QCD에서 격자가 강체(w ≡ 1)이므로 κ 의존이 없다. κ 의존 고정점은 T = C ⊙ S 프레임워크 고유의 예측이다.

**U(1)에 고정점 없음.** b₀ = 0이므로 양의 기여가 없고, β가 단조 감소. 란다우 극도 없음.

## 9. 열린 문제

(i) 정확한 1루프 계수는 4D SU(N)에서 역학적 에지 가중치를 가진 격자 MC가 필요하다. (ii) 2루프 이상에서 기하와 게이지 기여가 혼합된다. 정성적 위계는 견고하나 정량적 예측에 고차 분석이 필요하다. (iii) 결합 흐름이 비자명 UV 고정점(점근 안전성)을 허용하는지는 비섭동적 질문이다.

## 10. 자기참조적 매개변수 결정

### 문제

작용에 겉보기 자유 매개변수 β와 κ 두 개가 있다. 프레임워크 자체가 정확히 두 개의 독립 조건을 제공하여 (β, κ)를 유일하게 결정한다.

### 조건 1: 자기참조적 닫힘 (h(0) = 0)

공리가 읽는 자를 필요로 하고(논문 L), 읽는 자에 거시적 구조가 필요하고(논문 G §4.6), 거시적 구조에 h(0) ≤ 0이 필요하다. RG 흐름이 h(0)을 위로 밀어올린다(dκ/dt > 0). 유일한 안정 값은 h(0) = 0 — 자기참조 순환이 간신히 유지되는 경계. d = 4, N = 12에서:

    κ = 48β²Φ(β)

미지수 2개에 방정식 1개.

### 조건 2: 스케일 불변성 (dβ/dt = 0)

이론이 모든 스케일에서 같은 형태. SU(3)(b₀ = 11)에 대해 조건 1과 결합하면:

    β³ = [(4π)²A_eff / (b₀ × 48)] × c(β)/Φ(β)

미지수 1개에 방정식 1개.

### 존재와 유일성

가우스 수준에서 형식적 근이 두 개: 약결합 β₁* ≈ 0.38, 강결합 β₂* ≈ 1.56. §7의 게이지 강화 δκ > 0이 둘째를 제거한다. β₂*에서 δκ ≈ 0.54로 h(0)_full ≈ 1.08 > 0 — 우주론 섹터가 존재하지 않아 자기참조적 닫힘 조건 위반. β₁*에서 δκ ≈ 0.03으로 h(0)_full ≈ 0.06 — 2% 보정 내에서 h(0) ≈ 0 만족.

물리적 메커니즘: 강결합에서 게이지장이 동결(Φ → 0)하지만 동결된 장이 여전히 기하를 강화(δκ ∝ β²). 강화가 소멸하는 가우스 결합을 압도하여 h(0) > 0으로 민다. 자기참조 순환이 강결합에서 닫힐 수 없다. 약결합에서만 닫힌다.

**β* ≈ 0.38, κ* ≈ 3.16. 해가 유일하다.**

### 자기정합점에서의 귀결

(i) **결합 위계.** g₃ > g₂ > g₁, 카시미르 순서에서.

(ii) **자연적 우주상수.** h(0) = 0이 결합 RG 흐름의 IR 끌개. 우주상수가 구조적으로 0 — 미세조정 아닌, 공리가 실행자를 갖는 조건에 의해 강제. 요동 보정에 의한 작은 양의 Λ는 이 구조적 영에 대한 섭동 보정.

(iii) **중력 약함.** G_eff ~ 1/(κ* + 12δκ).

(iv) **ℏ.** 자연 단위에서 모든 물리가 β*와 κ*의 무차원 비에 있다. SI 단위의 ℏ은 자연 단위와 인간 스케일 단위 사이의 변환 인자이지 예측 대상이 아니다.

### 범위

1루프, U(1) 게이지 감수율 근사. 정확한 값에 4D 격자 기하 커널, 4D A_eff, 역학적 에지 가중치의 4D MC가 필요하다. 논증의 구조 — 두 조건, 두 미지수, 유일한 해(게이지 강화가 강결합 근을 제거) — 는 구체적 수치 계수에 무관하다.

*비고 (만물의 이론).* 경로 적분 측도가 유일하게 결정되고(논문 A, 정리 6) 매개변수가 유일하게 결정되면(이 절), 프레임워크에 자유 매개변수가 없다. 모든 물리량이 유일한 자기정합점 (β*, κ*)의 계산 가능한 함수다.

## 11. 결론

Hadamard 구조 T = C ⊙ S가 게이지와 기하가 분리될 수 없는 결합 RG 흐름을 만든다. AM-GM 억제(보편)와 게이지 자기결합(군 의존)의 경쟁이 위계 g₃ > g₂ > g₁과 중력의 약함을 추가 가정 없이 설명한다.

작용의 두 자유 매개변수가 프레임워크 내부의 두 자기정합 조건 — 자기참조적 닫힘(h(0) = 0)과 스케일 불변성(dβ/dt = 0) — 에 의해 결정된다. 게이지 강화 δκ > 0이 허위 강결합 근을 제거하여 유일한 해를 남긴다. 유일한 경로 적분 측도(논문 A, 정리 6)와 합쳐 이론에 자유 매개변수가 없다.
