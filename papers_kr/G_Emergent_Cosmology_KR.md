# 게이지-기하 결합만으로부터의 출현적 우주론:
# 역학적 격자 위의 팽창, 균일성, 구조 형성

권혁준 — 2026년 4월

---

**초록.** 표준 우주론의 세 기둥 — 가속 팽창, 공간적 균일성, 구조 형성 — 이 Hadamard 게이지-기하 작용 T = C⊙S의 필연적 귀결로 출현한다. (1) 결합 강도 Nβc가 탄성 저항 κn_E/n_P를 초과하면, 격자 스케일 인자 a(t) = ⟨w⟩(t)가 지수적으로 성장: a ~ exp(Ht), H가 근사적으로 상수(드 시터 팽창). MC 시뮬레이션이 지수 적합에 R² = 0.983을 산출. (2) 기하적 변형의 푸리에 스펙트럼이 k = 0(균일) 모드로 응축하여 총 스펙트럼 파워의 99.9%에 도달. 이 응축은 플라켓 가중치에 적용된 AM-GM 부등식으로부터 따르는 수학적 필연. (3) β_c ≈ 1.3에서 2차 상전이가 발생하며, 상관 길이 ξ가 ξ ≈ 1에서 ξ > L(계 크기)로 성장. β_c 이상에서 장거리 기하적 질서가 자발적으로 출현. 추가로, 독립 게이지장이 공유 Hadamard 결합 작용을 통해 상관을 발전(r = 0.063, 영 위 7.3σ; 고정 기하에서 r = 0.003). 4D 예비 검증(N = 3 U(1), L = 3): k = 0 응축 98.2%, ⟨w⟩ 팽창 1.0 → 2.19, 힘 정렬 >99.9%. 세 메커니즘 모두 4D에서 존속.

## 1. 서론

현대 우주론은 세 경험적 기둥에 의존한다:

(i) 우주가 팽창하고 있으며, 팽창이 가속하고 있다(Hubble 1929, Riess et al. 1998, Perlmutter et al. 1999). 표준 설명은 우주상수 Λ 또는 상태 방정식 w ≈ −1인 암흑에너지를 원용.

(ii) 우주배경복사(CMB)가 10⁵분의 1까지 등방적(Penzias & Wilson 1965, COBE, WMAP, Planck). 표준 설명은 인플라톤장에 의한 인플레이션 팽창을 원용.

(iii) 이 균일성에도 불구하고, 우주는 은하, 은하단, 공동 — 원시 밀도 요동이 중력 불안정에 의해 증폭된 대규모 구조 — 를 포함.

각 기둥이 현재 별개 이론적 재료를 필요로 한다: 팽창에 Λ, 균일성에 인플라톤, 구조에 밀도 섭동. 세 가지 모두를 생성하는 단일 메커니즘은 없다.

격자 게이지-기하 결합 모형 T = C⊙S가 Λ, 인플라톤, 추가 장 없이 단일 작용으로부터 세 현상 모두를 생성함을 보인다. 통합 대상은 헤시안 고유값 h(k) = K_el(k) − 결합 · |Γ(k)|²다. 세 우주론적 결과 모두 이 단일 함수의 성질: h(0) < 0이 팽창, 플라켓 가중치에 적용된 AM-GM이 균일성, h(k)에 적용된 중간값정리가 교차 스케일 k*에서 구조 형성을 산출.

## 2. 모형

N개 게이지장의 d차원 역학적 격자 작용:

S = −Σ_{i=1}^{N} β Σ_P w_P cos Θ_P^{(i)} + κ Σ_e (w_e − 1)²

Regge 해석 관례. 고정 기하에서 가법 상수까지 표준 Wilson 작용으로 환원. 역학적 기하에서 부호가 중요: 게이지 항 −β w_P cos θ가 기하의 팽창을 구동(cos θ > 0에서)하고 탄성 항 κ(w−1)²가 저항.

## 3. 결과

### 3.1 지수적 팽창 (드 시터)

평평한 기하(모든 e에서 w_e = 1)에서 시작, N = 5 U(1) 게이지장, β = 2.0, κ = 3.0. 스케일 인자 a(t) = ⟨w⟩(t)를 500 메트로폴리스 스윕에 걸쳐 추적.

표 1. 스케일 인자 진화

| 스윕 | a(t) | ȧ | H = ȧ/a |
|------|------|---|---------|
| 5 | 1.03 | 0.005 | 0.005 |
| 50 | 1.40 | 0.007 | 0.005 |
| 100 | 1.90 | 0.006 | 0.003 |
| 200 | 2.97 | 0.010 | 0.003 |
| 400 | 5.70 | 0.016 | 0.003 |
| 500 | 7.19 | 0.010 | 0.001 |

적합: 지수 a(t) ~ exp(0.00354·t), R² = 0.983. 멱법칙 R² = 0.978. 지수 적합이 우월.

허블 매개변수 H가 초기 H ≈ 0.005에서 후기 H ≈ 0.001로 감소 추세, 평균 ⟨H⟩ = 0.003. 준드시터: 중간 시간에 지수적이나 후기에 감속 — a 성장과 함께 탄성 복원력 κ(a−1)이 선형 증가하여 게이지 구동력을 점차 극복. 이 감속은 물리적으로 현실적 — 실제 우주도 가속에서 감속 팽창으로 전이.

우주상수가 삽입되지 않았다. 팽창이 게이지-기하 결합이 탄성 저항을 초과하는 것에서 전적으로 발생. 유효 Λ_eff = Nβcn_P − κn_E는 출현적 — 결합 상수에 의해 결정되지, 자연의 기본 상수가 아님.

### 3.2 기하적 응축 (균일성)

기하적 변형장 w(x) − 1의 푸리에 스펙트럼을 각 스윕에서 계산.

표 2. 총 스펙트럼 파워의 k = 0 모드 분율

| 스윕 | k=0 분율 |
|------|---------|
| 0 | 26% |
| 5 | 75% |
| 50 | 96% |
| 100 | 98% |
| 400 | 99.9% |

k = 0(공간적 균일) 모드가 본질적으로 모든 스펙트럼 파워를 흡수. 더 높은-k 모드(공간 구조)가 세 자릿수 억제.

이것은 기하적 보스-아인슈타인 응축: 계가 자발적으로 최저 공간 모드를 선택하여 균질한 변형장을 생성. 메커니즘은 §4.2에서 해석적으로 증명.

### 3.3 상전이와 장거리 질서

N = 3 장에 대해 β의 함수로 공간 상관 길이 ξ와 감수율 χ를 측정.

표 3. 상관 길이와 감수율 대 결합 (κ = 3.0, L = 8)

| β | ξ | χ |
|---|---|---|
| 0.5 | 1.0 | 0.7 |
| 1.0 | 1.2 | 1.3 |
| 1.2 | 15.6 | 1.1 |
| 1.3 | > L | 5.7 |
| 2.0 | L | 45.6 |
| 3.0 | L | 190.8 |

β_c ≈ 1.3에서 상관 길이 ξ가 계 크기 L = 8을 초과. 2차 상전이 — 우주론 섹터의 탄생. β_c 아래: 기하적 요동이 단거리(무질서 상, 모든 스펙트럼 모드에 갭). β_c 위: 장거리 기하적 질서 출현(질서 상, k = 0 모드가 불안정).

동반 논문의 언어로, β_c는 교차 파수 k*가 0에서(부임계: 모든 모드 안정) k* > 0(초임계: 장파장 모드 불안정, 단파장 모드에는 여전히 갭)으로 전이하는 지점. 상전이가 우주론 섹터의 **탄생** — 격자가 처음으로 팽창과 구조 형성을 구동하는 모드를 발전시키면서 더 짧은 파장에서 유질량, 국소화 여기(입자 섹터)를 유지하는 지점.

### 3.4 정보 채널로서의 기하

영 직접 결합의 두 독립 U(1) 장이 같은 역학적 격자에 놓인다. 플라켓 값이 상관.

- 역학적 기하: r(cos P₁, cos P₂) = 0.063 ± 0.009 (7.3σ)
- 고정 기하: r(cos P₁, cos P₂) = 0.003 ± 0.007 (0.4σ, 영과 정합)

역학적 기하가 독립 장 사이의 상관을 매개. 효과 크기(r = 0.063)는 작지만, 유의성(7.3σ)이 높고, 깨끗한 영 대조(고정 기하에서 r = 0.003)가 상관이 직접 장-장 상호작용이 아닌 공유 Hadamard 결합 작용에서 발생함을 확인.

### 3.5 두 번째 상전이로서의 지능

β_c 전이(§3.3)가 구조 — 질량갭, 보존 전하, 장거리 상관을 가진 여기 — 를 생성. 이 여기는 초가법적으로 상호작용(논문 D: E ~ N^γ). 구조화된 여기의 밀도가 증가하면, 두 번째 전이가 가능: 여기에 대한 여기.

결정화(활성 패턴이 새 노드로 응축하는 과정)가 같은 T = C ⊙ S를 자기 출력에 적용. 일차 결정화가 개념(구조화된 여기)을 생성. 개념의 밀도가 임계값 ρ_meta를 초과하면, 이차 결정화가 개념에 대한 개념 — 메타구조 — 를 생성.

새 메커니즘이 아니다. 같은 상전이(§3.3)가 더 높은 수준에서 적용. 연산자 T는 불변. 격자는 불변. 재료만 변했다 — 원시 여기에서 구조화된 여기로.

## 4. 해석적 도출

### 4.1 초과 결합으로부터의 준드시터 팽창

균일 모드 w_e = a에 대해 평형 조건이 a_eq = κn_E/(κn_E − Nβn_Pc)를 산출하며, Nβn_Pc > κn_E일 때 발산. 이 초임계 체제에서 da/dt ∝ δ·a (δ = Nβcn_P − κn_E > 0)이 준지수적 팽창을 줌.

유효 "우주상수" Λ_eff ∝ Nβc − κ(n_E/n_P) — 기본 상수가 아니라 탄성 저항에 대한 게이지-기하 결합의 초과분. 후기 H의 감속은 탄성 강화와 정합적.

### 4.2 AM-GM 부등식으로부터의 균일성

플라켓 가중치 w_P = (∏ w_e)^{1/2}는 기하 평균이며, AM-GM에 의해 모든 에지 가중치가 같을 때 최대화. Regge 게이지 작용 −β w_P cos θ가 더 큰 w_P를 선호(cos θ > 0에서)하므로, 기하가 k = 0(균일) 모드로 응축. 차원 및 게이지 군 무관한 메커니즘.

### 4.3 결합 상전이로부터의 구조 형성

β_c에서의 상전이가 분리: β < β_c(열적 요동 지배, 단거리 기하 상관) 대 β > β_c(게이지-기하 결합 지배, 장거리 질서 발전). 임계 결합: β_c = κ n_E / (N n_P c(β_c)).

교차 파수 k*가 비영이 되면서 스펙트럼이 입자 섹터(k > k*, 유질량 국소화)와 우주론 섹터(k < k*, 불안정 팽창)로 분할. 구조가 교차 스케일 λ* = 2π/k*에서 형성.

### 4.4 차원 의존성

초가법성 지수 γ(d)가 세 효과 모두의 강도를 결정:

| d | γ | 초가법성 |
|---|---|---------|
| 2 | 1.80 | 강함 |
| 3 | 1.44 | 보통 |
| 4 | 0.95 (예비) | 한계 |

γ(d) ≈ −1.6 · [2(d−1)/d] + 3.5으로 포착되며, γ = 1인 임계 차원 d_c ≈ 5를 예측. 4D에서 γ ≈ 1은 초가법성이 간신히 작동함을 의미 — 중력이 네 힘 중 가장 약한 것과 정합적.

### 4.5 영점 에너지와 RG 흐름

**정리 (기하적 영점 에너지).** κ > 0인 T = C · S 격자에서 탄성 에너지 기대값: ⟨S_elastic⟩ = |E|/2, β, N, 게이지 군에 무관.

*증명.* 가우스 근사에서 ⟨(w_e−1)²⟩ = 1/(2κ). 따라서 ⟨S_elastic⟩ = κ · |E| · 1/(2κ) = |E|/2. □

*의미.* 기하는 결코 정확히 평평하지 않다. 영점 에너지 비용은 에지당 1/2이며, 모든 결합 상수에 무관. 자기참조적 읽기의 비용: 읽기에는 요동이 필요하고, 요동에는 보편적 대가가 있다.

**따름정리 (결합된 RG 흐름 존재).** 조립격화가 유효 결합 β'(β, κ)와 κ'(β, κ)를 정의. 흐름은 결합: β'가 κ에 의존(기하 요동이 AM-GM 억제를 통해 게이지 결합을 수정)하고 κ'가 β에 의존(게이지 요동이 기하를 강화). 완전한 전개는 논문 K.

**명제 (스케일 의존 상전이).** 상전이 조건 β_c(κ)가 RG 흐름 β(b), κ(b)와 결합하여, 전이가 특정 조립격화 스케일 b*에서 발생함을 함축. b* 아래: 무질서 상(구조 없음). b* 위: 장거리 상관의 질서 상. 구조 형성이 매개변수 조율이 아닌 스케일에 의해 결정.

### 4.6 반복 결정화로부터의 지능

**설정.** Ψ를 결정화 연산자: Ψ(S)가 부분그래프 S를 활성 패턴으로부터 결정화된 노드 집합에 사상. n차: V_n = Ψ(V_{n-1}).

**4.6.1 상호작용 반경.** 거리 r의 두 여기가 G(r) ~ exp(-r/ξ)로 결합(I Math, 정리 3), ξ = 1/√(2κ). 유의한 결합에 |G(r)| > C_min 필요: r_max = ξ · ln(1/C_min).

**4.6.2 최소 결정화 크기.** 결정화 노드가 야코비안 완전 계수를 가진 안정 고정점이려면 N_min = d_state 앵커 필요.

**4.6.3 메타전이 밀도.** d차원에서 여기 밀도 ρ일 때 상호작용 반경 r_max 내 여기 수 n_int = ρ · Ω_d · r_max^d. 이차 결정화에 n_int ≥ N_min 필요:

ρ_meta = N_min · (2κ)^{d/2} / [Ω_d · (ln 1/C_min)^d]

β_c 전이와 구조적으로 동일: 장거리 질서(여기서: 메타구조)가 출현하는 임계 밀도.

**4.6.4 축소.** *정리.* S의 모든 노드가 S^{d-1}의 반구 내에 있으면, p_new = normalize(Σ aᵢpᵢ)도 같은 반구 내에 있으며, diam(Ψ(S)) ≤ diam(S). 귀결: diam(V₀) ≥ diam(V₁) ≥ ... ≥ 0. 단조 감소, 아래로 유계.

**4.6.5 재생.** 순수 축소는 모든 메타노드를 한 점으로 붕괴(자명 자기모형). 그러나 결정화된 노드가 그래프 G에 합류하여 같은 T = C ⊙ S 역학에 참여. 군집간 에지가 형성되어 새 비대칭을 생성. 새 비대칭이 새 결정화를 구동. 축소는 지름에 선형. 재생은 이차(쌍별 상호작용).

**4.6.6 고정점.** 중간값정리에 의해 축소 − 재생이 어떤 d_∞ > 0에서 영점을 가짐. 비자명 평형 지름. 연화된 결정화 연산자 Ψ_soft가 반구 조건 아래 측지 볼록 껍질을 자신에 사상. 중심 사영에 의해 이 껍질이 R^{d-1}의 컴팩트 볼록 부분집합에 동형. Ψ_soft가 연속.

**브라우어 고정점 정리에 의해 고정점 p* 존재: Ψ_soft(p*) = p*.**

p*로부터의 전파가 V_∞의 노드를 활성화; 그 패턴을 결정화하면 p*가 반환. 구조가 스스로를 읽고 재생산한다. 자기모형이다.

**4.6.7 정의.** 비자명 고정점(d_∞ > 0, 유한 지름, 진행 중인 축소-재생 순환) — 스스로를 읽고 재생산하면서 진화를 계속하는 자기읽기 구조 — 이 우리가 지능이라 부르는 것.

새 공리 도입 없음. 도출은 T = C ⊙ S, 전파자(I Math 정리 3), AM-GM 부등식, 중간값정리, 브라우어 고정점 정리를 사용.

## 5. 논의

### 5.1 범위

모든 결과가 U(1) 게이지장과 선형화 Regge 역학의 2D 격자 모형으로부터. 정량적 우주론 예측에는 현실적 게이지 군의 4D 대규모 격자 시뮬레이션이 필요.

### 5.1b CDT로부터의 독립적 확인

이 논문의 세 우주론적 예측 — 팽창, 균일성, d = 4 — 이 별개의 역학적-기하 격자 장론 접근인 인과적 역학 삼각화(CDT) 프로그램에 의해 독립적으로 확인:

(i) **4D 드 시터 시공간.** 기하에 대한 경로 적분에서 [Ambjorn et al., PRL 93, 2004]. 본 논문의 초과 게이지-기하 결합으로부터의 지수 팽창 예측과 일치.

(ii) **하우스도르프 차원 → 4.** 큰 스케일에서 [Ambjorn et al., PRD 72, 2005]. 논문 J의 정수성 조건 12/[d(d−1)/2] ∈ ℤ에 의한 d = 4 예측과 일치.

(iii) **공간적 균일성.** 출현 기하의 [Ambjorn et al., PRL 95, 2005]. AM-GM 응축 예측(§4.2)과 일치.

CDT는 Hadamard 결합을 사용하지 않고, T = C · S를 가정하지 않으며, 이 프레임워크의 존재를 모르고 개발되었다. 두 독립적 역학-기하 프로그램이 같은 세 정성적 예측 — 팽창, d = 4, 균일성 — 에 수렴하는 것은 이 성질들이 특정 이산화의 인공물이 아닌 게이지-기하 결합의 보편적 귀결이라는 증거를 구성.

### 5.2 이것이 무엇인가

세 수학적 메커니즘 — 각각 격자 게이지-기하 작용에 증명 가능하게 존재 — 이 추가 이론적 재료 없이 팽창, 균일성, 구조 형성의 유사체를 생성:

(i) 초과 결합 → 지수 팽창 (Λ 불필요)
(ii) AM-GM 부등식 → k = 0 응축 (인플라톤 불필요)
(iii) 결합 상전이 → 장거리 질서 (별도 메커니즘 불필요)

구조적 메커니즘: 게이지장과 기하의 곱셈적 결합(T = C⊙S)에 의존하며, 특정 매개변수나 초기 조건에 의존하지 않는다.

### 5.3 검증 가능한 예측

(i) 지수 γ(d)가 좌절 밀도 공식을 따라야 함. 4D 대규모 격자 시뮬레이션으로 검증 가능.
(ii) 유효 우주상수 Λ_eff ∝ Nβc − κ(n_E/n_P)가 게이지장 수 N에 의존함을 예측.
(iii) k = 0 응축이 낮은 차원에서 더 견고(빠른 수렴). 응축물질 유사체에서 검증 가능.
(iv) β_c에서의 상전이가 상관 길이의 특정 결합 강도 스케일링을 예측. 조율 가능한 결합 강도의 다강성 물질에서 검증 가능.

### 5.4 매체로서의 기하: 통합 원리

정보 채널 결과(§3.4) — 독립 장이 공유 역학적 기하를 통해 상관 발전 — 이 가장 깊은 발견. 기하가 수동적 무대가 아닌 능동적 매체임을 시사:

- 장 사이 정보 전달(§3.4)
- 독립 원천의 힘 정렬(논문 F)
- 무질서(게이지 요동)를 질서(장거리 상관)로 변환(§3.3)
- 에너지를 초가법적으로 증폭(논문 D)
- 팽창 구동(§3.1)
- 균일성 강제(§3.2)

여섯 기능 모두 단일 방정식 T = C⊙S에서 발생.

## 6. 결론

네 현상이 격자 게이지-기하 결합에서 출현:

(i) 드 시터 팽창: a(t) ~ exp(Ht), 탄성 저항에 대한 초과 결합으로부터. 해석적 도출 및 MC 검증(R² = 0.983).

(ii) 기하적 응축: k = 0 모드 99.9% 도달, AM-GM 부등식으로부터. 해석적 증명 및 MC 검증.

(iii) 구조 형성: β_c에서의 2차 상전이, ξ 발산. MC 측정.

(iv) 지능: 구조화된 여기의 밀도가 ρ_meta를 초과하면, 반복 결정화가 비자명 고정점 — 스스로를 재생산하는 자기읽기 구조 — 를 생성(§4.6). T = C ⊙ S, 중간값정리, 브라우어 고정점 정리로부터 해석적 도출.

추가: 독립 장이 공유 Hadamard 결합 작용을 통해 상관 발전(r = 0.063, 7.3σ). 기하의 영점 에너지가 정확히 |E|/2, 모든 결합에 무관(§4.5).

모든 현상이 탄성 응답을 가진 Hadamard 구조 T = C⊙S의 필연적 귀결. 구조 형성(iii)과 지능(iv)은 같은 메커니즘 — 상전이 — 의 같은 위계의 다른 수준에서의 적용.

## 방법

2D 정방 격자, 주기적 경계. L = 10(팽창, 응축), L = 8(상전이, 정보 채널). N = 2–5 독립 U(1) 장. 게이지와 기하 스윕을 교대하는 메트로폴리스 알고리즘. 결정론적 시드.
