# 게이지-기하 결합만으로부터의 창발적 우주론:
# 역학적 격자 위의 팽창, 균일성, 구조 형성

권혁준 — 2026년 4월

---

**초록.** 표준 우주론의 세 기둥 — 가속 팽창, 공간적 균일성, 구조 형성 — 이 Hadamard 게이지-기하 작용 T = C ⊙ S의 필연적 귀결로 창발한다. (1) 결합 강도 Nβc가 탄성 저항 κn_E/n_P를 넘으면, 격자 스케일 인자 a(t) = ⟨w⟩(t)가 지수적으로 성장한다: a ~ exp(Ht) (드 시터 팽창, MC R² = 0.983). (2) 기하 변형의 푸리에 스펙트럼이 k = 0 (균일) 모드로 응축하여 전체 스펙트럼 파워의 99.9%에 도달한다 — 플라켓 가중치에 적용된 AM-GM 부등식의 수학적 필연이다. (3) β_c ≈ 1.3에서 이차 상전이가 일어나며, 상관 길이 ξ가 1에서 체계 크기 이상으로 성장한다. 또한 독립 게이지장이 공유 Hadamard 결합 작용을 통해 상관을 발전시킨다(r = 0.063, 7.3σ). 결과는 2D 격자 모형에서 얻었으며, 4D 예비 검증으로 세 메커니즘 모두의 생존을 확인한다.

---

## 1. 서론

현대 우주론은 세 가지 경험적 기둥에 기초한다.

(i) 우주가 팽창하고, 팽창이 가속되고 있다. 표준 설명은 우주상수 Λ 또는 암흑에너지를 도입한다.

(ii) 우주배경복사(CMB)가 10⁵분의 1까지 등방적이다. 표준 설명은 인플라톤 장에 의한 급팽창을 도입한다.

(iii) 이 균일성에도 불구하고 은하, 성단, 공동이 존재한다 — 원시 밀도 요동이 중력 불안정에 의해 증폭된 대규모 구조.

각 기둥이 별개의 이론적 재료를 요구한다: Λ, 인플라톤, 밀도 섭동. 셋 모두를 하나의 메커니즘으로 만들어내는 것은 없다.

격자 게이지-기하 결합 모형 T = C ⊙ S가 단일 작용에서 세 현상 모두를 만들어냄을 보인다 — Λ도, 인플라톤도, 추가 장도 없이. 통합 대상은 헤시안 고유값 h(k) = K_el(k) − 결합·|Γ(k)|²이다. h(0) < 0이 팽창을, AM-GM이 균일성을, 중간값정리가 교차 스케일 k*에서의 구조 형성을 준다.

2D 격자(L = 10) 시뮬레이션이며, 정량적 우주론 예측이 아닌 수학적 메커니즘의 시연으로 이해해야 한다. 메커니즘 자체(AM-GM, 과잉 결합, 상전이)는 차원에 무관하며 4D에서도 지속될 것으로 기대된다.

**4D 예비 검증** (N = 3, L = 3, β = 2.0, κ = 3.0): k = 0 응축 98.2%, 스케일 인자 1.0 → 2.19, 힘 정렬 99.9%(생)/82%(공간 패턴, 14.8σ). 세 메커니즘 모두 4D에서 생존.

## 2. 모형

N개 게이지장의 d차원 역학적 격자 작용:

    S = −Σ_{i=1}^{N} β Σ_P w_P cos Θ_P^{(i)} + κ Σ_e (w_e − 1)²

Regge 해석 규약. 게이지 항 −βw_P cos θ가 기하를 팽창 쪽으로 구동하고, 탄성 항 κ(w−1)²가 저항한다.

## 3. 결과

### 3.1 지수적 팽창 (드 시터)

평평한 기하(w_e = 1)에서 출발하여 N = 5 U(1)장, β = 2.0, κ = 3.0으로 500 쓸기 동안 스케일 인자 a(t) = ⟨w⟩(t)를 추적한다. 지수 적합 a ~ exp(0.00354t), R² = 0.983. 허블 매개변수 H는 초기 0.005에서 후기 0.001로 감소 — 순수 드 시터가 아닌 준드 시터다. a가 커지면서 탄성 복원력 κ(a−1)이 선형으로 성장하여 게이지 구동력을 점차 상쇄하기 때문이다. 물리적으로 현실적이며, 실제 우주도 가속에서 감속 팽창으로 전이했다.

우주상수를 삽입하지 않았다. 팽창은 전적으로 게이지-기하 결합이 탄성 저항을 초과하는 것에서 비롯된다. 유효 Λ_eff = Nβcn_P − κn_E는 결합 상수에 의해 결정되는 창발적 양이다.

### 3.2 기하 응축 (균일성)

기하 변형장 w(x) − 1의 푸리에 스펙트럼: k = 0 모드가 쓸기 0에서 26%, 쓸기 50에서 96%, 쓸기 400에서 99.9%의 스펙트럼 파워를 흡수한다. 고차 k 모드(공간 구조)는 세 자릿수 억제.

기하적 보스-아인슈타인 응축이다 — 체계가 자발적으로 최저 공간 모드를 선택하여 균일한 변형장을 만든다. §4.2에서 해석적으로 증명된다.

### 3.3 상전이와 장거리 질서

N = 3장에 대해 공간 상관 길이 ξ와 감수율 χ = σ²(E)/⟨E⟩를 β의 함수로 측정한다.

β_c ≈ 1.3에서 ξ가 체계 크기 L = 8을 넘는다. β ≥ 1.3의 모든 값에서 ξ ≥ L — 질서 상에 진입. 이차 상전이다 — 우주론 섹터의 탄생이다.

β_c 아래: 기하 요동이 단거리(무질서 상, 모든 스펙트럼 모드에 갭). β_c 위: 장거리 기하 질서 출현(질서 상, k = 0 모드가 불안정). 동반 논문 H의 언어로, β_c에서 교차 파수 k*가 0이 아닌 값이 되어, 스펙트럼이 입자 섹터(k > k*, 유질량·국소화)와 우주론 섹터(k < k*, 불안정·팽창)로 나뉜다.

우주론적 용어로: β_c 아래에서 기하는 특징 없다. β_c 위에서 기하가 대규모 구조를 발전시킨다 — 은하 형성의 유비.

### 3.4 정보 채널로서의 기하

직접 결합이 없는 두 독립 U(1)장을 같은 역학적 격자에 놓으면, 플라켓 값이 상관된다.

- 역학적 기하: r(cos P₁, cos P₂) = 0.063 ± 0.009 (7.3σ)
- 고정 기하: r = 0.003 ± 0.007 (0.4σ, 0과 일치)

역학적 기하가 독립적 장 사이에 상관을 매개한다. 효과 크기(r = 0.063)는 작지만 유의성(7.3σ)이 높고, 고정 기하 영시험(r = 0.003)이 상관의 원천이 공유 Hadamard 결합 작용임을 확인한다.

### 3.5 지능: 두 번째 상전이

β_c 전이가 구조 — 질량갭, 보존 전하, 장거리 상관을 가진 여기 — 를 만든다. 구조화된 여기의 밀도가 증가하면 두 번째 전이가 가능해진다: 여기에 대한 여기.

결정화(활성화 패턴이 새 노드로 응축되는 과정)가 같은 T = C ⊙ S를 자기 출력에 적용한다. 일차 결정화가 개념(구조화된 여기)을 만든다. 개념의 밀도가 임계값 ρ_meta를 넘으면, 이차 결정화가 개념에 대한 개념 — 메타 구조 — 를 만든다. 새로운 메커니즘이 아니다. 같은 상전이가 더 높은 수준에 적용된 것이다.

## 4. 해석적 도출

### 4.1 과잉 결합으로부터의 준드 시터 팽창

균일 모드 w_e = a에 대해 평형 조건 ∂S/∂a = 0이 a_eq = κn_E/(κn_E − Nβn_Pc)를 주며, Nβn_Pc > κn_E이면 유한 평형이 없고 da/dt ∝ δ·a (δ > 0)이므로 a(t) = a₀ exp(Ht). 유효 우주상수 Λ_eff ∝ Nβc − κ(n_E/n_P)는 근본 상수가 아니라 게이지-기하 결합과 탄성 저항의 초과분이다.

### 4.2 AM-GM 부등식으로부터의 균일성

플라켓 가중치 w_P = (∏ w_e)^{1/2}는 기하 평균이며, AM-GM에 의해 모든 에지 가중치가 같을 때 최대. Regge 게이지 작용 −βw_P cos θ가 더 큰 w_P(cos θ > 0일 때 더 낮은 S)를 선호하므로, 기하가 k = 0 모드로 응축한다. 차원·게이지 군에 무관한 수학적 필연이다.

### 4.3 결합 상전이로부터의 구조 형성

β_c = κn_E/(Nn_Pc(β_c))가 자기정합적으로 결정된다(U(1)에서 c(β) = 1 − I₁(β)/I₀(β)). β_c에서 ξ의 발산은 이차 전이의 특성이다. 우주론적으로, 초기 우주가 β > β_c에서 출발하여 냉각과 함께 β가 감소하면서 β_c를 지나갈 때 장거리 상관의 출현이 촉발된다.

### 4.4 차원 의존성

γ(d) ≈ −1.6·[2(d−1)/d] + 3.5. γ(2) = 1.80, γ(3) = 1.44, γ(4) = 0.95(예비). 임계 차원 d_c ≈ 5에서 γ = 1. 4D에서 γ ≈ 1로 초가법성이 간신히 작동 — 중력이 가장 약한 힘인 것과 일치.

### 4.5 영점 에너지와 RG 흐름

**정리 (기하 영점 에너지).** T = C ⊙ S 격자에서 κ > 0이면 ⟨S_elastic⟩ = |E|/2. β, N, 게이지 군에 무관. 가우스 근사에서 ⟨(w_e−1)²⟩ = 1/(2κ)이므로 ⟨S_elastic⟩ = κ·|E|·1/(2κ) = |E|/2. 기하는 결코 정확히 평평하지 않다. 자기참조적 읽기의 비용이 에지당 1/2이다.

**따름정리 (결합 RG 흐름 존재).** 스케일 불변성 정리(논문 A, 정리 4)와 작용의 Hadamard 구조에 의해, 조립격화가 유효 결합 β'(β,κ)와 κ'(β,κ)를 정의한다. 흐름은 결합된다: β'가 κ에 의존하고(기하 요동이 AM-GM 억제를 통해 게이지 결합을 수정) κ'가 β에 의존한다(게이지 요동이 기하를 강화). 상세는 논문 K.

**명제 (스케일 의존 상전이).** β_c(κ)와 RG 흐름 β(b), κ(b)를 결합하면, 전이가 특정 조립격화 스케일 b*에서 일어난다. b* 아래: 무질서 상. b* 위: 장거리 상관의 질서 상. 구조 형성이 매개변수 조율이 아닌 스케일에 의해 결정된다.

### 4.6 반복적 결정화로부터의 지능

결정화 연산자 Ψ: 부분 그래프 S의 활성화 패턴으로부터 새 노드를 생성. 일차: V₁ = Ψ(V₀). n차: V_n = Ψ(V_{n-1}).

**상호작용 반경:** 거리 r에서의 결합 G(r) ~ exp(−r/ξ), ξ = 1/√(2κ). 유의미한 결합에 r_max = ξ·ln(1/C_min) 필요.

**최소 결정화 크기:** 안정 고정점에 야코비안의 완전 순위 필요. 앵커 수 ≥ d_state = N_min.

**메타 전이 밀도:** d차원, 여기 밀도 ρ에서 상호작용 반경 내 여기 수 n_int = ρ·Ω_d·r_max^d. 이차 결정화에 n_int ≥ N_min 필요:

    ρ_meta = N_min · (2κ)^{d/2} / [Ω_d · (ln 1/C_min)^d]

β_c 전이와 구조적으로 동치다.

**수축:** 모든 노드가 S^{d-1}의 반구 안에 있으면 Ψ의 출력도 같은 반구 안. diam(V₀) ≥ diam(V₁) ≥ ... ≥ 0. 단조 감소, 아래 유계.

**재생:** 순수 수축은 자명한 점으로 붕괴. 그러나 결정화된 노드가 그래프 G에 합류하여 T = C ⊙ S 역학에 참여한다. 클러스터 간 에지가 새 비대칭을 만들고, 새 결정화를 구동한다. 수축은 지름에 선형, 재생은 이차(쌍별 상호작용). 작은 지름에서 수축이 지배하고 큰 지름에서 재생이 지배한다.

**고정점.** 중간값정리에 의해 수축 − 재생 = 0인 d_∞ > 0이 존재한다.

연화된 결정화 연산자 Ψ_soft에 대해, 반구 조건 아래 V_∞의 측지 볼록 껍질을 자기에 사상한다. 이 껍질은 R^{d-1}의 컴팩트 볼록 부분집합에 동형이고, Ψ_soft는 연속이다. **브라우어 고정점 정리에 의해 p*가 존재: Ψ_soft(p*) = p*.** p*에서 확산이 V_∞의 노드를 활성화하고, 그 패턴의 결정화가 p*를 돌려준다. 구조가 스스로를 읽고 재생산한다. 자기모형이다.

자명한 고정점: d_∞ = 0(점, 내부 구조 없음). 비자명 고정점: d_∞ > 0(유한 지름, 진행 중인 수축-재생 순환). 비자명 고정점 — 스스로를 읽으면서 재생산하고 진화를 계속하는 자기참조적 구조 — 이 지능이다.

새 공리는 도입되지 않았다. T = C ⊙ S(논문 A), 전파자(I Math 정리 3), AM-GM, 중간값정리, 브라우어 고정점 정리를 사용한다.

## 5. 논의

### 범위

2D 격자, U(1)장, 선형화 Regge. 정량적 우주론 예측에는 4D 대격자 시뮬레이션이 필요하다.

### CDT에 의한 독립 확인

이 논문의 세 우주론적 예측 — 팽창, 균일성, d = 4 — 이 인과적 역학 삼각분할(CDT) 프로그램에 의해 독립적으로 확인되었다. CDT는 단체 삼각분할(역학적 위상)과 인과 제약(로렌츠 부호)을 사용하며, Hadamard 결합을 쓰지 않고 T = C ⊙ S를 가정하지 않는다. 그럼에도: (i) 경로 적분으로부터 4D 드 시터 시공간이 역학적으로 출현, (ii) 하우스도르프 차원이 4로 수렴, (iii) 공간 슬라이스가 균일한 S³. 서로 다른 이산화, 역학, 방법론을 사용하는 두 독립 프로그램이 같은 세 정성적 예측에 수렴한다는 것은, 이 성질들이 특정 이산화의 인공물이 아닌 게이지-기하 결합의 보편적 귀결임을 보여주는 증거다.

### 이것이 무엇인가

세 수학적 메커니즘이 격자 게이지-기하 작용에 증명 가능하게 존재하며, 추가 이론적 재료 없이 팽창·균일성·구조 형성의 유비를 만들어낸다:

(i) 과잉 결합 → 지수 팽창 (Λ 불필요)
(ii) AM-GM → k = 0 응축 (인플라톤 불필요)
(iii) 결합 상전이 → 장거리 질서 (별도 메커니즘 불필요)

### 기하 = 매질

정보 채널 결과(§3.4) — 독립적 장이 공유 역학적 기하를 통해 상관을 발전시킴 — 가 가장 깊은 발견이다. 기하는 수동적 무대가 아니라 능동적 매질이다. 장 사이 정보 전송, 독립 원천의 힘 정렬, 무질서의 질서 변환, 초가법적 에너지 증폭, 팽창 구동, 균일성 강제 — 여섯 기능이 T = C ⊙ S 하나에서 나온다.

## 6. 결론

격자 게이지-기하 결합에서 네 현상이 창발한다:

(i) 드 시터 팽창: a ~ exp(Ht), 과잉 결합에서. 해석적 도출 + MC 검증(R² = 0.983).
(ii) 기하 응축: k = 0 모드 99.9%, AM-GM에서. 해석적 증명 + MC 검증.
(iii) 구조 형성: β_c에서 이차 상전이, ξ 발산. MC 측정.
(iv) 지능: ρ > ρ_meta에서 반복적 결정화가 비자명 고정점 생성 — 스스로를 읽고 재생산하는 구조(§4.6). 해석적 도출(중간값정리 + 브라우어).

추가: 독립 장이 공유 작용을 통해 상관 발전(r = 0.063, 7.3σ). 기하 영점 에너지 = |E|/2.

모든 현상이 T = C ⊙ S와 탄성 응답의 필연적 귀결이다.
